# Chapter 24: Venous and Lymphatic Disease

## 核心整理

- 下肢靜脈回流依賴 **靜脈瓣、肌肉幫浦與呼吸**；瓣膜功能不全或阻塞導致 **靜脈高壓**，是靜脈疾病症狀與併發症的共同機轉。  
- **Venous thromboembolism（VTE）** 涵蓋 **DVT 與 PE**；成因遵循 **Virchow triad（血流停滯、內皮損傷、高凝狀態）**；風險含手術、制動、惡性、荷爾蒙、遺傳性血栓體質。  
- **DVT 診斷**：臨床機率（Wells）＋D-dimer（陰性可排除低機率）＋**加壓超音波（首選影像）**。  
- **VTE 治療**：以 **抗凝** 為主；相較舊有 heparin→warfarin，**DOAC（rivaroxaban、apixaban 等）** 已成多數非癌 VTE 的首選（免監測、方便）。療程依誘因（可逆 vs 未誘發/持續風險）決定。  
- **IVC filter** 僅用於 **抗凝禁忌或抗凝下仍反覆栓塞** 者，非常規；可回收型應適時取出。  
- **肺栓塞風險分層**：大面積（massive，休克）考慮溶栓/取栓；次大面積依右心與生物標記個別化。  
- **慢性靜脈疾病（CEAP 分級）**：從 telangiectasia、varicose veins 到水腫、皮膚變化（脂性皮膚硬化、色素沉著）與 **venous ulcer**（內踝 gaiter 區）。  
- \*\*靜脈潰瘍治療核心是加壓（compression）\*\*＋傷口照護；矯正淺靜脈逆流可加速癒合並降低復發。  
- **靜脈曲張處置**：有症狀或併發症者可 **endovenous ablation（雷射/射頻）、泡沫硬化劑或手術**；現多以微創血管內消融取代傳統 high ligation & stripping。  
- **淋巴水腫**：原發或續發（手術/放療/感染/寄生蟲）；處置以 **complete decongestive therapy（加壓、徒手引流、運動、皮膚照護）** 為主，選擇性顯微手術。  
- **蜂窩性組織炎與淋巴水腫互為因果**：反覆感染惡化淋巴引流，需積極治療感染並長期照護。

## Clinical pearl

- **低機率 DVT＋D-dimer 陰性可安全排除**：善用臨床機率評分避免不必要影像，但高機率者直接做超音波——實用 decision point。  
- **非癌 VTE 多數用 DOAC**：相較 warfarin 免監測、藥物/食物交互作用少；但嚴重腎功能不全、特定癌症或抗磷脂症候群需個別選擇。  
- **IVC filter 不是抗凝的替代品**：僅限抗凝禁忌/失敗，且要規劃取出，長期留置有併發症。  
- **靜脈潰瘍不加壓不會好**：排除動脈缺血（先量 ABI）後，compression 是癒合關鍵；忽略是常見陷阱。

## 快速比較表

| VTE 情境 | 首選處置 | 備註 |
| :---- | :---- | :---- |
| 非癌 DVT/PE | DOAC 抗凝 | 免常規監測 |
| 抗凝禁忌/反覆栓塞 | IVC filter | 適時取出 |
| Massive PE（休克） | 溶栓/取栓 | 個別化 |
| 癌症相關 VTE | LMWH 或特定 DOAC | 依癌別/出血風險 |

| 慢性靜脈病 | 表現 | 主要處置 |
| :---- | :---- | :---- |
| Varicose veins | 迂曲淺靜脈、症狀 | 消融/硬化/手術 |
| Venous ulcer | 內踝、滲液 | 加壓＋傷口照護 |
| 淋巴水腫 | 非凹陷性、皮膚增厚 | 減腫（CDT） |
| DVT 後症候群 | 慢性腫脹疼痛 | 加壓、症狀處置 |

## 後續證據更新

- **急性近端 DVT 常規加做導管溶栓（PCDT）未改善結局**：ATTRACT（Vedantham 等，NEJM 2017）顯示對近端 DVT，於抗凝之外加 **pharmacomechanical catheter-directed thrombolysis** 並 **未降低整體 post-thrombotic syndrome**，且增加出血。故不建議常規使用。**適用／限制**：可能對嚴重 iliofemoral、肢體威脅（如 phlegmasia）之選定病人考慮，但非常規。  
- **DOAC 成為多數 VTE 抗凝首選**：相較 2019 教材更確立——非癌 VTE 以 DOAC 取代 warfarin 為主流，並改變療程與居家治療模式。**適用／限制**：嚴重腎功能不全、抗磷脂症候群、特定藥物交互作用者仍需個別選擇。
