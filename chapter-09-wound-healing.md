# Chapter 09: Wound Healing

## 核心整理

- 正常傷口癒合分三（部分教材四）重疊期：**hemostasis/inflammation → proliferation → remodeling（maturation）**，時間上相互重疊而非截然分段。  
- **Inflammatory phase（0–數日）**：血小板止血與釋放 growth factors（PDGF、TGF-β）；neutrophils 清除細菌、其後 **macrophages** 為主導細胞，兼具清創與分泌生長因子、啟動修復。  
- **Proliferative phase（約 4 日–3 週）**：fibroblasts 產生 collagen（**type III 先增**）、angiogenesis、granulation tissue 形成、epithelialization 與 wound contraction（由 myofibroblasts 驅動）。  
- **Remodeling phase（數週–1 年以上）**：**type III collagen 逐漸被 type I 取代**、交聯增加。傷口強度持續上升，約 **6 週達最終約 80%** 的原始張力強度，永遠不會恢復到 100%。  
- **癒合型式**：primary intention（乾淨傷口直接縫合）、secondary intention（開放讓 granulation 與 contraction 填補，適合污染/感染傷口）、tertiary/delayed primary（先開放觀察數日再縫合）。  
- **Collagen 合成需 vitamin C**（proline/lysine hydroxylation 的輔因子）與充足 O₂；缺乏致傷口崩裂。**Zinc** 為多種酶輔因子，缺乏亦延遲癒合。  
- 影響癒合的全身因素：**組織灌流與氧合**（最關鍵）、糖尿病與高血糖、營養（蛋白質、vit C/A、zinc）、老化、類固醇（可用 **vitamin A** 部分逆轉其抑制作用）、化療/放療、吸菸、感染。  
- **細菌負荷 \>10⁵ CFU/g 組織** 顯著抑制癒合與植皮存活（β-hemolytic streptococcus 例外，量更低即有害）。  
- 慢性傷口（\>30 天未癒）三大類：**venous（最常見下肢潰瘍，內踝 gaiter 區）、arterial（末端、疼痛、無脈）、diabetic/neuropathic（足底受壓點）與 pressure injury**；病生理不同，處置策略也不同。  
- **異常疤痕**：**hypertrophic scar** 侷限於原傷口範圍、可能自行消退；**keloid** 超出原傷口界線、好發深膚色與特定部位（耳垂、胸前、肩），易復發。  
- 傷口照護基本原則：**清創（debridement）** 去除壞死與 biofilm、控制感染、維持濕潤平衡（moist wound healing）、減壓/減張、矯正全身因素與灌流。  
- **Negative pressure wound therapy（NPWT，VAC）**：對開放性、滲液多、granulation 需促進的傷口有幫助，機轉包含移除滲液、減少水腫、促進肉芽與傷口收縮。

## Clinical pearl

- **傷口不癒，先問「灌流與感染」**：慢性傷口停滯多因缺血或高細菌負荷/biofilm；未處理這兩者，任何敷料都無效——這是核心 decision point。  
- **糖尿病足要 offloading**：神經病變足底潰瘍若不減壓（total contact cast 等），再好的局部照護也難癒合，是常見被忽略的關鍵。  
- **類固醇病人給 vitamin A**：長期類固醇抑制發炎期與癒合，補充 vitamin A 可部分逆轉。  
- **keloid 別隨意單純切除**：單純手術切除 keloid 復發率高，需合併 pressure、silicone、類固醇注射或放療等多模式，否則越切越大是常見陷阱。

## 快速比較表

| 期別 | 主要細胞 | 關鍵事件 | 時間 |
| :---- | :---- | :---- | :---- |
| Inflammation | Neutrophil→macrophage | 止血、清創、釋放生長因子 | 0–數日 |
| Proliferation | Fibroblast、內皮 | 膠原、血管新生、上皮化、收縮 | 4日–3週 |
| Remodeling | Fibroblast/myofibroblast | III→I 型膠原、交聯 | 數週–\>1年 |

| 慢性潰瘍 | 典型位置 | 特徵 | 處置重點 |
| :---- | :---- | :---- | :---- |
| Venous | 內踝 gaiter 區 | 滲液、色素沉著 | 加壓治療（compression） |
| Arterial | 肢端、趾 | 疼痛、無脈、蒼白 | 血流重建 |
| Diabetic/neuropathic | 足底受壓點 | 無痛、胼胝 | Offloading、控糖、清創 |
| Pressure injury | 骨突處 | 分期依深度 | 減壓、清創、營養 |

## 後續證據更新

- **Closed-incision NPWT（ci-NPWT）可選擇性用於高風險 primary closure，但效益需依手術情境判讀**：Cochrane 系統性回顧（2022）顯示，預防性 NPWT 相較 standard dressing 可能降低 surgical site infection（SSI）；但不同手術類型、成本效益與其他 wound outcome 的證據確定性不一。**適用／限制**：可考慮於預期 SSI 風險高的切口，但不應作為所有乾淨低風險傷口的常規處置；也須留意 skin blistering。  
- **來源撤稿提醒**：先前使用的 Int Wound J 2023 脊椎手術統合分析已於 2024 年撤稿，故不作為本章證據依據。其餘核心癒合生理與慢性傷口原則相較 2019 教材大致穩定；濕潤傷口照護、清創與矯正灌流仍為基石。
