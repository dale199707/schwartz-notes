# Chapter 07: Trauma

## 核心整理

- 創傷處置以 **ATLS** 架構為骨幹：**primary survey（ABCDE）→ resuscitation → secondary survey → definitive care**，威脅生命的問題邊發現邊處理。  
- **A（Airway）＋頸椎保護**：確保氣道通暢並維持 cervical spine 固定；預期惡化（顏面/頸部傷、GCS 低）者及早確定氣道。  
- **B（Breathing）**：辨識並立即處理危及生命的胸傷——**tension pneumothorax（針刺/胸管減壓）、open pneumothorax、massive hemothorax、flail chest**。  
- **C（Circulation）＋止血**：控制外出血、建立大孔徑靜脈、辨識休克來源；出血是可預防創傷死亡的首因。四大隱藏失血腔：**胸、腹、後腹腔/骨盆、長骨**。  
- **D（Disability）**：GCS、瞳孔、局部神經；**E（Exposure/Environment）**：完全暴露檢視並防低體溫。  
- **FAST（focused assessment with sonography for trauma）** 快速偵測腹腔/心包積液；不穩定＋FAST 陽性者直接進手術室，不必等 CT。  
- **Damage control surgery**：對生理耗竭（致命三角：**hypothermia、acidosis、coagulopathy**）者先控制出血與污染、暫時關腹，復甦穩定後再確定性重建。  
- **Damage control resuscitation**：早期 balanced 血品輸注、限制晶體、允許性低血壓（無 TBI）、保溫、矯正凝血與鈣。  
- **骨盆骨折出血**：以骨盆固定（binder）、必要時 **angioembolization** 或 preperitoneal packing、REBOA 等控制；血流動力學不穩的骨盆出血死亡率高。  
- **實質器官損傷（肝、脾、腎）**：血流動力學 **穩定** 者多可 **non-operative management（NOM）**（含觀察、必要時血管栓塞）；不穩定者手術。  
- **Head injury**：處置目標為避免 **secondary injury**（低血壓、缺氧、顱內壓上升）；維持腦灌流壓，必要時神經外科介入；TBI 病人不適用 permissive hypotension。  
- **創傷後 VTE 風險高**：在出血風險可控後應盡早啟動機械與藥物預防。

## Clinical pearl

- **不穩定＋FAST 陽性＝進手術室**：別為了「更完整的 CT」拖延剖腹，這是創傷最關鍵的 decision point。  
- **想著四個看不見的失血腔**：血壓掉卻沒有明顯外出血，先系統性排查胸、腹、後腹腔/骨盆、長骨。  
- **打破致命三角勝過追求一次到位**：對生理耗竭者選 damage control、暫時關腹，比長時間確定性手術更能救命。  
- **TBI 病人要維持血壓與氧合**：單次低血壓或缺氧即顯著惡化預後，permissive hypotension 在此禁用——重要 red flag。

## 快速比較表

| Primary survey | 重點 | 立即處置範例 |
| :---- | :---- | :---- |
| A（氣道＋頸椎） | 通暢、保護頸椎 | 插管、頸圈 |
| B（呼吸） | 致命胸傷 | 針刺/胸管減壓 |
| C（循環） | 止血、休克 | 壓迫止血、血品復甦 |
| D（神經） | GCS、瞳孔 | 顱內壓處置 |
| E（暴露） | 完全檢視、防低溫 | 保溫 |

| 器官損傷 | 血流動力學穩定 | 血流動力學不穩 |
| :---- | :---- | :---- |

## 後續證據更新

- **創傷出血早期給 TXA、掌握 3 小時窗**：CRASH-2（Lancet 2010）證實受傷後 3 小時內 TXA 降低出血死亡、逾時有害，已納入多數創傷大出血流程。**適用／限制**：僅早期給藥有益。  
- **輕–中度創傷性腦損傷可考慮 TXA**：CRASH-3（Lancet 2019）顯示早期 TXA 對輕中度 TBI 可能降低頭傷相關死亡，對重度無益。**適用／限制**：效益侷限早期給藥、輕中度族群。  
- **大量輸血採 balanced 比例（近 1:1:1）**：PROPPR（JAMA 2015）支持早期高比例血漿與血小板，較 1:1:2 更快達成止血、減少 24 小時失血死亡。**適用／限制**：適用嚴重創傷大出血，實務常再以 viscoelastic 檢測滴定。
