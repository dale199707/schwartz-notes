# Chapter 43: Orthopedic Surgery

## 核心整理

- 骨折描述須系統化：**部位、型態（橫/斜/螺旋/粉碎）、移位/成角、是否關節內、開放 vs 閉鎖**；處置目標為 **復位（reduction）、固定（fixation）、復健**。  
- **開放性骨折（open fracture）** 是急症：**及早抗生素、破傷風預防、清創沖洗、骨折穩定**；污染與軟組織損傷程度（Gustilo 分級）影響感染與癒合。  
- **Compartment syndrome（腔室症候群）是外科急症**：**與外觀不成比例的疼痛、被動牽張痛** 為最早徵象；「6P」中脈搏消失是晚期。診斷靠臨床（±腔室壓），處置為 **緊急 fasciotomy**，延遲致肌肉壞死與永久失能。  
- **髖部骨折（老人）**：低能量跌倒常見；依部位（股骨頸 vs 轉子間）與移位決定 **內固定 vs 關節置換**；圍手術期需處理共病、疼痛與早期活動、預防 VTE 與譫妄。  
- **骨折併發症**：出血（骨盆/股骨）、脂肪栓塞、癒合不良（delayed/nonunion）、感染、關節僵硬、複雜性局部疼痛症候群。  
- **脫臼（dislocation）** 需及早復位（如肩、髖），評估神經血管；**髖關節後脫位** 有股骨頭缺血性壞死與坐骨神經傷害風險，屬急症。  
- **運動與軟組織傷害**：韌帶（如 ACL）、半月板、旋轉肌袖；依活動需求與不穩定程度選擇復健或手術修補/重建。  
- **脊椎**：退化性（椎間盤突出、狹窄）多先保守；**紅旗徵象**（進行性神經缺損、cauda equina、感染/腫瘤/骨折）需影像與手術評估。  
- **骨與軟組織腫瘤**：原發骨腫瘤（如 osteosarcoma，青少年、膝周）與轉移（成人最常見的骨腫瘤為轉移）；懷疑惡性者影像＋切片並轉介腫瘤骨科，避免不當切片。  
- **關節炎與置換**：末期退化性關節炎（膝/髖）以 **人工關節置換** 改善疼痛與功能；感染（PJI）為嚴重併發症。  
- **感染**：骨髓炎與化膿性關節炎需 **及早診斷、引流/清創＋抗生素**；化膿性關節炎會迅速破壞軟骨，屬急症。

## Codex 驗證補強（2026）

- **Open fracture** 應及早給 IV antibiotics、覆蓋無菌敷料、評估 tetanus 與 neurovascular status。現代 guidance 不支持僵化的「6-hour rule」；多數應在 24 小時內 irrigation/debridement，gross contamination、vascular compromise 或 compartment syndrome 則更早。[ACS TQIP Orthopaedic Trauma](https://www.facs.org/media/mkbnhqtw/ortho_guidelines.pdf)
- **Compartment syndrome** 以持續、與傷勢不成比例的 pain、pain on passive stretch、緊繃腔室與進行性神經症狀判斷；脈搏存在不能排除。無法可靠檢查時可測壓，常以 diastolic BP − compartment pressure ≤30 mmHg 輔助，但不可脫離臨床單獨套用。
- Older adult hip fracture 若無需要先處理的可逆不穩定問題，應儘快手術；AAOS 建議 admission 後 24–48 小時內可能改善 outcome。HIP ATTACK 支持加速流程可行，但未證實極度加速手術改善主要結局。[AAOS Hip Fracture Guideline](https://www.aaos.org/globalassets/quality-and-practice-resources/hip-fractures-in-the-elderly/hipfxcpg.pdf)
- Fragility fracture 是 osteoporosis 的 sentinel event；出院計畫應含 calcium/vitamin D adequacy、fall assessment、DXA/secondary cause evaluation，並由 fracture liaison service 或適當專科啟動 anti-osteoporosis treatment。

## Clinical pearl

- **與傷勢不成比例的疼痛＝腔室症候群直到證實為否**：等到無脈已太晚，臨床高度懷疑即 fasciotomy——最關鍵 decision point。  
- **開放性骨折要快給抗生素**：越早給藥、清創越徹底，感染率越低。  
- **化膿性關節炎是關節急症**：延遲引流數小時即可致不可逆軟骨破壞，是不可等的 red flag。  
- **懷疑骨惡性腫瘤別隨意切片**：不當切片路徑會影響日後保肢手術，先轉介專門中心。

## 快速比較表

| 急症 | 關鍵徵象 | 處置 |
| :---- | :---- | :---- |
| 腔室症候群 | 不成比例疼痛、被動牽張痛 | 緊急 fasciotomy |
| 開放性骨折 | 傷口通骨折 | 抗生素、清創、固定 |
| 化膿性關節炎 | 關節紅腫痛、發燒 | 引流/清創＋抗生素 |
| 髖後脫位 | 肢短內旋 | 及早復位、評估神經血管 |

| 骨折處置要素 | 內容 |
| :---- | :---- |
| 復位 | 恢復對位對線 |
| 固定 | 內/外固定或石膏 |
| 軟組織 | 保護、評估腔室與神經血管 |
| 復健 | 早期活動、功能恢復 |

## 後續證據更新

- **髖部骨折「加速手術」未顯著降低死亡，但可改善部分結果**：HIP ATTACK（Lancet 2020）在髖部骨折病人比較 **加速手術（診斷後數小時內）** 與標準照護，**主要結局（死亡與主要併發症）無顯著差異**，但加速組 **譫妄、疼痛、住院與部分併發症較少、較早下床**。**適用／限制**：仍支持避免不必要延遲；極高風險需優化的病人應個別權衡。  
- 相較 2019 教材，髖部骨折照護更強調 **多科（老年醫學共管）、早期手術與活動、VTE 與譫妄預防** 的整合路徑（fragility fracture programs）。**適用／限制**：需跨科資源與流程建置。
