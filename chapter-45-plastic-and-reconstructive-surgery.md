# Chapter 45: Plastic and Reconstructive Surgery

## 核心整理

- 重建的核心思維是 **reconstructive ladder（重建階梯）**：由簡至繁——**次級癒合 → 直接縫合 → 皮膚移植 → 局部皮瓣 → 遠處/游離皮瓣**；選最簡單且能達成目標者，但功能/美觀需求可「跳階」（reconstructive elevator）。  
- **皮膚移植（skin graft）**：**split-thickness（STSG）** 取材多、易存活但收縮多；**full-thickness（FTSG）** 收縮少、美觀佳但需良好受床；移植存活靠 imbibition→inosculation→revascularization，需良好血運受床（骨/肌腱裸露、放射傷口不佳）。  
- **皮瓣（flap）** 帶自身血供，用於覆蓋無法植皮的受床（裸露骨/肌腱/內固定、放射傷口）；分 **隨意型（random）** 與 **軸型（axial，含帶蒂與游離）**；游離皮瓣以顯微血管吻合重建血運。  
- **游離皮瓣的成敗在血管吻合與監測**：術後密切監測皮瓣灌流（顏色、微血管再充填、都卜勒），**靜脈栓塞** 常為早期失敗主因，及早再手術可挽救。  
- **傷口覆蓋原則**：先 **清創、控制感染、良好受床準備**，再選重建方式；合併骨折/內固定外露需骨科—整形合作（如下肢創傷）。  
- **燒傷重建**：急性期植皮覆蓋（見燒傷章），後期處理攣縮、增生性疤痕與功能/外觀重建。  
- **乳房重建**：分 **自體組織（如 DIEP、TRAM 皮瓣）** 與 **植入物（expander/implant）**；時機可即刻或延遲，需權衡放療、體型與病人偏好。  
- **顱顏／先天**：**cleft lip/palate（唇顎裂）** 依時程分期修補（唇約 3 月齡、顎約 9–12 月齡的常見原則），需長期多科（語言、齒顎、聽力）照護。  
- **周邊神經與手部**（與手外科重疊）：神經修補、肌腱重建、複雜組織缺損覆蓋。  
- **淋巴水腫的顯微手術**：如 lymphaticovenous anastomosis、vascularized lymph node transfer，為選擇性進階治療。  
- **美容與功能並重**：重建須兼顧腫瘤/創傷的根本處置與後續功能、外觀與心理層面。

## Codex 驗證補強（2026）

- Skin graft 需要 vascularized wound bed。STSG 可在健康 granulation tissue、vascularized periosteum 或 paratenon 上存活，但**裸露且無 periosteum 的 cortical bone、無 paratenon 的 tendon、hardware 或 irradiated fibrotic bed** 通常需先改善床面或以 flap coverage。
- Gustilo IIIB/C lower-extremity trauma 應早期採 orthoplastic strategy，同步規劃 fixation、serial debridement、vascular repair 與 durable soft-tissue coverage；coverage 越延遲，deep infection 與 nonunion 風險通常越高。
- Prepectoral breast reconstruction 可減少 animation deformity，但 evidence 多來自選擇性 observational cohort；skin-flap perfusion、radiation、BMI、smoking、ADM 使用與 center experience 會影響 infection、necrosis 與 implant loss，不能宣稱普遍優於 subpectoral。
- Cleft lip/palate repair timing 依 airway、feeding、growth、speech protocol 與 center expertise 個別化；「rule of 10s」是歷史性參考，不是所有中心的強制標準。
- Lymphovenous bypass、vascularized lymph-node transfer 與 liposuction 只適用 selected lymphedema phenotype；比較證據仍有限，必須與 compression、skin care、weight management 與 infection prevention 整合。

## Clinical pearl

- **植皮受床要有血運**：裸露骨（無骨膜）、肌腱（無腱旁組織）、放射或感染傷口植皮易失敗，這類受床需皮瓣——關鍵 decision point。  
- **游離皮瓣術後看靜脈端**：靜脈鬱血（皮瓣暗紫、快速再充填）常先於動脈問題，及早再探查可救皮瓣。  
- **重建階梯選「夠用的最簡單」**：但功能/美觀關鍵部位（如臉、手）可直接選較高階方法，不必拘泥逐階。  
- **下肢複雜創傷要骨—整形合作**：固定與軟組織覆蓋的時機協調影響感染與癒合。

## 快速比較表

| 重建方式 | 適用 | 特點 |
| :---- | :---- | :---- |
| 直接縫合 | 小、可對合傷口 | 最簡單 |
| STSG | 大面積、受床佳 | 取材多、收縮多 |
| FTSG | 臉/手小範圍 | 收縮少、美觀 |
| 局部皮瓣 | 鄰近組織可用 | 帶血供、覆蓋裸露結構 |
| 游離皮瓣 | 大/複雜缺損 | 顯微吻合、需監測 |

| 乳房重建 | 方式 | 考量 |
| :---- | :---- | :---- |
| 自體（DIEP/TRAM） | 自身組織 | 自然、手術較大 |
| 植入物 | Expander/implant | 較簡單、受放療影響 |

## 後續證據更新

- **植入物乳房重建趨向 prepectoral（胸大肌前）平面**：一篇系統性回顧/統合分析（Ann Surg Oncol 2022）比較 **prepectoral 與 subpectoral** 植入物乳房重建，顯示 prepectoral 在 **動態變形（animation deformity）與疼痛** 方面有優勢，整體併發症相近，成為愈趨主流的選項。**適用／限制**：需良好皮瓣血運與（常）搭配 acellular dermal matrix；受床品質與放療計畫影響適用性。  
- 相較 2019 教材，**ERAS 於游離皮瓣重建、以及淋巴水腫的顯微外科（LVA、VLNT）** 應用擴大，改善恢復與功能。**適用／限制**：屬專門技術，成效依病灶與中心經驗而異。
