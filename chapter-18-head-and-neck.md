# Chapter 18: Disorders of the Head and Neck

## 核心整理

- 頭頸部惡性腫瘤以 **squamous cell carcinoma（HNSCC）** 為主，來自上呼吸消化道黏膜（口腔、口咽、喉、下咽、鼻咽等），與 **吸菸、酗酒、HPV、EBV（鼻咽癌）** 相關。  
- **HPV-associated oropharyngeal cancer** 多見於 tonsil/base of tongue，整體預後較佳；p16 IHC 是 oropharyngeal SCC 常用 surrogate，但不是所有 head-and-neck site 的通用 HPV test。吸菸量、stage 與其他因素仍影響預後，不能因 p16+ 就視為低風險。  
- **頸部腫塊評估**：年齡與病史導向——成人單側無痛漸大頸部腫塊要排除惡性；先影像與 **FNA**，**避免對疑癌淋巴結貿然切開切片**（open biopsy 影響後續處置）。  
- **原發灶不明的頸部轉移**：以理學、內視鏡、影像（含 PET-CT）與定向切片尋找原發；HPV/EBV 狀態可指引原發部位。  
- **口腔癌**：與菸酒檳榔相關；處置以手術為主（原發切除＋頸部淋巴廓清），視分期加放/化療。  
- **喉癌**：聲門型早期以聲音沙啞表現、預後佳（放療或部分喉切除可保留發聲）；晚期需全喉切除或器官保留 chemoradiation。  
- **頸部淋巴分區（level I–VI）** 指導淋巴廓清範圍；**selective neck dissection** 依原發部位的引流站選擇，兼顧腫瘤控制與功能保留。  
- **唾液腺腫瘤**：腮腺最常見且多良性（**pleomorphic adenoma** 最常見，需完整切除防復發與惡性轉化；Warthin tumor）；愈小的唾液腺（如舌下腺）腫瘤惡性比例愈高。  
- **腮腺手術核心是保護 facial nerve（CN VII）**；惡性且神經受累時才犧牲，並考慮重建。  
- **Thyroglossal duct cyst** 與 branchial cleft anomaly 是先天性鑑別；前者常隨吞嚥/伸舌移動，確診後以 **Sistrunk procedure** 降低復發。成人新出現 lateral cystic neck mass 不能先假設為 branchial cyst：HPV-related oropharyngeal SCC 常以 cystic nodal metastasis 表現，必須完成影像、FNA/HPV-related work-up 至取得診斷。  
- **重建與功能**：頭頸腫瘤治療強調 **多科團隊**，兼顧腫瘤控制與吞嚥、發聲、外觀之功能重建（皮瓣）。

## Clinical pearl

- **成人單側頸部腫塊先 FNA、別 open biopsy**：疑似鱗癌轉移淋巴結若貿然切開切片，會破壞頸部平面、影響後續廓清與預後——最重要的 decision point。  
- **HPV/p16 會改變 oropharyngeal staging，但不自動允許降階**：p16 影響 AJCC staging 與預後討論；radiation/systemic intensity 的 de-escalation 應限 trial 或嚴格 guideline pathway，不能只因 p16+ 自行減量。  
- **腮腺手術以 facial nerve 為地標**：術中辨識與保護 CN VII 是避免永久顏面麻痺的關鍵。  
- **甲狀舌管囊腫要做 Sistrunk**：單純囊腫切除復發率高，需一併切除舌骨中段與管道。

## 快速比較表

| 部位/型別 | 相關因子 | 預後/處置重點 |
| :---- | :---- | :---- |
| HPV+ 口咽癌 | HPV/p16+ | 預後佳、治療分層 |
| 吸菸相關 HNSCC | 菸酒 | 預後較差、多模式 |
| 鼻咽癌 | EBV | 放/化療為主 |
| 聲門喉癌（早期） | 吸菸 | 保留發聲（放療/部分切除） |

| 頸部腫塊 | 特徵 | 處置 |
| :---- | :---- | :---- |
| 鱗癌轉移淋巴結 | 成人、單側漸大 | FNA、勿 open biopsy |
| Thyroglossal duct cyst | 中線、隨吞嚥動 | Sistrunk procedure |
| 疑 branchial cleft cyst | 側頸囊性腫塊 | 成人先排除 HPV-related cystic nodal metastasis；確診先天病灶後再完整切除 |
| Pleomorphic adenoma | 腮腺、良性最常見 | 完整切除、保護 CN VII |

## 後續證據更新

- **HPV 相關口咽癌「治療降階」須謹慎——不可用 cetuximab 取代 cisplatin**：RTOG 1016（Gillison 等，Lancet 2019）顯示對 HPV 陽性口咽癌，放療併 **cetuximab** 的 overall survival **劣於** 放療併 **cisplatin**，故 cisplatin 仍為併用化放療的標準。**適用／限制**：適用可耐受 cisplatin 者；降階治療應在臨床試驗或嚴格評估下進行，不可隨意以 cetuximab 替代。  
- 相較 2019 教材，**p16/HPV 狀態已納入口咽癌分期（AJCC 第 8 版分開分期）**，反映其顯著較佳預後；並推動經口機器人手術（TORS）等器官保留策略的研究。**適用／限制**：分期與術式選擇需多科團隊個別化。
- **成人 cystic neck mass 仍按 malignancy work-up 走到底**：FNA 或影像顯示 cystic 並不等於 benign；在沒有取得診斷前，應持續 upper-aerodigestive evaluation、appropriate imaging/ancillary testing，open biopsy 只在完整評估後考慮。
