# Chapter 42: Neurosurgery

## 核心整理

- 顱內容物遵循 **Monro-Kellie doctrine**：顱內為固定容積（腦、血、腦脊髓液），任一增加須以他者代償，代償耗盡後 **顱內壓（ICP）急升**。  
- **腦灌流壓（CPP \= MAP − ICP）** 是關鍵：處置重症腦傷的核心在 **維持 CPP、控制 ICP、避免 secondary injury（低血壓、缺氧、發燒、低/高碳酸血症）**。  
- **Traumatic brain injury（TBI）** 以 **GCS** 分輕/中/重；影像用 CT。**Epidural hematoma**（多為中腦膜動脈、lucid interval、雙凸透鏡狀）與 **subdural hematoma**（橋靜脈、新月狀、常見於老人/抗凝）需辨識與適時手術。  
- **顱內壓升高徵象**：頭痛、嘔吐、意識下降、瞳孔變化；**Cushing triad（高血壓、心搏過緩、不規則呼吸）** 為腦疝晚期 red flag。  
- **ICP 處置階梯**：抬高床頭、鎮靜止痛、維持正常二氧化碳與滲透治療（**hypertonic saline / mannitol**）、腦室外引流，難治者手術減壓；短暫過度換氣僅用於急性惡化橋接。  
- **Aneurysmal subarachnoid hemorrhage（SAH）**：突發「這輩子最劇烈頭痛」；CT（陰性可腰椎穿刺）；併發症含 **再出血、vasospasm/延遲性腦缺血（nimodipine 預防）、hydrocephalus**。動脈瘤需早期處理（夾閉或栓塞）。  
- **自發性腦內出血（ICH）**：與高血壓、抗凝相關；處置以血壓與凝血管理、ICP 控制為主，手術角色依位置/大小/惡化個別化（如小腦出血較積極）。  
- **腦腫瘤**：原發（如 glioma、meningioma）與轉移；處置依型別與位置行手術（最大安全切除）、放療、化療；**meningioma** 多良性可切除。  
- **脊椎急症**：**cauda equina syndrome**（鞍區麻木、尿滯留/失禁、雙側坐骨神經痛）需 **緊急 MRI 與減壓**；脊髓壓迫（腫瘤/膿瘍/血腫）亦為急症。  
- **Hydrocephalus**：腦脊髓液循環/吸收障礙致腦室擴大與 ICP 升高，處置為分流（shunt）或內視鏡第三腦室造廔。  
- **神經外科感染/其他**：腦膿瘍、脊椎硬膜外膿瘍需引流＋抗生素；功能性神經外科（癲癇、動作障礙）為次專科領域。

## Codex 驗證補強（2026）

- Severe TBI 的 ICP monitoring/治療需整合 clinical exam 與 imaging；Brain Trauma Foundation 建議在 **ICP >22 mmHg** 時治療，CPP target 約 **60–70 mmHg**，不應為維持 CPP 而常規 aggressive fluid/pressor 推至 >70。[Brain Trauma Foundation](https://braintrauma.org/coma/guidelines/severe-tbi)
- **Corticosteroid 會增加 severe TBI mortality，禁用於改善 outcome 或降低 ICP**。Prolonged prophylactic hyperventilation 可能加重 cerebral ischemia；僅在 impending herniation 作短暫 bridge，同步安排 definitive ICP treatment。
- Aneurysmal SAH 應儘早以 enteral **nimodipine** 降低 delayed cerebral ischemia，並由有經驗團隊盡量在 24 小時內 secure aneurysm。clipping 或 coiling 依 anatomy、age、hematoma、durability 與中心經驗選擇，不能籠統宣稱 coiling 對所有患者較優。[2023 AHA/ASA aSAH Guideline](https://www.ahajournals.org/doi/epdf/10.1161/STR.0000000000000436)
- Negative noncontrast CT 排除 SAH 的能力取決於發作後時間、掃描品質與判讀；若超過早期高敏感時窗或臨床懷疑仍高，需依情境續做 LP 或 CTA。**Delayed cerebral ischemia 不等同 angiographic vasospasm**，臨床惡化需排除 hydrocephalus、seizure、rebleeding 與 systemic cause。

## Clinical pearl

- **突發「最劇烈頭痛」想 SAH**：thunderclap headache 是不可錯過的 red flag，CT 陰性仍需進一步（腰穿/CTA）。  
- **重症腦傷的一切在避免 secondary injury**：單次低血壓或缺氧顯著惡化預後，維持血壓與氧合是核心 decision point。  
- **cauda equina 是時間依賴急症**：鞍區麻木＋尿滯留需緊急 MRI 與減壓，延遲致永久失能/失禁。  
- **Cushing triad 是腦疝晚期**：高血壓＋心搏過緩＋呼吸異常代表已危急，需立即降 ICP 處置。

## 快速比較表

| 出血型態 | 來源/影像 | 特徵 |
| :---- | :---- | :---- |
| Epidural | 中腦膜動脈、雙凸透鏡 | Lucid interval、可急速惡化 |
| Subdural | 橋靜脈、新月狀 | 老人/抗凝、亞急性 |
| SAH | 動脈瘤、基底池 | 劇烈頭痛、vasospasm 風險 |
| ICH | 高血壓/抗凝 | 深部核團常見 |

| 急症 | 關鍵徵象 | 處置 |
| :---- | :---- | :---- |
| 腦疝 | Cushing triad、瞳孔 | 降 ICP、減壓 |
| Cauda equina | 鞍麻、尿滯留 | 緊急 MRI＋減壓 |
| Hydrocephalus | 腦室擴大、ICP↑ | 分流/造廔 |

## 後續證據更新

- **破裂動脈瘤適合者「栓塞優於夾閉」（ISAT）**：ISAT（Molyneux 等，Lancet 2002）在 **同時適合兩種治療** 的破裂顱內動脈瘤，顯示 **endovascular coiling** 相較 neurosurgical clipping 一年後獨立存活較佳，確立栓塞為多數合適病灶的首選。**適用／限制**：適用兩者皆可行的病灶；部分寬頸、特定位置或大型血腫仍以夾閉/手術為宜，需神經血管團隊評估與長期追蹤（再治療率）。  
- 相較 2019 教材，**急性缺血性中風的機械取栓** 已成大血管阻塞的標準（時間窗延伸），改變急性中風的血管內處置流程。**適用／限制**：需影像篩選與具備介入能力的中風中心；本章僅概述，細節見中風處置指引。
