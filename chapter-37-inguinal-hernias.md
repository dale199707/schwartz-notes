# Chapter 37: Inguinal Hernias

## 核心整理

- **腹股溝管解剖**：內環（deep ring，transversalis fascia）到外環（superficial ring，external oblique aponeurosis）；內含男性精索/女性圓韌帶。**Hesselbach triangle**（下腹壁動脈、腹直肌外緣、鼠蹊韌帶）是 direct hernia 的突出區。  
- **Indirect（間接）** 疝經 **內環外側於下腹壁動脈** 突出（沿精索，先天性 processus vaginalis 未閉），最常見、各年齡；**direct（直接）** 經 Hesselbach 三角（下腹壁動脈內側）、後天腹壁薄弱。  
- **Femoral hernia** 經 femoral canal（鼠蹊韌帶下方、股靜脈內側）突出，**女性相對較多、嵌頓/絞窄風險高**，多建議修補。  
- **臨床診斷** 以病史與理學為主（站立/Valsalva 時鼠蹊突起、可復位性）；不確定時可影像（超音波/CT）。  
- **併發症**：**incarceration（嵌頓，不可復位）** 與 **strangulation（絞窄，血流受阻缺血）**——後者為外科急症，表現劇痛、壓痛、皮膚變化、腸阻塞徵象。  
- **修補方式**：**open tension-free mesh（Lichtenstein）** 為經典標準；**腹腔鏡（TEP/TAPP）** 補片修補於 **雙側、復發疝、需早日恢復活動** 者具優勢（術後疼痛少、恢復快）。  
- **Mesh 修補復發率低於非補片縫合修補（如 Bassini、Shouldice）**；Shouldice 為非補片術式中結果最佳者。  
- **手術要點**：辨識並保護 **ilioinguinal、iliohypogastric、genitofemoral 神經**（避免慢性疼痛）與精索構造；避免張力。  
- **慢性術後鼠蹊疼痛（inguinodynia）** 是重要併發症，與神經處理與補片相關；預防重於治療。  
- **絞窄或嵌頓合併腸阻塞** 需緊急手術，評估腸活性；污染時補片使用需審慎。  
- **特殊情境**：女性鼠蹊疝需警覺 femoral hernia；兒童多為間接疝（先天），以高位疝囊結紮（herniotomy）為主、通常不需補片。

## Clinical pearl

- **不可復位＋劇痛＝想絞窄、儘速手術**：嵌頓合併壓痛、皮膚變化或腸阻塞是不能等的 red flag。  
- **女性鼠蹊腫塊警覺 femoral hernia**：股疝嵌頓/絞窄風險高，多建議修補，勿當單純腹股溝疝輕忽。  
- **雙側或復發疝考慮腹腔鏡**：TEP/TAPP 在這些情境恢復快、疼痛少——重要 decision point。  
- **保護三條神經防慢性疼痛**：術中辨識並妥善處理神經，是預防 inguinodynia 的關鍵。

## 快速比較表

| 型別 | 突出位置（相對下腹壁動脈） | 特點 |
| :---- | :---- | :---- |
| Indirect | 外側、經內環 | 最常見、沿精索、先天性 |
| Direct | 內側、Hesselbach 三角 | 後天腹壁薄弱 |
| Femoral | 鼠蹊韌帶下、股靜脈內側 | 女性多、絞窄風險高 |
| 兒童疝 | 間接為主 | 高位結紮、通常免補片 |

| 修補方式 | 適用 | 特點 |
| :---- | :---- | :---- |
| Lichtenstein（open mesh） | 多數初發單側 | 經典標準、復發低 |
| 腹腔鏡 TEP/TAPP | 雙側、復發、早恢復 | 疼痛少、恢復快 |
| Shouldice（非補片） | 不用補片時 | 非補片中最佳 |
| 緊急手術 | 絞窄/嵌頓 | 評估腸活性 |

## 後續證據更新

- **微症狀的腹股溝疝可「觀察等待」**：Fitzgibbons 等（JAMA 2006）RCT 顯示 **無症狀或微症狀男性** 腹股溝疝採 watchful waiting，**急性絞窄極為罕見**，安全可行；但約多年追蹤下 **多數最終因症狀出現而接受手術**（crossover 高）。**適用／限制**：適用男性、無症狀/微症狀者；**女性（股疝風險）、有症狀或嵌頓病史者仍建議修補**。  
- 相較 2019 教材，**腹腔鏡修補於雙側與復發疝的角色更確立**，並更強調 **慢性術後鼠蹊疼痛的預防**（神經處理、補片與固定方式選擇）。**適用／限制**：術式選擇需依疝型、病人因素與術者經驗個別化。
