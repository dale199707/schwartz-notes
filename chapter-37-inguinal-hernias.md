# Chapter 37: Inguinal Hernias

## 核心整理

- **腹股溝管解剖**：內環（deep ring，transversalis fascia）到外環（superficial ring，external oblique aponeurosis）；內含男性精索/女性圓韌帶。**Hesselbach triangle**（下腹壁動脈、腹直肌外緣、鼠蹊韌帶）是 direct hernia 的突出區。  
- **Indirect（間接）** 疝經 **內環外側於下腹壁動脈** 突出（沿精索，先天性 processus vaginalis 未閉），最常見、各年齡；**direct（直接）** 經 Hesselbach 三角（下腹壁動脈內側）、後天腹壁薄弱。  
- **Femoral hernia** 經 femoral canal（鼠蹊韌帶下方、股靜脈內側）突出，**女性相對較多、嵌頓/絞窄風險高**，多建議修補。  
- **臨床診斷** 以病史與理學為主（站立/Valsalva 時鼠蹊突起、可復位性）；不確定時可影像（超音波/CT）。  
- **併發症**：**incarceration（嵌頓，不可復位）** 與 **strangulation（絞窄，血流受阻缺血）**——後者為外科急症，表現劇痛、壓痛、皮膚變化、腸阻塞徵象。  
- **修補方式**：mesh-based repair 是多數成人標準；**open Lichtenstein** 與 **laparo-endoscopic TEP/TAPP** 都是第一線選項，依病人、疝型、麻醉條件與術者經驗決定。若具專業能力，bilateral hernia 優先考慮 TEP/TAPP；復發疝則使用與前次不同的 tissue plane。  
- **Mesh 修補復發率低於非補片縫合修補（如 Bassini、Shouldice）**；Shouldice 為非補片術式中結果最佳者。  
- **手術要點**：辨識並保護 **ilioinguinal、iliohypogastric、genitofemoral 神經**（避免慢性疼痛）與精索構造；避免張力。  
- **慢性術後鼠蹊疼痛（CPIP）** 是重要併發症；open repair 應辨識相關 nerves 並避免不必要損傷。不能把 routine prophylactic neurectomy 當成所有病人的標準，既有 CPIP 則需排除 recurrence、由多專科採藥物/interventional treatment，選定者才考慮 neurectomy ± mesh removal。  
- **絞窄或嵌頓合併腸阻塞** 需緊急手術並評估 bowel viability。clean 或部分 clean-contaminated emergency repair 可考慮 macroporous polypropylene mesh；若 gross enteric spillage、peritonitis 或無法控制 infection，才更需避免永久 mesh 或採 staged plan。  
- **特殊情境**：女性鼠蹊疝需警覺 femoral hernia；兒童多為間接疝（先天），以高位疝囊結紮（herniotomy）為主、通常不需補片。

## Clinical pearl

- **不可復位＋劇痛＝想絞窄、儘速手術**：嵌頓合併壓痛、皮膚變化或腸阻塞是不能等的 red flag。  
- **女性鼠蹊腫塊警覺 femoral hernia**：股疝嵌頓/絞窄風險高，多建議修補，勿當單純腹股溝疝輕忽。  
- **雙側宜 laparo-endoscopic；復發看前次平面**：有經驗時 bilateral hernia 優先 TEP/TAPP；前次 anterior repair 復發改 posterior TEP/TAPP，前次 posterior repair 復發則改 anterior repair。  
- **保護三條神經防慢性疼痛**：術中辨識並妥善處理神經，是預防 inguinodynia 的關鍵。

## 快速比較表

| 型別 | 突出位置（相對下腹壁動脈） | 特點 |
| :---- | :---- | :---- |
| Indirect | 外側、經內環 | 最常見、沿精索、先天性 |
| Direct | 內側、Hesselbach 三角 | 後天腹壁薄弱 |
| Femoral | 鼠蹊韌帶下、股靜脈內側 | 女性多、絞窄風險高 |
| 兒童疝 | 間接為主 | 高位結紮、通常免補片 |

| 修補方式 | 適用 | 特點 |
| :---- | :---- | :---- |
| Lichtenstein（open mesh） | 多數初發單側 | 經典標準、復發低 |
| Laparo-endoscopic TEP/TAPP | 雙側；前次 anterior repair 後復發 | 疼痛少、恢復快，需專業經驗 |
| Shouldice（非補片） | 不用補片時 | 非補片中最佳 |
| Anterior repair | 前次 posterior repair 後復發 | 使用不同 tissue plane |

## 後續證據更新

- **微症狀的腹股溝疝可「觀察等待」**：Fitzgibbons 等（JAMA 2006）RCT 顯示 **無症狀或微症狀男性** 腹股溝疝採 watchful waiting，**急性絞窄極為罕見**，安全可行；但約多年追蹤下 **多數最終因症狀出現而接受手術**（crossover 高）。**適用／限制**：適用男性、無症狀/微症狀者；**女性（股疝風險）、有症狀或嵌頓病史者仍建議修補**。  
- **HerniaSurge 2023 update** 更強調 surgeon expertise 與 plane-based strategy：bilateral 優先 laparo-endoscopic；recurrent hernia 應避開前次 dissected plane。女性因 occult femoral hernia 與 emergency presentation 風險，應及時修補並優先考慮可檢視 myopectineal orifice 的 laparo-endoscopic approach。  
- **緊急病例不是一律禁用 mesh**：在 clean field 及部分 bowel resection 但無 gross contamination 的 clean-contaminated field，synthetic mesh 可降低 recurrence；gross spillage/peritonitis 則須個別化。**適用／限制**：證據品質有限，仍以病人穩定度、污染程度與術者判斷為準。
