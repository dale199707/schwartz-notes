# Chapter 15: Molecular Biology, The Atomic Theory of Disease, and Precision Surgery

## 核心整理

- 分子生物學的核心法則：**DNA → RNA → 蛋白質**（transcription、translation）；基因表現受表觀遺傳（甲基化、組蛋白修飾）與訊息傳遞調控,是理解疾病與治療標的的基礎。  
- **癌症為基因體疾病**:driver mutation 累積驅動 hallmarks(見腫瘤章);分 **oncogene(功能獲得)** 與 **tumor suppressor(功能喪失)**,以及 DNA 修復/MMR 缺陷。  
- **精準醫療(precision medicine)** 依個體分子特徵選擇治療:以 **biomarker** 指引標靶與免疫治療,而非僅依器官/組織學。  
- **定序技術(NGS)** 使腫瘤基因體 profiling 常規化,偵測可用藥突變(actionable mutation)、融合基因與 tumor mutational burden。  
- **液態切片(liquid biopsy/ctDNA)**:以血中循環腫瘤 DNA 偵測突變、監測治療反應與 **微量殘存疾病(MRD)**、早期偵測復發,對外科的術後監測日益重要。  
- **標靶治療對象** 例:EGFR、ALK、ROS1、BRAF、HER2、KIT/PDGFRA、NTRK 融合等;使用前需分子檢測確認。  
- **免疫治療 biomarker**:PD-L1、MSI-high/dMMR、TMB(見腫瘤章),其中 **MSI-high/dMMR 為跨癌別(tumor-agnostic)** 的免疫治療反應指標。  
- **遺傳性癌症症候群** 的分子基礎(BRCA、Lynch/MMR、APC 等)影響 **預防性手術、篩檢與家族諮詢**。  
- **對外科的意涵**:分子分型改變 **手術時機與範圍**(如 neoadjuvant 標靶/免疫先行)、切緣與淋巴處置的必要性,以及術後監測(ctDNA)。  
- **標本處理與生物資料庫**:正確的檢體採集、保存與註記是分子檢測與研究的前提;外科在組織取得上扮演關鍵角色。  
- **限制與倫理**:並非所有突變皆可用藥、檢測有假陽/陰與偶發發現(incidental findings),需審慎判讀與知情同意。

## Codex 驗證補強（2026）

- **Tumor genomic testing 不等於 germline testing**：腫瘤檢體發現可能具遺傳意義的 variant 時，應依基因、variant allele fraction、家族史與腫瘤型別轉介 genetic counseling，並以正常組織確認 germline status。
- **ctDNA / molecular residual disease（MRD）** 在多種 solid tumor 具有預後價值，但「能預測復發」不等於「依結果加減治療會改善存活」；除已有適應症或 guideline 支持的情境外，不應把 ctDNA 單獨當作常規 treatment-selection 工具。
- Tissue-agnostic therapy 仍有明確邊界。FDA 的 pembrolizumab **TMB-high** accelerated approval限於 unresectable/metastatic、經先前治療後進展、無滿意替代療法，且須 FDA-approved test；≥10 mutations/Mb 不是任何平台、任何期別都可直接套用的通用門檻。[FDA TMB-H indication](https://www.fda.gov/drugs/drug-approvals-and-databases/fda-approves-pembrolizumab-adults-and-children-tmb-h-solid-tumors)
- **MSI-H/dMMR** 是重要 tumor-agnostic biomarker，但實際適應症仍取決於 cancer type、stage、line of therapy、檢測方法與當地核准標示，不能簡化成「任何腫瘤都可使用 pembrolizumab」。

## Clinical pearl

- **開刀前想「這個腫瘤要不要做分子檢測」**:driver/免疫 biomarker 會改變治療順序(先系統治療或手術),漏檢會錯失最佳策略——關鍵 decision point。  
- **ctDNA 陽性 MRD 是復發前兆**:術後 ctDNA 可先於影像提示殘存/復發,正改變輔助治療與追蹤決策(仍在演進)。  
- **MSI-high 是跨癌別訊號**:任何實體瘤發現 dMMR/MSI-high,都要想到免疫治療。  
- **檢體品質決定檢測成敗**:不當固定/取樣會使分子檢測失敗,外科取材需符合分子病理要求。

## 快速比較表

| 檢測 | 偵測內容 | 臨床用途 |
| :---- | :---- | :---- |
| NGS panel | 突變/融合/TMB | 標靶/免疫選藥 |
| MMR/MSI | 錯配修復缺陷 | 免疫治療、Lynch 篩檢 |
| ctDNA(液態切片) | 血中腫瘤 DNA | MRD、監測、復發偵測 |
| 免疫染色(PD-L1 等) | 蛋白表現 | 免疫治療分層 |

| 分子標的 | 相關癌別(例) | 治療方向 |
| :---- | :---- | :---- |
| EGFR/ALK | 肺腺癌 | TKI |
| BRAF | 黑色素瘤/大腸 | BRAF/MEK 抑制 |
| NTRK 融合 | 跨癌別 | TRK 抑制劑 |
| dMMR/MSI-high | 跨癌別 | Checkpoint inhibitor |

## 後續證據更新

- **「tumor-agnostic(不論器官)」標靶治療成真**:larotrectinib(Drilon 等,NEJM 2018)顯示帶 **NTRK 基因融合** 的多種成人與兒童腫瘤,不論原發器官皆有高反應率,確立 **依分子標記而非器官** 的治療典範(FDA tissue-agnostic 核准)。**適用／限制**:僅適用具該分子改變者(NTRK 融合罕見),需分子檢測確認。  
- 相較 2019 教材,**ctDNA/MRD 於術後監測與輔助治療決策** 的角色快速擴展,可能改變「誰需要輔助治療、如何追蹤復發」。**適用／限制**:多數情境仍在臨床試驗驗證,尚未全面成為標準,需個別化與指引更新。
