# Chapter 29: Colon, Rectum, and Anus

## 核心整理

- 大腸血供分 **SMA（右側至橫結腸近端）** 與 **IMA（橫結腸遠端至直腸上段）**，交界如 splenic flexure 為 watershed（缺血好發）；直腸另有中/下直腸動脈供應，影響手術與吻合。  
- **憩室疾病（diverticulosis/diverticulitis）**：低纖維、壓力相關；acute diverticulitis 分 uncomplicated 與 complicated（膿瘍、穿孔、瘻管、阻塞）。多數 uncomplicated 可門診/內科處理，complicated 依 Hinchey 分級決定引流或手術。  
- **下消化道出血** 多源自憩室或血管發育不良（angiodysplasia）；多自限，處置含復甦、內視鏡、CT angiography/血管栓塞，必要時手術。  
- **發炎性腸病**：**Ulcerative colitis（連續性、黏膜層、直腸起始）** 與 **Crohn colitis（跳躍性、透壁性）**；UC 手術（total proctocolectomy）可治癒且降低長期癌變風險。  
- **大腸直腸癌（CRC）致病途徑**：adenoma-carcinoma sequence（APC→KRAS→TP53）、MSI/dMMR（含 Lynch）、鋸齒狀途徑；篩檢（大腸鏡等）可切除腺瘤預防癌症。  
- **CRC 分期以 TNM**，關鍵預後含 T、淋巴結、遠端轉移與分子特徵（MSI、KRAS/NRAS/BRAF，指引標靶/免疫）。  
- **結腸癌手術**：依血管分佈行節段性切除＋區域淋巴結廓清（建議 **≥12 顆**）；腹腔鏡與開腹腫瘤學結果相當。  
- **直腸癌處置核心是 TME（total mesorectal excision）**：完整切除直腸繫膜可大幅降低局部復發；依高度決定 LAR（保肛）或 APR（腹會陰切除、永久造口）。  
- **直腸癌 neoadjuvant**：局部進展（cT3–4 或 N+）多採術前放化療（或短程放療）以降期、降低局部復發。  
- **肛門疾病**：hemorrhoids（分級、依級別保守或手術）、anal fissure（後正中、括約肌高張，治療含鬆弛與局部藥物）、anorectal abscess/fistula（引流＋處理瘻管）。  
- **肛門癌（squamous cell）** 與 HPV 相關，治療以 **Nigro protocol（chemoradiation）** 為主而非首選手術，手術（APR）留給殘存/復發。  
- **腸阻塞/穿孔急症**：左側阻塞性癌可先支架或造口；穿孔性憩室炎/癌需手術與污染控制。

## Clinical pearl

- **直腸癌成敗看 TME 與環周切緣（CRM）**：不完整的繫膜切除或 CRM 陽性顯著增加局部復發——是直腸癌手術的核心 decision point。  
- **肛門鱗癌別一開始就 APR**：Nigro chemoradiation 為首選、可保肛，貿然手術是常見錯誤。  
- **年輕、多發或右側腫瘤想 Lynch/MSI**：影響手術範圍、家族篩檢與免疫治療決策，務必做 MMR/MSI。  
- **左側大腸阻塞注意近端缺血/穿孔**：閉環效應下盲腸擴張最危險，評估要涵蓋近端。

## 快速比較表

| 疾病 | 特徵 | 主要處置 |
| :---- | :---- | :---- |
| Uncomplicated diverticulitis | 局部發炎 | 內科（部分免抗生素） |
| Complicated diverticulitis | 膿瘍/穿孔/瘻管 | 引流/手術（依 Hinchey） |
| Ulcerative colitis | 連續、黏膜、直腸起 | 內科；手術可治癒 |
| Crohn colitis | 跳躍、透壁 | 保留腸段、處理併發症 |

| 腫瘤部位 | 手術原則 | 輔助/新輔助 |
| :---- | :---- | :---- |
| 結腸癌 | 節段切除＋≥12 淋巴結 | 化療（III 期等） |
| 直腸癌 | TME、LAR/APR | 常需 neoadjuvant CRT |
| 肛門鱗癌 | 手術非首選 | Nigro chemoradiation |

## 後續證據更新

- **局部進展直腸癌走向 total neoadjuvant therapy（TNT）**：RAPIDO（Lancet Oncol 2021）顯示 **短程放療後接術前化療再手術** 相較傳統長程放化療＋術後化療，降低 disease-related treatment failure 與遠端轉移、提高 pCR。**適用／限制**：適用高風險局部進展直腸癌；長期局部控制與各方案取捨仍在演進，需多科決策。  
- **dMMR/MSI-high 局部進展直腸癌免疫治療或可免手術**：單中心研究（Cercek 等，NEJM 2022）中 **dostarlimab** 治療 dMMR 局部進展直腸癌達到極高的臨床完全緩解、部分免除手術與放療。**適用／限制**：**僅 dMMR/MSI-high**、樣本小、隨訪仍在累積，屬 practice-changing 但尚需更大型驗證。  
- **III 期結腸癌輔助化療時程可個別化（IDEA）**：IDEA 合作分析（JCO 2018）顯示對 **低風險（T1–3 N1）** 者 **3 個月 CAPOX** 與 6 個月療效接近且神經毒性顯著較低。**適用／限制**：高風險（T4 或 N2）仍傾向 6 個月；方案（CAPOX vs FOLFOX）影響結論。
