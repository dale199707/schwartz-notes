# Chapter 13: Physiologic Monitoring of the Surgical Patient

## 核心整理

- 監測目的是 **早期偵測灌流/氧合不足** 並導引處置，核心在「組織是否得到足夠氧輸送（DO₂）以滿足需求」，而非單一數字。  
- **基本監測**：心電圖、非侵入性血壓、**pulse oximetry（SpO₂）**、體溫、尿量；圍手術期並監測 **end-tidal CO₂（capnography，確認氣管內管位置與通氣）**。  
- **Pulse oximetry 侷限**：低灌流、動作、指甲油、異常血紅素（**碳氧血紅素、變性血紅素**）會失準；CO 中毒時 SpO₂ 可假性正常，需 co-oximetry。  
- **動脈導管（arterial line）**：連續血壓與反覆採血/血氣；也可由動脈波形推估心輸出（脈波分析）。  
- **中心靜脈導管與 CVP**：CVP 為單點壓力，**對容量反應性預測能力有限**；現代偏好 **動態指標**。  
- **容量反應性評估**：**pulse pressure variation / stroke volume variation（正壓通氣、規律竇律下）**、**passive leg raise** 或補液試驗後的每搏量變化，優於單看 CVP。  
- **心輸出量監測**：從 pulmonary artery catheter（熱稀釋）到微創（脈波分析、食道超音波、生物阻抗）；選擇依臨床需求與侵入性權衡。  
- **組織灌流指標**：**lactate 與其清除、base deficit、central/mixed venous O₂ saturation（ScvO₂/SvO₂）**，反映全身氧供需平衡，用於復甦導引。  
- **神經監測**：重症或神經外科可用顱內壓（ICP）、腦灌流壓與（選擇性）腦氧監測。  
- **Goal-directed therapy**：以生理終點（每搏量、乳酸清除等）滴定輸液與升壓/強心，避免過度或不足復甦。  
- **監測的原則是「改變處置才有意義」**：侵入性監測本身不治療，須能導引具體介入且益處大於風險。

## Clinical pearl

- **CVP 不是可靠的容量指標**：以單一 CVP 數字決定補液常誤導；改用動態指標（PPV/SVV、passive leg raise）——重要 decision point。  
- **SpO₂ 正常不等於氧合/攜氧正常**：CO 中毒、低灌流、異常血紅素時會失準，臨床不符時查血氣/co-oximetry。  
- **lactate 趨勢可輔助評估復甦**：趨勢比單一數值更有意義；但 lactate 並非灌流的專一指標，應與 capillary refill、尿量及整體臨床狀態合併判讀。  
- **監測要能改變處置**：加裝侵入性監測前先問「結果會改變我的處置嗎」，否則徒增風險。

## 快速比較表

| 監測 | 反映 | 侷限 |
| :---- | :---- | :---- |
| Pulse oximetry | 動脈血氧飽和 | 低灌流、CO、動作失準 |
| Capnography | 通氣/導管位置 | 需正確取樣 |
| CVP | 單點靜脈壓 | 容量反應性差 |
| 動態指標（PPV/SVV） | 容量反應性 | 需正壓通氣＋竇律 |

| 灌流指標 | 意義 | 用途 |
| :---- | :---- | :---- |
| Lactate/清除 | 無氧代謝 | 復甦導引 |
| Base deficit | 代謝性酸中毒程度 | 出血/灌流評估 |
| ScvO₂/SvO₂ | 氧供需平衡 | 休克導引 |
| 尿量 | 腎灌流 | 簡易床邊指標 |

## 後續證據更新

- **常規肺動脈導管（PAC）未改善結局**：ESCAPE（Binanay 等，JAMA 2005）在嚴重症狀性心衰住院病人顯示，在臨床評估之外加 **PAC 導引治療未改善存活或住院天數，且增加院內不良事件**；此與其他重症族群結果一致，故現多以 **微創心輸出監測與動態指標** 取代，PAC 保留給特定複雜情形（如嚴重右心/肺高壓）。**適用／限制**：屬「去侵入化」趨勢；PAC 於選定病人仍有角色。  
- 相較 2019 教材，**以動態容量反應性與 goal-directed／limited 復甦** 導引輸液已更確立，避免以 CVP 為主的過度補液。**適用／限制**：動態指標有前提（正壓通氣、規律心律），心律不整/自主呼吸時需改用其他方法。
