# Chapter 13: Physiologic Monitoring of the Surgical Patient

## 核心整理

- 監測目的是 **早期偵測灌流/氧合不足** 並導引處置，核心在「組織是否得到足夠氧輸送（DO₂）以滿足需求」，而非單一數字。  
- **基本監測**：心電圖、非侵入性血壓、**pulse oximetry（SpO₂）**、體溫、尿量；圍手術期並監測 **end-tidal CO₂（capnography，確認氣管內管位置與通氣）**。  
- **Pulse oximetry 侷限**：低灌流、動作、指甲油、異常血紅素（**碳氧血紅素、變性血紅素**）會失準；CO 中毒時 SpO₂ 可假性正常，需 co-oximetry。  
- **動脈導管（arterial line）**：連續血壓與反覆採血/血氣；也可由動脈波形推估心輸出（脈波分析）。  
- **中心靜脈導管與 CVP**：CVP 為單點壓力，**對容量反應性預測能力有限**；現代偏好 **動態指標**。  
- **容量反應性評估**：PPV/SVV 只在 controlled mechanical ventilation、規律心律、足夠 tidal volume、無顯著右心衰竭/胸腹壓干擾等條件下可靠；低 tidal-volume ventilation 可考慮 tidal-volume challenge。Spontaneous breathing 或 arrhythmia 時優先用 **passive leg raise＋即時 stroke volume/CO 變化**或小量 fluid challenge。Fluid responsiveness 只表示 CO 可能上升，不等於病人一定需要或能安全承受輸液。  
- **心輸出量監測**：從 pulmonary artery catheter（熱稀釋）到微創（脈波分析、食道超音波、生物阻抗）；選擇依臨床需求與侵入性權衡。  
- **組織灌流指標**：serial lactate、base deficit、capillary refill、urine output、mental status 與 ScvO₂/SvO₂ 可提供互補資訊。Lactate 並非「無氧代謝專一指標」，可受 catecholamine、clearance、藥物等影響；ScvO₂/SvO₂ 正常或偏高也可能見於 extraction failure/distributive shock。  
- **神經監測**：重症或神經外科可用顱內壓（ICP）、腦灌流壓與（選擇性）腦氧監測。  
- **Goal-directed therapy**：對選定高風險手術，使用可重複評估的 flow/perfusion endpoint 滴定 fluid 與 vasoactive drug，可能降低部分併發症；不同 device、protocol 與族群異質，未一致證實 mortality benefit，且不可為追求單一數字造成 fluid overload 或不必要 inotrope。  
- **監測的原則是「改變處置才有意義」**：侵入性監測本身不治療，須能導引具體介入且益處大於風險。

## Clinical pearl

- **CVP 不是可靠的容量指標**：以單一 CVP 數字決定補液常誤導；改用動態指標（PPV/SVV、passive leg raise）——重要 decision point。  
- **SpO₂ 正常不等於氧合/攜氧正常**：CO 中毒、低灌流、異常血紅素時會失準，臨床不符時查血氣/co-oximetry。  
- **lactate 趨勢可輔助評估復甦**：趨勢比單一數值更有意義；但 lactate 並非灌流的專一指標，應與 capillary refill、尿量及整體臨床狀態合併判讀。  
- **監測要能改變處置**：加裝侵入性監測前先問「結果會改變我的處置嗎」，否則徒增風險。

## 快速比較表

| 監測 | 反映 | 侷限 |
| :---- | :---- | :---- |
| Pulse oximetry | 動脈血氧飽和 | 低灌流、CO、動作失準 |
| Capnography | 通氣/導管位置 | 需正確取樣 |
| CVP | 單點靜脈壓 | 容量反應性差 |
| 動態指標（PPV/SVV） | 容量反應性 | 需正壓通氣＋竇律 |

| 灌流指標 | 意義 | 用途 |
| :---- | :---- | :---- |
| Serial lactate | 氧供需、adrenergic drive 與 clearance 的綜合訊號 | 趨勢輔助；非灌流專一指標 |
| Base deficit | 代謝性酸中毒程度 | 出血/灌流評估 |
| ScvO₂/SvO₂ | 氧供需平衡 | 休克導引 |
| 尿量 | 腎灌流 | 簡易床邊指標 |

## 後續證據更新

- **PAC 不作常規，但不是被微創裝置全面取代**：ESCAPE 在特定 advanced heart-failure 族群未改善主要結局且增加不良事件；PAC 仍可用於 diagnosis/phenotyping 不確定、refractory or mixed shock、severe RV failure/pulmonary hypertension 或評估 MCS 的選定病人。重點是有明確決策問題與會解讀資料的團隊。  
- 相較 2019 教材，**以動態容量反應性與 goal-directed／limited 復甦** 導引輸液已更確立，避免以 CVP 為主的過度補液。**適用／限制**：動態指標有前提（正壓通氣、規律心律），心律不整/自主呼吸時需改用其他方法。
