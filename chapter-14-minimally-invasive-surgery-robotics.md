# Chapter 14: Minimally Invasive Surgery, Robotics, NOTES, and Single-Incision Surgery

## 核心整理

- **微創手術（MIS）** 以小切口、內視鏡與影像放大取代大切口，目標是 **降低傷口相關併發症、疼痛與住院、加速恢復**，同時維持等同的腫瘤學/治療結果。  
- **腹腔鏡生理學**：**CO₂ 氣腹（pneumoperitoneum）** 造成腹內壓上升，影響靜脈回流、心輸出、通氣（高碳酸血症）與尿量；對心肺功能受限者需留意。  
- **氣腹相關風險**：**CO₂ embolism、皮下氣腫、氣胸、腹壓對血流動力學影響**；建立氣腹（Veress/Hasson）與置放 trocar 時有血管/腸道傷害風險。  
- **能量器械**：electrocautery、advanced bipolar、ultrasonic（超音波刀）；需注意側向熱傷害與非目標組織灼傷。  
- **腹腔鏡已成多種手術標準**：cholecystectomy、appendectomy、抗逆流手術、bariatric、部分大腸與腫瘤手術；腫瘤手術須確保與開腹相當的切除品質。  
- **Robotic surgery（機器人手術）** 優勢：三維視野、可轉腕器械、震顫過濾、符合人體工學，利於 **狹小空間精細操作**（如骨盆、深部）；缺點為成本、缺乏觸覺回饋、設置時間。  
- **機器人手術證據取向**：多顯示與腹腔鏡結果 **相當**，優勢因手術與部位而異；不應假設「機器人一定更好」。  
- **NOTES（natural orifice transluminal endoscopic surgery）** 經自然孔道（胃、陰道等）進入腹腔，追求「無疤」；仍多屬研究/選擇性應用。  
- **Single-incision laparoscopic surgery（SILS）**：經單一切口（常經臍）操作，美容佳但器械擁擠、學習曲線；效益相對有限。  
- **學習曲線與訓練**：MIS/機器人有明確學習曲線，**模擬訓練與經驗量** 影響安全與結果。  
- **病人選擇與中轉開腹**：嚴重沾黏、解剖不清、出血或血流動力學不穩時，**適時轉開腹是安全決策而非失敗**。

## Clinical pearl

- **中轉開腹是判斷力、不是失敗**：解剖不清或出血控制困難時及時轉開腹，是保護病人的關鍵 decision point。  
- **氣腹對心肺受限者要當心**：高腹內壓與高碳酸血症會影響血流動力學與通氣，術前評估與術中溝通很重要。  
- **能量器械的側向熱傷害易被忽略**：非目標組織（如腸壁、膽管）的熱傷可能延遲表現，是可預防的陷阱。  
- **別假設「機器人一定更好」**：術式選擇應依實證與部位，而非新技術本身。

## 快速比較表

| 技術 | 主要優點 | 主要限制 |
| :---- | :---- | :---- |
| 腹腔鏡 | 恢復快、應用廣、成本較低 | 二維視野、器械自由度有限 |
| 機器人 | 3D、可轉腕、精細 | 成本高、無觸覺、設置久 |
| NOTES | 無外部疤 | 技術/器械未成熟 |
| SILS | 美容佳 | 器械擁擠、學習曲線 |

| 氣腹影響 | 機轉 | 臨床意義 |
| :---- | :---- | :---- |
| 血流動力學 | 腹壓↑、回流↓ | 低血壓、心輸出↓ |
| 呼吸 | CO₂ 吸收 | 高碳酸血症、需調整通氣 |
| 併發症 | CO₂ embolism、皮下氣腫 | 需警覺與監測 |

## 後續證據更新

- **早期子宮頸癌不宜常規用微創根治性子宮切除（LACC）**：LACC（Ramirez 等，NEJM 2018）意外顯示 **微創（含機器人）radical hysterectomy** 相較開腹，**無病存活與整體存活較差、復發較多**；促使指引改回以 **開腹** 為早期子宮頸癌標準。**適用／限制**：此為特定癌別的重要警訊，提醒「微創≠各癌別皆等效」，不可外推到所有手術。  
- **機器人對直腸癌未顯著優於腹腔鏡（ROLARR）**：ROLARR（Jayne 等，JAMA 2017）顯示機器人相較傳統腹腔鏡直腸癌手術，**未顯著降低中轉開腹率**或改善主要結局。**適用／限制**：反映機器人優勢因手術/部位而異，需以實證而非技術新穎性選擇；個別高難度骨盆情境仍可能受益。
