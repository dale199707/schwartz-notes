# Chapter 34: The Spleen

## 核心整理

- 脾臟功能：**免疫（清除被包膜細菌、產生 opsonin/antibody、含 marginal zone B cells）** 與 **過濾（清除老化紅血球、Howell-Jolly bodies、culling/pitting）**，並為血液儲備與髓外造血潛在場所。  
- 解剖上位於左上腹、第 9–11 肋下，與胃、胰尾、左腎、結腸脾曲相鄰；血供為 **splenic artery（腹腔幹分支）**，pancreatic tail 緊鄰脾門，手術時易傷。  
- **手術脾切除適應**：血液疾病（如頑固性 ITP、遺傳性球形紅血球症、某些溶血性貧血）、外傷、脾腫瘤/囊腫、脾動脈瘤、症狀性脾腫大或 hypersplenism。  
- **ITP（免疫性血小板低下）**：一線為 corticosteroid、IVIG；**脾切除** 為部分難治者選項，但因藥物（含 TPO receptor agonist、rituximab）進步而使用下降。  
- **創傷脾損傷**：血流動力學 **穩定** 者多可 **non-operative management（NOM）**（觀察 ± 血管栓塞），保留脾臟與免疫功能；不穩定者手術（修補或脾切除）。  
- **脾臟分級（AAST）** 協助評估，但處置主要依 **血流動力學與生理反應** 而非單看等級。  
- **Accessory spleen（副脾）** 常見；血液疾病做脾切除時需一併尋找移除，否則疾病復發。  
- **脾腫大鑑別廣**：感染、門脈高壓/充血、血液惡性（lymphoma、leukemia、myeloproliferative）、儲積病等；巨脾手術出血風險高。  
- **脾臟感染/膿瘍** 少見，處置為抗生素 ± 引流或脾切除。  
- **微創與保脾**：腹腔鏡脾切除為選擇性適應的標準；對部分病灶可 **partial splenectomy** 或栓塞以保留免疫功能（尤其兒童）。  
- **術後併發症**：出血、胰尾損傷（胰漏）、左下肺不張、**脾切除後血小板增多與血栓風險**、以及最嚴重的 **OPSI**。

## Clinical pearl

- **脾切除前務必疫苗＋衛教**：擇期手術應提前（理想術前 ≥2 週）接種 **encapsulated organisms 疫苗（pneumococcus、meningococcus、Hib）**，並教育發燒早就醫——預防 OPSI 的關鍵 decision point。  
- **OPSI 是終身風險、進展極快**：脾切除後的發燒不可輕忽，需低門檻經驗性抗生素，是不可錯過的 red flag。  
- **創傷穩定就保脾**：血流動力學穩定的脾損傷優先 NOM/栓塞以保留免疫，別急著切脾。  
- **血液病脾切除要找副脾**：遺漏 accessory spleen 是 ITP 等復發的常見陷阱。

## 快速比較表

| 情境 | 血流動力學 | 主要處置 |
| :---- | :---- | :---- |
| 創傷脾損傷 | 穩定 | NOM ± 血管栓塞 |
| 創傷脾損傷 | 不穩定 | 手術（修補/脾切除） |
| 難治性 ITP | — | 藥物優先，選擇性脾切除 |
| 遺傳性球形紅血球症 | — | 脾切除改善溶血 |

| 脾切除照護 | 內容 | 目的 |
| :---- | :---- | :---- |
| 疫苗 | Pneumococcus、meningococcus、Hib | 預防 OPSI |
| 衛教 | 發燒即就醫、備用抗生素 | 早期處置 |
| 血栓警覺 | 監測血小板 | 防栓塞 |
| 尋找副脾 | 術中檢查 | 防血液病復發 |

## 後續證據更新

- **ITP 一線與脾切除定位以 ASH 2019 指引為準**：相較 2019 教材，**American Society of Hematology 2019 ITP 指引**（Neunert 等，Blood Adv 2019）建議一線短療程 corticosteroid、避免長期類固醇，二線含 **TPO receptor agonist、rituximab 與脾切除** 並重共享決策，且對穩定的成人 ITP 傾向 **延後脾切除**（給疾病自然緩解時間）。**適用／限制**：多為 conditional 建議、certainty 有限，需個別化。  
- 創傷脾損傷持續朝 **NOM＋血管栓塞** 保脾方向；脾切除後 **OPSI 預防（疫苗、衛教、必要時預防性抗生素）** 為標準照護。**適用／限制**：NOM 需具備即時手術與監測後援；栓塞角色依中心經驗。
