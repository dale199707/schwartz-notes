# Chapter 51: Understanding, Evaluating, and Using Evidence for Surgical Practice

## 核心整理

- 實證外科是將 **最佳研究證據、臨床專業判斷與病人價值/偏好** 三者整合以做決策;證據是輔助而非取代臨床判斷。  
- **證據層級(evidence hierarchy)**:系統性回顧/統合分析與高品質 RCT 位於頂端,其下為世代/病例對照、病例系列與專家意見;但層級高不等於必然適用於眼前病人。  
- **外科研究的特殊挑戰**:難以盲化、有 **學習曲線**、術者與中心差異、器械持續演進、以及倫理上難以做假手術對照,使 RCT 較內科少而困難。  
- **隨機對照試驗(RCT)** 關鍵概念:隨機化(減少選擇偏差)、分派隱藏、意向治療分析(ITT)、盲化、樣本數/檢定力、主要 vs 次要終點。  
- **偏差(bias)與干擾(confounding)**:選擇、表現、偵測、流失偏差;觀察性研究需以設計/統計(如配對、調整、傾向分數)處理干擾,但殘餘干擾難免。  
- **結果判讀**:效應大小與 **信賴區間** 比單看 p 值重要;區分 **統計顯著 vs 臨床意義**;注意相對 vs 絕對風險與 NNT。  
- **外部效度(適用性)**:試驗族群、術者經驗與情境是否與自己的病人相符,影響能否外推。  
- **證據品質分級**:如 **GRADE** 系統依研究設計、一致性、直接性、精確度與偏差評估證據確定性與建議強度。  
- **臨床指引**:整合證據形成建議,但需注意利益衝突、時效與是否適用在地情境;指引是參考而非命令。  
- **從研究到實作**:知識轉譯(knowledge translation)與品質改善將證據落實到臨床,並以 audit 檢視成效。

## Codex 驗證補強（2026）

- Evidence hierarchy 是起點而非自動排名：有嚴重 bias、heterogeneity 或 indirectness 的 systematic review，不會因名稱就優於高品質、直接且可重現的研究。外科證據還要看 learning curve、surgeon/center volume、co-intervention 與 intervention fidelity。
- GRADE 將 **certainty of evidence** 與 **strength of recommendation** 分開；RCT 通常從 high certainty 起始、observational study 從 low 起始，再依 risk of bias、inconsistency、indirectness、imprecision、publication bias 等升降級。Strong recommendation 也不等於每位患者都必須照做。[Official GRADE Book](https://book.gradepro.org/guideline/overview-of-the-grade-approach)
- p value 不能表示 effect size、臨床重要性或「虛無假設為真的機率」；應一起看 absolute effect、confidence interval、baseline risk、multiplicity 與 missing data。
- Noninferiority trial 必須先檢查 margin 是否臨床合理、assay sensitivity、adherence/crossover，以及 intention-to-treat 與 per-protocol 是否一致；「未達顯著差異」不等於 noninferior。
- Surgical innovation 可依 IDEAL stages 從 Idea、Development、Exploration、Assessment 到 Long-term study 漸進評估；早期 case series 的可行性結果不能直接推成 comparative effectiveness。[IDEAL Collaboration](https://www.ideal-collaboration.net/)

## Clinical pearl

- **層級高的證據仍要問「適用嗎」**:RCT 族群/術者若與自己的病人差異大,外推需謹慎——關鍵判讀 decision point。  
- **看信賴區間,不只看 p 值**:效應大小與精確度比二元的「顯著與否」更有臨床意義。  
- **當心學習曲線效應**:新術式早期結果可能被學習曲線扭曲,評估需考慮術者經驗階段。  
- **統計顯著 ≠ 臨床重要**:大樣本可讓微小差異達統計顯著,須判斷是否對病人有意義。

## 快速比較表

| 研究設計 | 優點 | 侷限 |
| :---- | :---- | :---- |
| RCT | 減少選擇偏差、因果推論強 | 外科難盲化、成本、外部效度 |
| 世代研究 | 可研究預後/罕見暴露 | 干擾、失訪 |
| 病例對照 | 適合罕見結果 | 回憶/選擇偏差 |
| 病例系列 | 產生假說 | 無對照、偏差大 |

| 判讀概念 | 意義 |
| :---- | :---- |
| 意向治療(ITT) | 保留隨機化、反映真實 |
| 信賴區間 | 效應精確度 |
| 絕對風險/NNT | 臨床衝擊 |
| GRADE | 證據確定性與建議強度 |

## 後續證據更新

- **外科創新應遵循評估框架(IDEAL)**:IDEAL 框架(McCulloch 等;2019 Ann Surg 之演進版)提出外科創新的分期評估(Idea、Development、Exploration、Assessment、Long-term study),主張 **「沒有評估就不該創新」**,以減少未經證實的術式/器械擴散。**適用／限制**:為方法學建議,落實需期刊、學會與監管配合。  
- 相較 2019 教材,**登錄研究(registry)、真實世界證據(RWE)、以病人為中心的結果(PROMs)** 與嚴謹的外科試驗方法更受重視,補足傳統 RCT 的不足。**適用／限制**:RWE 有干擾風險,需審慎設計與判讀。
