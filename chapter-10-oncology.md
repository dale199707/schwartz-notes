# Chapter 10: Oncology

## 核心整理

- 癌症是 **多步驟基因體疾病**：累積 driver mutation 使細胞獲得 Hanahan-Weinberg 提出的 **hallmarks**（持續增殖訊號、逃避生長抑制、抗凋亡、無限複製、血管新生、侵襲轉移，及後續補充的免疫逃避、代謝重編程等）。  
- 基因層面分 **oncogene（功能獲得，顯性，如 KRAS、HER2、MYC）** 與 **tumor suppressor（功能喪失，通常需 two-hit，如 TP53、RB、APC）**；另有 DNA 修復/mismatch repair 基因缺陷。  
- **遺傳性癌症症候群** 影響外科決策：**BRCA1/2**（乳癌/卵巢癌，考慮 risk-reducing 手術）、**Lynch syndrome/HNPCC**（MMR 缺陷、大腸直腸與子宮內膜癌）、**FAP（APC）**、MEN、Li-Fraumeni 等。  
- **癌症分期以 TNM 為主**（primary Tumor、Node、Metastasis），決定治療與預後；病理分期（p）較臨床分期（c）精確。  
- **腫瘤外科原則**：達成 **negative margin（R0）** 的 en-bloc 切除、適當的 **淋巴結廓清/評估**、避免腫瘤破裂與播種；**sentinel lymph node biopsy** 於乳癌與黑色素瘤可在減少廓清併發症下取得分期資訊。  
- **多模式治療**：手術、放療、化療、標靶、免疫治療的組合；**neoadjuvant（術前）** 可縮小腫瘤、評估反應、提高可切除性，**adjuvant（術後）** 降低復發。  
- **化療** 依細胞週期作用機轉分類，毒性具劑量限制性（骨髓抑制、黏膜炎、神經毒性等）。  
- **標靶治療** 針對特定分子驅動（如 HER2、EGFR、ALK、BRAF、KIT）；使用前需 **biomarker 檢測**，療效與抗藥性和驅動基因狀態相關。  
- **免疫治療**：**immune checkpoint inhibitors（anti-PD-1/PD-L1、anti-CTLA-4）** 解除 T 細胞抑制，改變多種實體癌治療；具特殊 **immune-related adverse events（irAE）**（腸炎、肺炎、內分泌病變等）。  
- **預測性 biomarker** 指引免疫/標靶用藥：**PD-L1 表現、tumor mutational burden（TMB）、MSI-high/dMMR**（後者為跨癌別免疫治療反應指標）。  
- **腫瘤篩檢** 針對高風險族群（如大腸鏡、乳房攝影、低劑量肺 CT、子宮頸抹片），目標為早期偵測降低死亡率；須權衡假陽性與過度診斷。  
- **腫瘤標記與影像**（CEA、CA19-9、AFP、PSA 等）用於監測與復發追蹤，多非診斷用；PET-CT 協助分期與偵測轉移。

## Clinical pearl

- **開刀前先分期、驗 biomarker**：分子檢測（MMR/MSI、驅動基因）會改變是否先做 neoadjuvant 或系統治療——跳過會錯失最佳順序，是關鍵 decision point。  
- **免疫治療的副作用像自體免疫**：irAE（腹瀉/腸炎、呼吸困難/肺炎、內分泌異常）表現多樣，早辨識、適時類固醇；當成單純感染處理是常見陷阱。  
- **Margin 與 en-bloc 決定腫瘤手術成敗**：為了縮小切口而犧牲 R0 或造成腫瘤破裂，會顯著惡化預後。  
- **dMMR/MSI-high 是跨癌別的免疫治療訊號**：任何實體瘤發現 MSI-high，都要想到 checkpoint inhibitor 的角色。

## 快速比較表

| 基因類別 | 機轉 | 例子 | 外科意涵 |
| :---- | :---- | :---- | :---- |
| Oncogene | 功能獲得（顯性） | KRAS、HER2、MYC | 標靶治療對象 |
| Tumor suppressor | 功能喪失（two-hit） | TP53、APC、RB | 遺傳症候群、預防手術 |
| MMR 缺陷 | 錯配修復失能 | Lynch（MLH1 等） | MSI-high、免疫治療反應 |

| 治療模式 | 時機/角色 | 代表 |
| :---- | :---- | :---- |
| 手術 | 局部根治 | R0 en-bloc、SLNB |
| Neoadjuvant | 術前縮瘤/評估反應 | 化放療、免疫 |
| Adjuvant | 術後降復發 | 化療、標靶 |
| 免疫/標靶 | 依 biomarker | Checkpoint inhibitor、TKI |

## 後續證據更新

- **dMMR/MSI-high 轉移大腸癌一線改用免疫治療**：KEYNOTE-177（NEJM 2020）顯示 **pembrolizumab** 相較化療顯著延長 MSI-high/dMMR 轉移大腸癌的 progression-free survival 並毒性較低，確立 checkpoint inhibitor 為此族群一線。**適用／限制**：僅適用 **MSI-high/dMMR** 者，須先做 MMR/MSI 檢測；MSS 腫瘤不適用。  
- **免疫檢查點抑制劑已擴展至多癌別的圍手術期治療**：相較 2019 教材，checkpoint inhibitor 從晚期擴及可切除疾病的 neoadjuvant/adjuvant（例如肺、食道、黑色素瘤等），改變手術與系統治療的整合順序（詳見各器官章）。**適用／限制**：適應症與 biomarker 依癌別而異，須多科團隊與分子檢測支持。
