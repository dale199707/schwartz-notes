# Chapter 35: Abdominal Wall, Omentum, Mesentery, and Retroperitoneum

## 核心整理

- 前腹壁結構由外而內：皮膚、Scarpa 筋膜、肌層（external/internal oblique、transversus、rectus abdominis）與 **rectus sheath、linea alba、腹膜**；弓狀線（arcuate line）以下後鞘缺如，影響手術層次與補片放置。  
- **Ventral/incisional hernia**：腹壁筋膜缺損致內容物突出；**incisional hernia** 多因先前正中切口癒合不良（感染、肥胖、糖尿病、營養不良、術後咳嗽/腹壓）。  
- **臨床評估**：可復位性、缺損大小與位置、皮膚狀況、是否嵌頓/絞窄；大型或複雜疝需 CT 評估缺損與 **loss of domain**。  
- **修補原則**：多數 **midline incisional hernia** 以 mesh 補強可降低復發；但極小的 selected primary umbilical/epigastric defect 可依大小、病人風險與共享決策考慮縫合，不能概括成所有腹壁疝都一定要 mesh。  
- **Retromuscular（sublay / Rives-Stoppa）** 是多數 midline incisional hernia 優先考慮的補片平面；能安全完成時應關閉 fascial defect、恢復中線，避免以 mesh bridging 取代筋膜併合。  
- **Component separation**（前分離 anterior 或後分離 transversus abdominis release, TAR）用於 **大型缺損** 以達 **中線筋膜無張力併合**與功能重建。  
- **急性併發症**：**incarceration（嵌頓，不可復位）** 與 **strangulation（絞窄，缺血）**——後者是外科急症（劇痛、壓痛、皮膚變化、腸阻塞/腹膜炎徵象），需緊急手術。  
- **污染/感染情境** 需區分 clean-contaminated、contaminated 與 active uncontrolled infection。現有 randomized evidence 顯示，在可單期重建的 clean-contaminated/contaminated ventral hernia，永久 synthetic mesh 的 2 年復發低於 biologic mesh，安全結果相近；不能因「有污染」便自動改用昂貴 biologic mesh。  
- **其他腹壁病灶**：**desmoid tumor（韌帶樣纖維瘤，侵襲性但不轉移，與 FAP/APC 相關）**、rectus sheath hematoma（抗凝、咳嗽，多保守）、臍/上腹疝。  
- **Omentum/mesentery**：omental infarction、mesenteric cyst、**mesenteric panniculitis/sclerosing mesenteritis**（多影像診斷、常保守）。  
- **Retroperitoneum**：**retroperitoneal sarcoma**（多為 liposarcoma、leiomyosarcoma）以 **完整 en-bloc 切除（常含受累鄰近器官）** 為主；lymphoma、germ cell 轉移等鑑別需切片與多科評估。

## Clinical pearl

- **絞窄疝是不能等的急症**：疝不可復位＋劇痛、壓痛、皮膚變色或腸阻塞徵象，需緊急手術評估腸活性——關鍵 red flag。  
- **切口疝多數要補片，但不是每個小缺損都一樣**：midline incisional hernia 通常以 mesh、優先 retromuscular plane，並盡可能完成 fascial closure；小型 primary hernia 則依缺損與病人風險個別化。  
- **retroperitoneal sarcoma 不要「先切一塊看看」再說**：計畫性 en-bloc 完整切除比破壞後再處理預後好得多；先影像＋core biopsy 與多科規劃。  
- **desmoid 侵襲卻不轉移**：處置已趨向主動監測/系統治療，非一律積極手術（易復發），需個別化。

## 快速比較表

| 補片放置層 | 位置 | 特點 |
| :---- | :---- | :---- |
| Onlay | 筋膜表面 | 技術簡單、皮下併發症較多 |
| Sublay/retromuscular | 肌後、腹膜前 | 耐久、復發低（Rives-Stoppa） |
| Underlay/IPOM | 腹腔內 | 需防腸沾黏面 |
| Component separation | 加做筋膜釋放 | 用於大缺損中線併合 |

| 腹壁/腹膜後病灶 | 性質 | 主要處置 |
| :---- | :---- | :---- |
| Midline incisional hernia | 筋膜缺損 | 多數 mesh；優先 retromuscular＋fascial closure |
| Desmoid tumor | 侵襲不轉移 | 監測/系統治療/選擇性手術 |
| Rectus sheath hematoma | 出血 | 多保守 |
| Retroperitoneal sarcoma | 惡性 | En-bloc 完整切除 |

## 後續證據更新

- **高風險正中剖腹「預防性補片」可降低切口疝**：PRIMA（Jairam 等，Lancet 2017）顯示對高風險（如 AAA、肥胖）正中剖腹關腹時 **預防性放置補片（尤其 sublay）**，相較單純縫合顯著降低 2 年 incisional hernia 發生率。**適用／限制**：效益見於高風險族群；補片相關併發症與成本需權衡，並非所有剖腹皆需。  
- **EHS 2023 midline incisional hernia guideline** 建議 mesh repair，並優先考慮 retromuscular plane；若技術可行，應關閉 defect、避免 bridging。**適用／限制**：指引主要涵蓋可在不採 advanced component separation 下關閉的成人 midline incisional hernia。  
- **污染術野不是 biologic mesh 的同義詞**：Rosen 等 RCT 顯示，clean-contaminated/contaminated 單期修補中 synthetic mesh 的 2 年 recurrence 較低，而 safety outcome 相近。**適用／限制**：結果來自具專業經驗中心，不可外推至未控制 infection 或無法充分 debridement 的情境。  
- **Desmoid tumor 已轉向先監測**：對無急迫危害的病人多先 active surveillance；成人若 disease progression 且需 systemic therapy，phase 3 DeFi trial 顯示 **nirogacestat** 可改善 progression-free survival 與症狀。**適用／限制**：需專科中心確認診斷、症狀與治療時機。
