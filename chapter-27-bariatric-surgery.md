# Chapter 27: The Surgical Management of Obesity

## 核心整理

- **肥胖是慢性代謝疾病**，與 type 2 diabetes、高血壓、血脂異常、OSA、NAFLD、心血管疾病與多種癌症相關；BMI 為主要分級指標（合併亞洲族群較低門檻的考量）。  
- **手術（metabolic/bariatric surgery）適應（傳統）**：**BMI ≥40**，或 **BMI ≥35 合併重大共病**；且經正規減重仍無法達成/維持。近年學會建議對合併 T2DM 者放寬門檻。  
- 減重手術機轉不只「限制與吸收不良」，更透過 **腸道荷爾蒙（GLP-1、PYY、ghrelin 變化）與膽酸/菌叢** 改善代謝，故稱 **metabolic surgery**。  
- **Sleeve gastrectomy（袖狀胃切除）**：切除大彎胃約 80%，限制容量並降 ghrelin；目前最常見、技術較單純、無腸道吻合，但可能加重/誘發 GERD。  
- **Roux-en-Y gastric bypass（RYGB）**：小胃囊＋腸道改道，減重與代謝（尤其 GERD、糖尿病）效果佳，但有吻合、內疝、傾食症候群、微量營養素缺乏等風險。  
- **其他術式**：adjustable gastric band（已少用）、biliopancreatic diversion/duodenal switch（減重最強但營養併發症高，選擇性使用）。  
- **糖尿病改善/緩解** 常在顯著減重前即發生，反映神經內分泌機轉；手術對 T2DM 控制優於單純藥物（見後續證據）。  
- **術前評估**：多科（營養、心理、內科共病）評估，戒菸，篩 OSA，衛教長期行為改變與追蹤依從性。  
- **早期併發症**：吻合/釘合線 **leak（心搏過速、發燒常為早期唯一徵象）**、出血、VTE；leak 是最危險併發症之一。  
- **晚期併發症**：狹窄、邊緣潰瘍、內疝（RYGB，可致絞窄性阻塞）、膽結石、傾食症候群、**營養缺乏（鐵、B12、鈣、維生素 D、thiamine）** 需終身補充與監測。  
- **長期照護**：規律追蹤體重、共病、營養標記與心理支持；體重復胖或共病復發需再評估。

## Clinical pearl

- **術後持續心搏過速就當 leak 處理**：bariatric 術後 tachycardia 常是釘合線滲漏的最早、有時唯一徵象——不可等到明顯腹膜炎，是關鍵 red flag。  
- **RYGB 後腹痛想內疝**：即使影像不明顯，RYGB 後陣發性腹痛/阻塞要高度警覺 internal hernia，延遲可致腸絞窄壞死。  
- **sleeve 與嚴重 GERD 要謹慎**：明顯 GERD/Barrett 者 RYGB 可能較適合，選錯術式會惡化逆流。  
- **終身補充微量營養素**：忽略 B12、鐵、鈣、維生素 D、thiamine 補充是可預防的長期陷阱（thiamine 缺乏可致 Wernicke）。

## 快速比較表

| 術式 | 機轉 | 優點 | 主要風險 |
| :---- | :---- | :---- | :---- |
| Sleeve gastrectomy | 限制、降 ghrelin | 單純、無腸吻合 | 可能加重 GERD |
| RYGB | 限制＋改道、荷爾蒙 | 減重/代謝/GERD 佳 | 內疝、傾食、營養缺乏 |
| Adjustable band | 限制 | 可調、可逆 | 效果有限、已少用 |
| BPD/duodenal switch | 強吸收不良 | 減重最強 | 營養併發症高 |

| 併發症 | 時機 | 關鍵徵象 |
| :---- | :---- | :---- |
| Leak | 早期 | 心搏過速、發燒 |
| VTE | 早期 | 呼吸困難、腫脹 |
| Internal hernia（RYGB） | 晚期 | 陣發腹痛、阻塞 |
| 營養缺乏 | 晚期 | 貧血、神經症狀 |

## 後續證據更新

- **代謝手術對 T2DM 控制優於單純藥物（STAMPEDE 5 年）**：STAMPEDE（Schauer 等，NEJM 2017，5 年結果）顯示對肥胖合併 type 2 diabetes 者，**手術（RYGB 或 sleeve）＋藥物** 相較 **強化藥物治療**，達成血糖目標的比例顯著較高、用藥更少、生活品質更佳。**適用／限制**：研究族群為特定 BMI 與糖尿病者；長期仍需追蹤共病與營養，非人人適用。  
- **趨勢：從「bariatric」到「metabolic surgery」並放寬適應**：相較 2019 教材，多個學會已建議對合併 T2DM 者 **降低手術 BMI 門檻**，強調代謝效益。**適用／限制**：仍需多科評估與長期追蹤依從性；門檻依各學會與族群（含亞洲較低 BMI）調整。
