# Chapter 50: Optimizing Perioperative Care — Enhanced Recovery

## 核心整理

- **Enhanced Recovery After Surgery(ERAS)** 是以實證的 **多項圍手術期措施組合(bundle)**,目標為減輕手術壓力反應、加速功能恢復、降低併發症與縮短住院。  
- 核心理念:以 **一連串小的實證介入的累積效應** 取代單一措施;成效與 **整體遵從度(compliance)** 高度相關。  
- **術前**:衛教與期望設定、避免長時間禁食、**術前 2 小時可清流質/碳水飲品(carbohydrate loading)**、戒菸戒酒、營養與貧血優化、prehabilitation。  
- **術中**:微創術式、**避免過量輸液(goal-directed / zero-balance)**、保溫(維持正常體溫)、短效麻醉藥、預防 PONV。  
- **鎮痛採多模式、opioid 節制**:區域麻醉/局部浸潤＋acetaminophen＋NSAID,減少 opioid 相關腸麻痺與副作用。  
- **管路極簡化**:盡量避免或及早移除 **鼻胃管、引流管與導尿管**,降低感染與不動。  
- **術後早期進食與活動**:術後早期恢復口服飲食與 **早期下床活動**,促進腸功能與減少併發症(肺、VTE、肌肉流失)。  
- **預防併發症的標準措施**:VTE 預防、抗生素預防時機、血糖控制、SSI 預防組合。  
- **審計與回饋(audit)**:ERAS 需持續量測遵從度與結果並回饋改善,是計畫成功的關鍵環節。  
- **跨團隊協作**:外科、麻醉、護理、營養、復健與病人共同參與;病人是主動參與者而非被動接受者。  
- **可推廣性**:ERAS 原則已從大腸手術擴展到多種手術領域,但需依手術特性調整。

## Codex 驗證補強（2026）

- ERAS 是 procedure-specific、multidisciplinary pathway，不是所有患者套同一張 checklist。執行時應追蹤 compliance 與 outcome，並為 frailty、malnutrition、obstruction、organ failure 與 emergency surgery 調整。
- Clear fluid 至 2 小時與 carbohydrate loading 只適合選定 elective、低 aspiration-risk 患者；diabetes、gastroparesis、bowel obstruction、active GI symptom 或高風險 GLP-1 使用者需個別化，carbohydrate drink 不是 universal requirement。
- **Goal-directed fluid therapy** 與「zero balance」不是同義詞。目標是避免 hypovolemia 與 salt/water overload，以 clinical context、dynamic response、urine output 趨勢與 organ perfusion 個別化；單一監測值不能取代整體判斷。
- Multimodal analgesia 應減少 opioid，但 NSAID/COX-2 inhibitor 要考慮 renal function、bleeding、cardiovascular risk 與 procedure-specific anastomotic concern；不要為達 opioid-free 而增加其他傷害。
- Early feeding、mobilization 與移除 tube/drain 通常有益，但 nasogastric tube、urinary catheter、drain 的時機依手術與 complication risk 調整，不以「全部 POD1 拔除」作僵化品質指標。[ASCRS/SAGES ERAS guideline](https://pubmed.ncbi.nlm.nih.gov/36515747/)

## Clinical pearl

- **ERAS 的效果來自「整套遵從」**:單挑幾項做、其他忽略,效益會打折——遵從度是核心 decision point。  
- **少管路、早進食、早下床**:這三件事看似簡單,卻是降低併發症與加速恢復的骨幹。  
- **避免過量輸液**:術中/術後液體過量延遲腸功能與增加併發症,採目標導向的「等容量」而非大量補給。  
- **術前碳水飲品可安全給到術前 2 小時**:改善舒適與代謝,打破「一律午夜後禁食」的舊習。

## 快速比較表

| 階段 | 傳統做法 | ERAS 做法 |
| :---- | :---- | :---- |
| 術前禁食 | 午夜後全禁 | 術前 2 小時清流質/碳水 |
| 輸液 | 較寬鬆/大量 | 目標導向、避免過量 |
| 鎮痛 | opioid 為主 | 多模式、opioid 節制 |
| 術後 | 延遲進食/臥床 | 早期進食、早下床 |

| 措施 | 目的 |
| :---- | :---- |
| 保溫 | 減少感染/凝血/心臟事件 |
| 少管路 | 降感染、促活動 |
| PONV 預防 | 加速進食恢復 |
| Audit/回饋 | 持續改善遵從度 |

## 後續證據更新

- **ERAS 已有正式的實證指引並成為標準**:ERAS® Society 的圍手術期照護指引(例如 Gustafsson 等,擇期大腸手術指引,World J Surg 2019)整合各項措施的實證與建議,**採用 ERAS 與更高遵從度可降低併發症與住院天數**。**適用／限制**:多為中低品質證據的組合建議,需依手術別調整;真正效益取決於落實與 audit。  
- 相較 2019 教材,ERAS 從大腸手術 **擴展至肝膽、婦科、泌尿、胸腔、骨科等多領域**,並更強調 **prehabilitation 與 opioid 節制**。**適用／限制**:各領域證據成熟度不一,需個別化與在地流程建置。
