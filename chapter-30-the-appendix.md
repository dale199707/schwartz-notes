# Chapter 30: The Appendix

## 核心整理

- 闌尾為盲腸底部的管狀構造，基部三條 taenia coli 匯聚處為定位標誌；位置多變（retrocecal 最常見），影響症狀表現與診斷難度。  
- **Acute appendicitis** 機轉多為 **管腔阻塞（fecalith、淋巴增生等）→ 內壓上升 → 缺血 → 細菌增生 → 發炎/壞死/穿孔**。  
- **典型臨床病程**：先臍周內臟痛（referred），隨壁層腹膜受累 **轉移至右下腹 McBurney 點**；伴厭食、噁心、低度發燒。  
- **理學徵象**：McBurney 壓痛、rebound、Rovsing、psoas、obturator sign；穿孔後可轉為瀰漫性腹膜炎。  
- **診斷輔助**：白血球/CRP、臨床評分（如 Alvarado）；影像以 **CT**（成人準確度高）為主，兒童/孕婦優先 **超音波** 減少輻射，必要時 MRI。  
- **Uncomplicated vs complicated**：後者含穿孔、膿瘍、壞疽；complicated 常需不同處置路徑（引流、延遲手術）。  
- **標準治療為 appendectomy**（腹腔鏡為主流，恢復快、傷口併發症少）；穿孔合併局限膿瘍者可先 **經皮引流＋抗生素**，是否 interval appendectomy 需個別化。  
- **抗生素治療** 對選定 uncomplicated appendicitis 是可行替代（見後續證據），但有復發風險。  
- **闌尾腫瘤**（術後病理常意外發現）：**neuroendocrine tumor（carcinoid，最常見，多位於尖端、\<1–2 cm 多可闌尾切除治癒）**、mucinous neoplasm（LAMN，可致 pseudomyxoma peritonei）、adenocarcinoma。  
- **Appendiceal mucocele/黏液性腫瘤** 應避免術中破裂（防腹膜播散）；破裂或 pseudomyxoma peritonei 需 cytoreduction ± HIPEC 之專門處置。  
- **特殊族群**：孕婦（最常見的非產科急腹症，延遲手術增加穿孔與胎兒風險）、老人（表現不典型、穿孔率高）。

## Clinical pearl

- **闌尾切除病理見 mucinous neoplasm 要轉介**：LAMN 有 pseudomyxoma peritonei 風險，需專門評估——別當一般良性闌尾處理。  
- **孕婦疑闌尾炎別因怕輻射而拖延**：優先超音波/MRI，但診斷不明且臨床強烈懷疑時，延遲手術的穿孔風險高於手術本身——關鍵 decision point。  
- **膿瘍優先引流、避免立即困難手術**：穿孔合併局限膿瘍者，經皮引流＋抗生素常較急做困難闌尾切除安全。  
- **老人不典型、警覺穿孔**：症狀輕微不代表病程輕，穿孔率與併發症在長者更高，是易被低估的 red flag。

## 快速比較表

| 類別 | 特徵 | 主要處置 |
| :---- | :---- | :---- |
| Uncomplicated | 局限發炎、未穿孔 | 闌尾切除（或選擇性抗生素） |
| Complicated（穿孔） | 瀰漫腹膜炎 | 手術＋沖洗 |
| 合併局限膿瘍 | 已包覆 | 經皮引流＋抗生素 ± interval |
| 闌尾腫瘤 | 病理意外發現 | 依型別（NET/黏液/腺癌） |

| 族群 | 診斷首選影像 | 注意 |
| :---- | :---- | :---- |
| 成人 | CT | 準確度高 |
| 兒童 | 超音波 | 減少輻射 |
| 孕婦 | 超音波→MRI | 避免延遲 |
| 老人 | CT | 表現不典型、穿孔率高 |

## 後續證據更新

- **無併發闌尾炎可考慮「抗生素優先」**：CODA（NEJM 2020）顯示成人 uncomplicated appendicitis 抗生素治療對 30 天健康狀態 **non-inferior**，但約 **四分之一在一年內仍接受闌尾切除**，且有 **appendicolith** 者失敗與併發症較高。**適用／限制**：適合選定、充分告知復發風險者；有 appendicolith、穿孔或膿瘍者較不適合。  
- 相較 2019 教材，強化了對 uncomplicated appendicitis 的 **shared decision-making**（手術 vs 抗生素）與影像分級（是否 complicated）引導處置。**適用／限制**：長期復發與成本／生活品質取捨仍需個別化。
