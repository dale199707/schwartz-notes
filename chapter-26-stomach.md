# Chapter 26: Stomach

## 核心整理

- 胃血供豐富（left/right gastric、left/right gastroepiploic、short gastrics），故切除後常可靠側枝維持吻合灌流；淋巴引流沿主要血管分站，是 D1/D2 廓清的解剖基礎。  
- **胃酸分泌** 由 parietal cell 經 **H⁺/K⁺-ATPase** 進行，受 gastrin、histamine（ECL 細胞）、acetylcholine 調控；PPI 抑制質子泵是治療基石。  
- **Peptic ulcer disease** 兩大主因：**H. pylori 感染** 與 **NSAID**；治療含 PPI、根除 H. pylori、停 NSAID。多數潰瘍已內科化，手術多留給併發症。  
- **潰瘍併發症**（外科重點）：**perforation（游離氣體、腹膜炎，需手術修補如 Graham patch）、bleeding（內視鏡止血為先）、gastric outlet obstruction、intractability**。  
- **消化道出血**：上消化道出血先復甦、PPI、內視鏡止血（注射/熱凝/夾）；失敗者血管栓塞或手術；後壁十二指腸潰瘍侵蝕 gastroduodenal artery 為大出血典型。  
- **胃癌（adenocarcinoma）** 危險因子：H. pylori、慢性萎縮性胃炎/腸上皮化生、高鹽/醃漬飲食、吸菸、家族史（含 **CDH1 遺傳性瀰漫型胃癌**）。Lauren 分型分 intestinal 與 diffuse（含 signet-ring、linitis plastica）。  
- **胃癌分期與可切除性**：EUS（T、N）、CT/PET、**diagnostic laparoscopy＋腹腔沖洗細胞學** 排除腹膜轉移；分期決定手術 vs 系統治療。  
- **胃癌手術**：依位置行 subtotal 或 total gastrectomy，達 **adequate margin＋D2 lymphadenectomy**（東亞標準、西方漸採），重建以 Roux-en-Y 為主。  
- **早期胃癌（局限黏膜、無不良特徵）** 可考慮 **endoscopic resection（EMR/ESD）**，避免大手術。  
- **GIST（gastrointestinal stromal tumor）**：源自 Cajal 細胞、多帶 **KIT/PDGFRA** 突變；處置為 **R0 切除（不需大範圍淋巴廓清）**，中高風險或不可切除者用 **imatinib**（酪胺酸激酶抑制劑）。  
- **其他胃腫瘤**：MALT lymphoma（與 H. pylori 相關，根除菌可能消退）、neuroendocrine tumor（type 依 gastrin/萎縮性胃炎背景分類）。  
- **術後問題**：dumping syndrome、貧血（B12、鐵吸收）、營養不良；需長期補充與飲食調整。

## Clinical pearl

- **胃潰瘍要切片、十二指腸潰瘍通常不用**：胃潰瘍可能是惡性，需內視鏡切片並追蹤癒合——漏做是重要陷阱。  
- **胃癌開刀前先腹腔鏡分期**：CT 可能低估腹膜轉移，diagnostic laparoscopy＋沖洗細胞學能避免不必要的剖腹——關鍵 decision point。  
- **GIST 不做淋巴廓清、避免破裂**：以 R0 切除為目標、保護假包膜、避免腫瘤破裂（增加腹膜播種）；治療核心是分子（KIT）而非廣泛淋巴清掃。  
- **穿孔性潰瘍是外科急症**：突發劇痛＋游離氣體要儘速手術，延遲致瀰漫性腹膜炎與敗血症的 red flag。

## 快速比較表

| 潰瘍併發症 | 首選處置 | 備註 |
| :---- | :---- | :---- |
| Perforation | 手術修補（Graham patch） | 游離氣體、腹膜炎 |
| Bleeding | 內視鏡止血＋PPI | 失敗→栓塞/手術 |
| Outlet obstruction | 減壓、內視鏡/手術 | 慢性瘢痕 |
| Intractability | 評估惡性、手術 | 排除胃癌 |

| 胃腫瘤 | 來源/關聯 | 主要處置 |
| :---- | :---- | :---- |
| Adenocarcinoma | H. pylori、萎縮性胃炎 | 胃切除＋D2、圍手術期化療 |
| GIST | Cajal 細胞、KIT/PDGFRA | R0 切除、imatinib |
| MALT lymphoma | H. pylori | 根除菌 ± 治療 |
| Early gastric cancer | 局限黏膜 | EMR/ESD |

## 後續證據更新

- **可切除胃／GE junction 腺癌圍手術期化療改用 FLOT**：FLOT4（Al-Batran 等，Lancet 2019）顯示 **perioperative FLOT（docetaxel＋oxaliplatin＋5-FU/leucovorin）** 相較舊有 ECF/ECX 顯著延長 overall survival，成為體能佳者的新標準。**適用／限制**：適合 **ECOG 體能佳、可耐受三藥** 者；毒性較高，需多科評估。  
- **早期胃癌內視鏡切除（ESD）與 D2 廓清為現行方向**：相較部分西方舊觀念，符合條件的早期胃癌以 ESD 可達治癒且保留胃；進展期採 D2 lymphadenectomy 已被接受為標準。**適用／限制**：ESD 僅適用嚴格條件（腫瘤大小、深度、無潰瘍/脈管侵犯）；不符者仍需手術。
