# Chapter 19: Chest Wall, Lung, Mediastinum, and Pleura

## 核心整理

- **孤立性肺結節（solitary pulmonary nodule, SPN）** 評估看惡性風險：年齡、吸菸史、大小、邊緣（spiculated 高風險）、成長速度、位置（上葉較高風險）。穩定 ≥2 年的實質結節多屬良性；依 **Fleischner** 準則決定追蹤或切片。  
- **肺癌組織學**：**non-small cell lung cancer（NSCLC，約 85%）** 含 adenocarcinoma（最常見、與非吸菸者相關）、squamous cell（中央型、與吸菸強相關）、large cell；**small cell lung cancer（SCLC）** 惡性度高、早期轉移、以化放療為主而非手術。  
- **NSCLC 分期（TNM）** 決定治療：早期（I–II、部分 IIIA）以手術為主；分期需含 PET-CT 與 **縱膈淋巴結評估**（EBUS-TBNA 或 mediastinoscopy），因 N2/N3 影響可切除性甚鉅。  
- **可手術性 vs 可切除性** 兩概念分開評估：可切除性看解剖分期；可手術性看 **心肺功能儲備**（predicted postoperative FEV₁、DLCO、必要時 exercise testing）。  
- **肺癌標準切除** 為 **lobectomy＋systematic mediastinal lymph node dissection/sampling**；lobectomy 長期優於 wedge（歷史 LCSG 資料）。近端/跨葉腫瘤可能需 pneumonectomy 或 sleeve resection（保留肺實質）。  
- **驅動基因與免疫檢查**：進行性/轉移 NSCLC（尤其 adenocarcinoma）應檢測 **EGFR、ALK、ROS1** 等 driver mutation 與 **PD-L1**，以決定 targeted therapy 或 immunotherapy。  
- **縱膈腫瘤依區分鑑別**：**前縱膈** 4T（thymoma、teratoma/germ cell、thyroid、terrible lymphoma）；**中縱膈**（囊腫、淋巴結、血管）；**後縱膈**（多為 neurogenic tumor）。**thymoma 與 myasthenia gravis** 相關。  
- **Pleural effusion**：以 **Light's criteria** 區分 transudate（心衰、肝腎）與 exudate（感染、惡性、發炎）；不明原因 exudate 需引流分析＋考慮 pleural biopsy。  
- **Parapneumonic effusion / empyema** 分期（exudative→fibrinopurulent→organizing）：complicated effusion 或膿胸需引流；纖維化包覆期常需 VATS decortication。  
- **Pneumothorax**：primary spontaneous（高瘦年輕男性、bleb 破裂）、secondary（COPD 等基礎肺病，較危險）、tension（血流動力學不穩，需立即針刺/胸管減壓）。反覆或持續漏氣考慮手術（bleb 切除＋pleurodesis）。  
- **Chest wall tumors**：原發者良惡皆有（chondroma、chondrosarcoma、Ewing、desmoid 等），惡性原發胸壁腫瘤處置常為 **wide en-bloc resection＋重建**。  
- **Malignant pleural mesothelioma** 與 asbestos 暴露相關，預後差；處置為多模式（手術、化療、放療）綜合。

## Clinical pearl

- **切肺前，縱膈先分期**：影像看似早期但 N2 陽性會改變治療（傾向先做 neoadjuvant 而非直接開刀），**EBUS/mediastinoscopy 的病理 N 分期是關鍵 decision point**。  
- **前縱膈腫塊怕氣道/大血管壓迫**：仰臥麻醉誘導可能造成 airway 或 SVC 塌陷（尤其大 anterior mass），麻醉前務必評估，是重要 red flag。  
- **同時評估「切得掉」與「撐得住」**：可切除但肺功能不足（低 ppoFEV₁/DLCO）者需重新權衡術式或非手術治療。  
- **張力性氣胸是臨床診斷**：不要等 X 光——低血壓＋單側呼吸音消失＋氣管偏移就該立即減壓。

## 快速比較表

| 縱膈分區 | 常見腫瘤 | 關聯 |
| :---- | :---- | :---- |
| 前縱膈 | Thymoma、germ cell、lymphoma、thyroid | Myasthenia gravis（thymoma） |
| 中縱膈 | 支氣管/心包囊腫、淋巴結 | 感染/轉移 |
| 後縱膈 | Neurogenic tumor | 兒童多神經源性 |

| 積液性質 | Light's 判讀 | 常見病因 |
| :---- | :---- | :---- |
| Transudate | 皆不符 exudate 標準 | 心衰、肝硬化、腎病 |
| Exudate | 蛋白/LDH 比值符任一 | 感染、惡性、發炎 |
| Empyema | 膿、低 pH/糖 | 併肺炎、需引流 |
| Malignant | Exudate＋cytology | 轉移、mesothelioma |

## 後續證據更新

- **肺癌篩檢族群放寬（USPSTF 2021）**：相較 2013 版，2021 建議 **50–80 歲、≥20 pack-year、現吸菸或戒菸 \<15 年者每年低劑量 CT（LDCT）**（原為 55–80 歲、≥30 pack-year），涵蓋更多女性與少數族裔高風險者。**適用／限制**：需能耐受並願接受後續處置者；假陽性與過度診斷仍需共享決策。  
- **可切除 NSCLC 走向圍手術期全身治療**：**CheckMate 816（NEJM 2022）** 顯示 neoadjuvant **nivolumab＋化療** 較單純化療顯著提升 pathologic complete response 與 event-free survival，改寫早期可切除 NSCLC 的術前策略。**適用／限制**：適用可切除 IB–IIIA、無 EGFR/ALK 標的者；需多科團隊評估。  
- **EGFR 突變早期 NSCLC 術後 osimertinib**：**ADAURA（NEJM 2020）** 顯示完全切除、**EGFR 突變** 的 IB–IIIA NSCLC 術後輔助 osimertinib 大幅延長 disease-free survival（後續並顯示 overall survival 獲益）。**適用／限制**：僅適用 EGFR 突變者，需先做分子檢測。
