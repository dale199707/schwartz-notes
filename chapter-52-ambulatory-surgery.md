# Chapter 52: Ambulatory Surgery

## 核心整理

- **Ambulatory(day)surgery** 指病人於 **同日出院、不需過夜住院** 的手術;已成為許多手術的標準路徑,兼顧安全、成本與病人便利。  
- **病人選擇是安全的關鍵**:評估共病(ASA 分級)、社會支持(有人照顧與接送)、居家環境與距離醫療資源,篩選適合日間手術者。  
- **適合的手術**:侵襲性較低、預期出血/體液位移少、術後疼痛可用口服/區域鎮痛控制、無需複雜術後監測者(如疝氣、內視鏡、部分乳房/體表、關節鏡等)。  
- **共病考量**:OSA、控制不良的心肺疾病、病態肥胖、複雜糖尿病等需審慎評估,部分不適合純日間路徑。  
- **麻醉策略配合日間流程**:短效藥物、**區域麻醉/局部浸潤、多模式與 opioid 節制鎮痛**,以利快速甦醒、控制疼痛與 PONV。  
- **PONV 與疼痛是最常見延遲出院/非計畫住院的原因**:預防性處置是日間手術成功的核心。  
- **出院準備(discharge criteria)**:生命徵象穩定、清醒定向、疼痛與噁心可控、能排尿(視手術)、無活動性出血、有照顧者與明確衛教/回診安排。  
- **明確衛教與安全網**:書面出院指示、警示症狀、止痛與傷口照護說明、24 小時聯絡管道,降低返診/併發症。  
- **設施與流程**:獨立日間手術中心或院內日間單位,強調高效流程、標準化與品質監測。  
- **與 ERAS 的協同**:enhanced recovery 原則(見 ERAS 章)使更多、更大的手術得以安全地朝門診/短住院方向發展。

## Codex 驗證補強（2026）

- Ambulatory eligibility 不是只看 ASA class、BMI 或 OSA diagnosis。要整合疾病控制、procedure invasiveness/duration、預期 opioid、airway/respiratory risk、facility rescue capacity、可靠照護者與返院距離；selected stable ASA III 或 treated OSA 可能合適，unoptimized disease 則不合適。
- Discharge 應達 stable vital signs、controlled pain/nausea、可安全移動、無 active bleeding、能理解指示且有 escort/support。**是否必須排尿**依 procedure、neuraxial block、urologic risk 與 retention risk 決定，不是所有患者的 universal criterion。
- OSA 患者需判斷是否能在術後使用 CPAP、是否需高劑量 opioid、PACU 是否反覆 obstruction/desaturation；若風險無法在現場安全監測或 rescue，應轉 inpatient/extended observation。
- Medication planning 要納入 SGLT2 inhibitor 術前停 3–4 天，以及 GLP-1 的 aspiration-risk 個別化；同時給明確 written instructions，避免 day-of-surgery 才發現不適合麻醉。
- Same-day discharge 不是 ERAS 的必然終點；只有達成 clinical criteria 且 home environment 可安全接手時才出院，未預期 admission 應被視為需要分析的 system outcome，而非催促患者離院的理由。

## Clinical pearl

- **選對病人比選對手術更關鍵**:社會支持與共病決定日間手術是否安全,缺乏照顧者是常見被忽略的因素——重要 decision point。  
- **顧好疼痛與 PONV**:這兩者是延遲出院與非計畫住院的主因,預防性多模式處置是成功關鍵。  
- **出院前確認能安全在家**:符合出院準則＋明確安全網,勝過單看時間到就放行。  
- **OSA/病態肥胖要審慎**:呼吸抑制風險使部分病人不適合純日間路徑,需個別評估。

## 快速比較表

| 適合日間手術 | 較不適合 |
| :---- | :---- |
| ASA I–II(或穩定 III) | 控制不良重大共病 |
| 有照顧者/近醫療 | 獨居/偏遠、無支援 |
| 疼痛可口服/區域控制 | 需複雜術後監測 |
| 出血/體液位移少 | 大量失血風險 |

| 出院準則 | 內容 |
| :---- | :---- |
| 生命徵象 | 穩定 |
| 意識 | 清醒定向 |
| 症狀 | 疼痛/噁心可控 |
| 安全網 | 照顧者、衛教、回診 |

## 後續證據更新

- **Enhanced recovery 推動手術朝門診化/短住院發展**:ERAS® 的圍手術期實證措施(例如 Gustafsson 等,大腸手術指引,World J Surg 2019)透過多模式鎮痛、微創、少管路與早期活動,使原本需住院的手術得以更安全地縮短住院甚至日間化。**適用／限制**:仍需嚴格病人選擇與安全網;並非所有手術/病人皆適合門診路徑。  
- 相較 2019 教材,**日間手術範圍擴大**(含部分過去需住院的術式),並更強調 **標準化出院準則與居家安全網**。**適用／限制**:擴大範圍需以病人選擇與結果監測為前提。
