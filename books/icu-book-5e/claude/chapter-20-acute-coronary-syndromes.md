# Chapter 20: Acute Coronary Syndromes

## 核心整理

- **2025 年 ACC/AHA/ACEP/NAEMSP/SCAI 發表新版 acute coronary syndromes 指引（Circulation 2025），將 STEMI 與 NSTE-ACS 整合於單一文件。** 這是本章的主要依據，取代先前分立的 STEMI 與 NSTEMI 指引。

- **Troponin 上升代表心肌損傷，不等於心肌梗塞。** 依 Fourth Universal Definition，診斷 MI 需要**troponin 的上升和／或下降、至少一個數值超過第 99 百分位 upper reference limit，並且有急性心肌缺血的臨床證據**。**Cardiac troponin 本身無法區分心肌損傷的亞型，絕不可單獨用來診斷。**

- **Type 1 與 Type 2 MI 的機轉不同，處置也不同——這是 ICU 最關鍵的區分。** Type 1 源自 **atherosclerotic plaque 破裂、糜爛或潰瘍造成的管腔內血栓**；Type 2 則是**氧氣供需失衡造成的缺血性心肌損傷，與斑塊破裂及冠狀動脈血栓無關**。

- **在未經篩選的住院病人中，acute non-ischaemic myocardial injury 與 type 2 MI 合計約占所有可測得 troponin 上升的一半。** **這意味著 ICU 中大多數的 troponin 異常不是 type 1 MI**——把它們一律當作 ACS 處理，會導致不必要的抗血小板治療、抗凝與侵入性檢查。

- **Type 2 MI 的治療核心是矯正供需失衡的原因，而非抗血栓治療。** ICU 中的常見驅動因素包括：sepsis 與全身性發炎、嚴重貧血、缺氧、心搏過速（含新發 AF，見 Ch.19）、低血壓或高血壓危象、以及外源性 catecholamine。**處理這些因素，而不是加上 DAPT。**

- **High-sensitivity troponin 應用於加速「排除」與「確立」MI，配合連續心電圖與連續 troponin 進行動態風險評估。** 高敏感度檢測的價值在於**變化幅度（delta）與趨勢**，單一數值的意義有限——尤其在腎功能不全、心衰竭與重症病人，基礎值本就可能偏高。

- **STEMI 的決定性治療是及時再灌流，時間就是心肌。** 在能及時執行的中心，primary PCI 優於溶栓；無法在建議時間內轉送者應考慮溶栓後再轉送。**這一點在 2025 年指引中未改變。**

- **2025 指引將 complete revascularization 列為建議策略。** 對 STEMI 或 NSTEMI ACS 病人建議完整血管重建，方式（PCI 或 CABG）依冠狀動脈病灶複雜度與共病決定。**對血流動力學穩定的 STEMI 合併非罪責病灶者，完整血管重建（同次或分階段）可降低後續事件。**

- **這與 cardiogenic shock 的策略相反，必須分清楚。** CULPRIT-SHOCK 顯示在 **AMI 合併 cardiogenic shock** 時，只處理罪責病灶優於立即多血管 PCI（見 Ch.16）。**「穩定者做完整血管重建、休克者只做罪責病灶」是兩個方向相反但各有證據的建議。**

- **DAPT 的建議為 aspirin 加上 P2Y12 inhibitor，且對多數病人偏好 ticagrelor 或 prasugrel 而非 clopidogrel。** 無高出血風險者建議至少 12 個月；偏好 ticagrelor／prasugrel 的理由是主要不良心血管事件的減少幅度較大。**高出血風險者需個別化縮短療程或改用 clopidogrel。**

- **2025 指引將 intravascular imaging 導引 PCI 提升為 Class 1 建議。** 這是相對於先前版本的實質升級，反映影像導引可改善支架置放品質與臨床結果。

- **同一份指引強調 PCI 應優先採 radial 而非 femoral 途徑**，以降低出血、血管併發症與死亡風險。**這是少數同時改善出血與死亡的技術性建議。**

- **ICU 病人的 ACS 診斷困難，因為典型症狀常被遮蔽。** 插管鎮靜、譫妄或術後鎮痛的病人無法主訴胸痛；此時的線索是新出現的心電圖變化、無法解釋的低血壓或肺水腫、新發心律不整、以及 troponin 的動態變化。**對高風險族群應主動安排連續心電圖，而非等待症狀。**

- **重症病人的抗血栓治療必須權衡出血風險，且此權衡常偏向保守。** 合併 thrombocytopenia、近期手術、活動性出血或需要侵入性處置時，**DAPT 與抗凝的風險可能超過效益**。決策應與心臟科共同進行並明確記錄理由。

- **ACS 之外的 troponin 上升鑑別診斷應主動涵蓋**：pulmonary embolism（右心受壓）、myocarditis、Takotsubo syndrome、sepsis 相關心肌損傷、腎衰竭、心律不整、以及主動脈剝離。**其中主動脈剝離最關鍵——誤診為 ACS 而給予抗血小板與抗凝可能致命。**

## Clinical pearl

- **【Decision point】看到 troponin 上升，先問「這是 type 1 還是 type 2」。** Type 1（斑塊破裂＋血栓）需要抗血栓治療與血管重建評估；**type 2（供需失衡）需要的是矯正 sepsis、貧血、缺氧、心搏過速或低血壓**。在未經篩選的住院病人中，非缺血性心肌損傷與 type 2 MI 合計約占 troponin 上升的一半——**在 ICU 這個比例只會更高。**

- **【Red flag】以 ACS 處理前，先確認不是 aortic dissection。** 胸痛合併脈搏或血壓左右不對稱、新出現的 aortic regurgitation 雜音、縱膈腔增寬、或疼痛呈撕裂樣並向背部放射時，**給予抗血小板與抗凝可能致命**。此時應先影像確認。

- **【常見陷阱】把 CULPRIT-SHOCK 的結論套用到穩定的 STEMI 病人。** 兩者方向相反：**穩定者 → 2025 指引建議完整血管重建（同次或分階段）；合併 cardiogenic shock 者 → CULPRIT-SHOCK 支持只處理罪責病灶**（30 天死亡或 RRT 45.9% vs 55.4%，P = 0.01）。判斷依據是有無休克，不是病灶數目。

- **【常見陷阱】在插管鎮靜的病人身上等待胸痛主訴。** ICU 病人的 ACS 常以**新出現的心電圖變化、無法解釋的低血壓或肺水腫、新發心律不整**表現。對高風險族群（已知冠心病、術後、嚴重貧血、大手術）應主動安排連續心電圖與 troponin 趨勢，而非依賴症狀。

## 後續證據更新

- **2025 年 ACC/AHA/ACEP/NAEMSP/SCAI ACS 指引（Circulation 2025）把 STEMI 與 NSTE-ACS 整合為單一文件**，是本章最重要的結構性更新。三項最具實務影響的內容為：**complete revascularization 成為建議策略**、**intravascular imaging 導引 PCI 升為 Class 1**、以及**明確偏好 radial 途徑**。

- **「complete revascularization」與「culprit-only in shock」的並存需要被正確理解為互補而非矛盾。** 前者適用於血流動力學穩定者，後者來自 CULPRIT-SHOCK 在 cardiogenic shock 族群的證據。**臨床上的判別點是有無休克。**

- **Intravascular imaging 由建議升為 Class 1，是介入技術層面近年最明確的升級**，反映影像導引對支架擴張與貼壁的改善可轉化為臨床結果。

- **Type 1 與 Type 2 MI 的區分在 ICU 的重要性持續上升。** 現有資料顯示未經篩選的住院病人中，非缺血性心肌損傷與 type 2 MI 合計約占 troponin 上升的一半。**然而 type 2 MI 的最佳治療策略仍是明確的 evidence gap**——目前沒有隨機試驗證明對 type 2 MI 給予抗血小板治療或侵入性檢查可改善結果，**因此現行做法（矯正誘因為主）是基於病生理推論而非試驗證據。**

- **Fourth Universal Definition of Myocardial Infarction（2018）仍是現行的診斷架構**，尚未見取代版本；其核心原則（troponin 動態變化＋急性缺血臨床證據，且 troponin 不能單獨區分亞型）未受挑戰。

- 就 STEMI 的及時再灌流原則、primary PCI 優於溶栓（在可及時執行時）、以及 aortic dissection 必須在抗血栓治療前排除而言，**目前沒有會改變臨床決策的新證據**；這些屬於已確立的原則。
