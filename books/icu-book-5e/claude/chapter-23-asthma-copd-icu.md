# Chapter 23: Asthma and COPD in the ICU

## 核心整理

- **兩者的共同致命機轉是 dynamic hyperinflation 與 auto-PEEP（intrinsic PEEP），而非低氧。** 呼氣氣流受限使肺泡在下一次吸氣前來不及排空，肺容積逐次累加。**後果是胸內壓上升、靜脈回流下降、血壓崩潰、以及氣壓傷**——這是插管後最常見的致命併發症。

- **兩者的病生理與治療反應仍有實質差異，不應完全等同。** Asthma 以可逆的支氣管收縮與黏液栓塞為主，對支氣管擴張劑與類固醇反應較好；COPD 的氣流受限有結構性成分（肺氣腫、氣道重塑），可逆程度有限，且常合併呼吸肌疲乏與高碳酸血症的慢性代償。

- **插管前應盡可能用盡非侵入性選項——尤其在 COPD。** NIV 應作為**急性高碳酸血症呼吸衰竭合併呼吸性酸中毒住院病人的第一線通氣模式**；NIV 與高流量氧氣系統可改善氣體交換、減少呼吸功與插管需求，並縮短住院天數與改善存活。

- **Asthma 使用 NIV 的證據弱於 COPD。** 嚴重氣喘發作時 NIV 可作為橋接，但**不應延誤必要的插管**；病人若已疲乏、意識改變或無法配合，應直接插管。

- **插管本身是嚴重氣喘最危險的時刻。** 誘導藥物首選 **ketamine 或 propofol**（兩者皆有支氣管擴張作用，**ketamine 造成的低血壓較少**）；應使用適合體型的**最大內徑氣管內管（如 7.5–8 mm）**以降低氣道阻力與尖峰壓力。

- **插管後的通氣策略以「限制過度充氣」為唯一優先，而非矯正血氣。** 具體做法：**低呼吸速率、低潮氣量、延長呼氣時間（降低 I:E 比）、高吸氣流速**。**Permissive hypercapnia（pH 約 7.25–7.30）是被接受的代價**，早期常需深度鎮靜與神經肌肉阻斷以避免呼吸器不同步。

- **嚴重氣喘的外加 PEEP 多數建議設為零或低於 intrinsic PEEP 的水準，以最小化過度充氣。** **這與 ARDS 的高 PEEP 策略方向相反**（見 Ch.24）——把 ARDS 的通氣習慣套用到重度氣喘會直接造成傷害。

- **COPD 的外加 PEEP 邏輯不同，適度 PEEP 可能有幫助。** 在有動態氣道塌陷（expiratory flow limitation）的 COPD 病人，外加 PEEP 至略低於 auto-PEEP 可**減少觸發呼吸器所需的吸氣努力**，改善人機同步；**但仍須避免超過 auto-PEEP 而加重過度充氣。**

- **插管後突然低血壓，第一個假設應是 auto-PEEP 而非敗血症或低血容。** **處置是立即斷開呼吸器讓病人完全吐氣**（可同時輕壓胸廓），若血壓迅速回升即確診。**第二個必須排除的是 tension pneumothorax。** 這兩者的處置在數十秒內即可完成並救命。

- **藥物治療的核心組合：吸入型 β₂-agonist、muscarinic antagonist、以及全身性 corticosteroid。** 嚴重發作時加上 **magnesium sulphate**。這是兩種疾病共通的基礎，差別在於 COPD 對支氣管擴張劑的反應幅度較小。

- **COPD exacerbation 的 corticosteroid 療程為 5 天，劑量相當於 prednisone 40 mg／日**，口服或靜脈依臨床狀況決定。**惟須注意：一項 meta-analysis 的次族群分析顯示，ICU 族群的效果差異未達統計顯著**——現行 5 天建議主要建立在非 ICU 族群的證據上。

- **COPD exacerbation 的抗生素療程亦為 5 天，適用於痰液呈膿性、既往肺部感染史等情況。** 不是所有 exacerbation 都需要抗生素——**痰量增加、痰液變膿、呼吸困難加重這三項的組合是傳統的判斷依據。**

- **嚴重氣喘的 magnesium sulphate 靜脈給予有其角色，但屬輔助而非核心。** 它是相對安全、低成本的介入，適用於對初始支氣管擴張劑反應不佳者；**不應取代持續的吸入治療與全身性類固醇。**

- **Ketamine 除了作為誘導藥物，在持續鎮靜中也因支氣管擴張作用而受青睞。** 但目前**沒有高品質隨機證據證明 ketamine 相較其他鎮靜藥物可改善嚴重氣喘的臨床結果**——其使用建立在藥理機轉與臨床經驗之上。

- **兩者的脫離呼吸器過程都容易延長，且原因不同。** 氣喘多因黏液栓塞與氣道發炎未完全消退；COPD 則常合併呼吸肌無力、營養不良與心血管共病。**COPD 病人的計畫性拔管後 NIV 可降低再插管率**（見 Ch.30）。

## Clinical pearl

- **【Red flag】嚴重氣喘或 COPD 病人插管後血壓驟降。** 第一個動作**不是給輸液或升壓劑，而是把病人與呼吸器斷開，讓他完全吐氣**（必要時輕壓胸廓）。若血壓在數十秒內回升，即為 auto-PEEP 造成的靜脈回流受阻。**第二個必須排除的是 tension pneumothorax**（單側呼吸音消失、氣管偏移、頸靜脈怒張、超音波無 lung sliding）。

- **【常見陷阱】把 ARDS 的通氣習慣套用到重度氣喘。** ARDS 用高 PEEP 開放肺泡；**嚴重氣喘則多數建議外加 PEEP 設為零或低於 intrinsic PEEP**，並以低呼吸速率、延長呼氣時間為核心。**方向完全相反**——搞混會直接加重過度充氣與氣壓傷。

- **【Decision point】血氣顯示 PaCO₂ 高、pH 低時，要抗拒「加大通氣量」的直覺。** 嚴重氣喘插管後的初期目標是**限制過度充氣而非正常化血氣**；permissive hypercapnia（pH 約 7.25–7.30）是可接受的。**提高呼吸速率或潮氣量會縮短呼氣時間、堆疊更多 auto-PEEP，使情況惡化而非改善。**

- **【常見陷阱】COPD 病人一律給抗生素與長療程類固醇。** GOLD 建議類固醇與抗生素**皆為 5 天**，抗生素限於痰液呈膿性或有既往肺部感染史等情況。**另須留意：類固醇的效益主要來自非 ICU 族群的資料，meta-analysis 的 ICU 次族群未達顯著**——這不代表不該用，但也不支持在 ICU 延長療程。

## 快速比較表

| 面向 | 嚴重氣喘 | COPD exacerbation |
|---|---|---|
| 核心病生理 | 可逆的支氣管收縮＋黏液栓塞 | 結構性氣流受限＋呼吸肌疲乏＋慢性高碳酸血症代償 |
| 對支氣管擴張劑的反應 | 較佳 | 有限 |
| NIV 的角色 | 可作橋接，**證據較弱，不可延誤插管** | **第一線**（急性高碳酸血症呼吸衰竭合併呼吸性酸中毒） |
| 外加 PEEP | **多數建議設為零或低於 intrinsic PEEP** | 適度 PEEP（略低於 auto-PEEP）可**改善觸發與人機同步** |
| 插管誘導藥物 | Ketamine 或 propofol（ketamine 低血壓較少） | 同左 |
| 氣管內管口徑 | **盡可能大（如 7.5–8 mm）**以降低阻力 | 同左 |
| 類固醇 | 全身性，急性期必要 | **5 天，相當於 prednisone 40 mg／日** |
| 抗生素 | 一般不需要（除非有感染證據） | **5 天**，限痰液呈膿性或有既往肺部感染史等 |
| Magnesium sulphate | 嚴重發作時的輔助治療 | 角色較不確定 |

| 呼吸器設定原則（嚴重阻塞性疾病） | 做法 | 理由 |
|---|---|---|
| 呼吸速率 | **低** | 延長呼氣時間，減少 air trapping |
| 潮氣量 | **低** | 限制過度充氣 |
| 吸氣流速 | **高** | 縮短吸氣時間，相對延長呼氣 |
| I:E 比 | **延長呼氣（如 1:3 以上）** | 讓肺泡有時間排空 |
| 外加 PEEP（氣喘） | **零或低於 intrinsic PEEP** | 避免疊加過度充氣 |
| 外加 PEEP（COPD 有 flow limitation） | 略低於 auto-PEEP | 降低觸發所需努力，改善同步 |
| 血氣目標 | **Permissive hypercapnia（pH 約 7.25–7.30）** | 限制過度充氣優先於正常化血氣 |
| 鎮靜／肌肉鬆弛 | 早期常需深度鎮靜 ± NMBA | 避免人機不同步加重 air trapping |

## 後續證據更新

- **本章的核心原則（限制 dynamic hyperinflation 優先於矯正血氣、permissive hypercapnia、低呼吸速率與延長呼氣）在近年並無方向性改變**，屬於已確立的生理導向策略。**這也意味著此領域缺乏大型隨機試驗**——多數做法建立在生理推論與觀察性資料之上。

- **NIV 於 COPD 急性高碳酸血症呼吸衰竭的第一線地位持續獲得支持**，可改善氣體交換、減少插管、縮短住院並改善存活。**相對地，NIV 於嚴重氣喘的證據仍然薄弱**，兩者不應等同看待。

- **COPD exacerbation 的 corticosteroid 5 天療程有一個值得注意的證據限制：** meta-analysis 的次族群分析顯示 **ICU 族群的效果差異未達統計顯著**。這不推翻使用類固醇的建議，但提示**現行療程建議主要外推自非 ICU 族群**，且不支持在 ICU 病人延長療程。

- **Ketamine 在嚴重氣喘的使用建立在藥理機轉（支氣管擴張、較少低血壓）與臨床經驗之上，尚無高品質隨機證據證明可改善臨床結果。** 引用時應誠實標示此限制。

- **GOLD 報告持續每年更新**（近年版本強調早期偵測、多重共病與人工智慧應用），但**就 ICU 期的急性處置而言，核心建議（NIV 第一線、類固醇與抗生素各 5 天）未有方向性改變。**

- 就 auto-PEEP 的辨識與處置（斷開呼吸器讓其吐氣）、tension pneumothorax 的鑑別、以及嚴重氣喘應使用大口徑氣管內管而言，**目前沒有會改變臨床決策的新證據**；這些屬於穩定的生理與操作知識。
