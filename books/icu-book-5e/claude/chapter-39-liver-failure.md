# Chapter 39: Liver Failure

## 核心整理

- **Acute liver failure (ALF) 與 acute-on-chronic liver failure (ACLF) 是兩個病生理與處置完全不同的疾病，混淆兩者會導致嚴重的決策錯誤。** ALF 發生在**無既有肝病**的病人，以 coagulopathy 與 hepatic encephalopathy 為定義；**ACLF 則是既有慢性肝病急性惡化並合併器官衰竭**，核心特徵為 systemic inflammation。

- **ACLF 的病生理與單純的失代償性肝硬化不同，這是近年最重要的概念澄清。** ACLF 的腎功能障礙機轉**以 systemic inflammation 與 cytokine storm 為特徵**，而非單純的血流動力學（splanchnic vasodilation）問題——**這解釋了為何它的處置與預後與一般失代償不同。**

- **ACLF 有多套並存的定義系統（EASL-CLIF、APASL、NACSELD），且納入標準不同。** 這造成文獻比較與臨床溝通的困難；**引用 ACLF 相關資料時應說明採用哪一套定義。** EASL 架構下 ACLF 依器官衰竭數目分級。

- **AASLD 於 2023 年發表 ACLF 的 practice guidance，APASL 於 2024 年發表 ACLF 合併 AKI 的臨床實務指引**——這反映該領域正快速標準化中。

- **ALF 最重要的可治療病因是 acetaminophen（paracetamol）中毒，而 N-acetylcysteine 應在懷疑時即給予，不等待確診。** NAC 的安全性高、成本低；**延遲給予是可預防的傷害**（見 Ch.52）。其他病因包括病毒性肝炎、藥物、autoimmune、Wilson disease、Budd-Chiari 與 heat stroke（見 Ch.43）。

- **ALF 的最主要死因是腦水腫與顱內壓上升，這是它與 ACLF 最關鍵的臨床差異。** 高風險徵象包括快速進展的 encephalopathy、高血氨（一般 > 150–200 µmol/L）與年輕病人。**處置包括床頭抬高、避免刺激、控制發燒、hypertonic saline 或 mannitol、以及考慮 renal replacement therapy 清除氨。**

- **ACLF 病人的 hepatic encephalopathy 較少造成致命性腦水腫**，處置以找誘因（感染、消化道出血、電解質異常、便秘、鎮靜藥物）與 lactulose、rifaximin 為主。**兩者的處置重點完全不同。**

- **肝病的凝血異常是「再平衡（rebalanced）」而非單純出血傾向——這是最常被誤解的一點。** 促凝與抗凝因子同時減少；**INR 上升不代表出血風險上升，也不代表病人「自動抗凝」**。**因此不應僅因 INR 異常而預防性輸注 FFP**（見 Ch.13），且**肝硬化病人仍需 VTE 預防評估**（見 Ch.5）。

- **Hepatorenal syndrome–AKI (HRS-AKI) 的診斷需排除其他 AKI 原因，並在停用利尿劑與擴容試驗後仍持續。** 擴容通常使用 albumin（見 Ch.10 肝硬化相關適應症）。

- **Terlipressin 合併 albumin 是 HRS-AKI 的建議治療。** 一項針對 ACLF 合併 HRS-AKI 的前瞻性世代研究顯示 **65% 的病人對 terlipressin 有反應**，**但 21% 出現不良反應**——這個效益與風險的並列是用藥決策的核心。

- **Terlipressin 的不良反應具體且需主動監測：** 缺血性併發症（指端、腸道、心肌）、**呼吸衰竭**（尤其在已有肺部問題或大量 albumin 輸注者）、以及低血鈉。**在 ACLF 第三級或已有呼吸問題的病人，風險特別高。**

- **Spontaneous bacterial peritonitis 的診斷依 ascitic fluid 的 PMN ≥ 250/mm³，且 albumin 併用抗生素可降低腎功能惡化與死亡。** 這是 albumin 在肝病的少數強證據適應症之一（見 Ch.10）。**所有新入院或臨床惡化的腹水病人都應做診斷性 paracentesis。**

- **食道靜脈曲張出血的處置有明確順序：** 復甦（**限制性輸血策略，見 Ch.12**）、血管收縮藥物（terlipressin 或 octreotide）、**預防性抗生素**（可降低死亡率，是少數有存活效益的介入）、以及 12 小時內的內視鏡治療。難治性出血考慮 balloon tamponade 過渡至 TIPS。

- **感染是 ACLF 最常見的誘因與死因，且臨床表現常不典型。** 肝硬化病人的免疫功能障礙使發燒與白血球上升可能不明顯；**懷疑的門檻應該很低**，並注意 multidrug-resistant 病原的比例較高。

- **肝移植的評估應及早啟動，因為 ALF 與高級別 ACLF 的病程可在數日內惡化至無法移植。** ALF 有 King's College criteria 等預後工具；**ACLF 的預後隨器官衰竭數目急遽惡化**。**「等看看會不會好」在這兩種疾病中都是高風險策略。**

## Clinical pearl

- **【Decision point】第一步永遠是分清楚是 ALF 還是 ACLF——它們的致命併發症不同。** **ALF：腦水腫與顱內壓上升是主要死因**，需積極的顱內壓相關處置與早期移植評估。**ACLF：systemic inflammation 與多器官衰竭是主軸**，encephalopathy 較少致命，重點在找誘因（尤其感染）與器官支持。**把 ACLF 的處置套用到 ALF（或反之）會錯過真正的致命問題。**

- **【Red flag】懷疑 acetaminophen 中毒就給 N-acetylcysteine，不要等血中濃度或確診。** NAC 安全、便宜，**延遲給予是可預防的傷害**。同理，ALF 病人出現快速進展的 encephalopathy 加上血氨 > 150–200 µmol/L，應立即啟動腦水腫的預防與處置，**而非等待影像確認**——CT 對早期腦水腫的敏感度有限。

- **【常見陷阱】因為 INR 上升而預防性輸 FFP，或因此不給 VTE 預防。** 肝病的凝血是**再平衡**狀態：促凝與抗凝因子同時減少。**INR 不預測肝病病人的出血風險**，FFP 也無法有效矯正（見 Ch.13）；**同時肝硬化病人仍有血栓風險，需要 VTE 預防評估**（見 Ch.5）。這兩個錯誤方向相反但源自同一個誤解。

- **【常見陷阱】對 HRS-AKI 使用 terlipressin 而未主動監測缺血與呼吸併發症。** 前瞻性資料顯示 ACLF 合併 HRS-AKI 者 **65% 有反應、但 21% 出現不良反應**。**缺血性併發症（指端、腸道、心肌）與呼吸衰竭需要每日主動檢視**——尤其在已有肺部問題或接受大量 albumin 的病人。

## 快速比較表

| 面向 | Acute liver failure (ALF) | Acute-on-chronic liver failure (ACLF) |
|---|---|---|
| 背景肝病 | **無既有肝病** | **既有慢性肝病** |
| 定義核心 | Coagulopathy ＋ hepatic encephalopathy | 急性惡化 ＋ **器官衰竭**；核心為 systemic inflammation |
| 腎功能障礙機轉 | 多為 ATN 或血流動力學 | **Systemic inflammation 與 cytokine storm** |
| **主要死因** | **腦水腫與顱內壓上升** | **多器官衰竭與感染** |
| Encephalopathy 的意義 | **可致命（腦水腫）**；需積極處置 | 較少致命；重點在找誘因 |
| 常見可治療病因 | **Acetaminophen（→ 立即 NAC）**、病毒、藥物、Wilson、Budd-Chiari | **感染**、消化道出血、酒精、藥物 |
| 定義系統 | 相對一致 | **多套並存（EASL-CLIF、APASL、NACSELD）**，引用時須註明 |
| 指引 | — | AASLD practice guidance（2023）；APASL AKI in ACLF（2024） |

| 併發症 | 診斷 | 處置重點 |
|---|---|---|
| **HRS-AKI** | 排除其他 AKI 原因；停利尿劑＋albumin 擴容後仍持續 | **Terlipressin ＋ albumin**（反應率約 65%）；**主動監測缺血與呼吸併發症（不良反應約 21%）** |
| **Spontaneous bacterial peritonitis** | Ascitic fluid **PMN ≥ 250/mm³** | 抗生素 ＋ **albumin**（降低腎功能惡化與死亡）；新入院或惡化者皆應診斷性 paracentesis |
| **食道靜脈曲張出血** | 內視鏡 | 限制性輸血（Ch.12）→ terlipressin／octreotide → **預防性抗生素（有存活效益）** → 12 小時內內視鏡；難治者 balloon tamponade → TIPS |
| **Hepatic encephalopathy（ACLF）** | 臨床分級 | 找誘因（感染、出血、電解質、便秘、鎮靜藥）；lactulose、rifaximin |
| **腦水腫（ALF）** | 快速進展 encephalopathy ＋ 血氨 > 150–200 µmol/L | 床頭抬高、避免刺激、控制發燒、hypertonic saline／mannitol、考慮 RRT 清除氨 |
| **凝血異常** | INR 上升 | **「再平衡」狀態**；**不因 INR 預防性輸 FFP**（Ch.13）；**仍需 VTE 預防評估**（Ch.5） |

## 後續證據更新

- **ACLF 的病生理理解已從「嚴重的失代償」轉為「以 systemic inflammation 與 cytokine storm 為核心的獨立症候群」。** 這個轉變的實務意涵是：**ACLF 的腎功能障礙機轉與單純失代償性肝硬化不同**，因此處置與預後判斷也不應直接套用。

- **該領域正快速標準化：AASLD 於 2023 年發表 ACLF practice guidance，APASL 於 2024 年發表 ACLF 合併 AKI 的臨床實務指引。** **但多套定義系統（EASL-CLIF、APASL、NACSELD）並存仍是文獻比較與臨床溝通的實質障礙**——**引用 ACLF 資料時必須說明採用哪一套定義**，否則納入族群可能完全不同。

- **Terlipressin 用於 HRS-AKI 的效益與風險已被同一份前瞻性資料並列量化（反應率 65%、不良反應 21%）。** **這種效益與代價成對呈現的方式，比單獨引用反應率更誠實**，也更貼近實際的用藥決策。**其不良反應（缺血性併發症、呼吸衰竭、低血鈉）需要主動監測而非被動等待。**

- **肝病凝血的「再平衡」概念已廣泛接受，但臨床實踐的落差仍大。** 因 INR 而預防性輸 FFP、以及因 INR 而不給 VTE 預防，這兩個方向相反的錯誤在實務上仍常見。**這與 Ch.13 的證據（FFP 無法有效矯正輕中度 INR）直接呼應。**

- **本章的多數處置原則（NAC 用於 acetaminophen、SBP 的 albumin 適應症、靜脈曲張出血的預防性抗生素）證據基礎穩固，近年沒有方向性改變。** **預防性抗生素於靜脈曲張出血是少數具存活效益的介入**，值得強調。

- 就 ALF 與 ACLF 的區分、ALF 腦水腫為主要死因、ACLF 以感染為最常見誘因與死因、以及肝移植評估應及早啟動而言，**目前沒有會改變臨床決策的新證據**；這些屬於已確立的臨床架構。
