# Chapter 035: Fundoplication, Laparoscopic

## 核心整理

- Indication：symptomatic GERD 是最常見適應，最常用 floppy 360-degree Nissen 技術；反覆 aspiration pneumonia 或 reflux 誘發的 asthma 是重要適應；PPI 不耐、順從性差或終身用藥成本亦為考量。
- 術前必做 esophageal function studies（manometry 或 video esophagography）以規劃 full 或 partial fundoplication，並偵測與 reflux 無關的 underlying dysmotility；肺功能評估對高風險（aspiration／asthma）病人特別重要。
- Positioning：supine split-leg 或 low lithotomy、reverse Trendelenburg 頭高至少 30°；置 elastic／sequential compression stockings 與 orogastric tube。
- Port：Veress 或 open Hasson 進腹，camera port 置於 midline 稍左、距 xiphoid 約 15 cm；另置 subcostal 操作 port 與 subxiphoid liver retractor 顯露 hiatus。
- 解剖步驟：以 ultrasonic dissector 分開 lesser omentum 的 pars flaccida，注意可能存在的 aberrant left hepatic artery 需辨識並保留；分離 left/right crus 與 phrenoesophageal ligament，顯露食道後方 hiatal defect。
- Fundus 游離：分斷 short gastric vessels（距胃約 1 cm 以減 thermal injury），器械尖端須清楚可見以免部分切斷下一條血管造成難控出血；注意勿傷 left gastric artery。
- 食道 mediastinal 游離至腹腔內有 ≥ 3 cm 無張力食道，保留 anterior 與 posterior vagus；勿盲目向 hiatus 上方剝離以免造成 pleural opening。
- Crural closure：食道後方以 2 針 0 non-absorbable braided suture（intracorporeal 或 endoscopic suturing device）由左向右靠攏 crus。
- Wrap：先確認 gastric mobility 足夠，將 upper greater curvature 繞至食道後方，做「shoeshine」左右滑動測試確保無張力；置 56–60 French dilator（尖端須完全進胃以免低估口徑）。
- 以 3 針建立 wrap（自 cephalad 起，每針 triple bite 含食道 seromuscular partial-thickness），最後一針錨定 wrap 於 right crus／食道側壁防下滑；術後多不需 NG 減壓，clear liquids 漸進至軟食，暫時性 dysphagia 可調整飲食。

## Clinical pearl

- 分斷 pars flaccida 時警覺 aberrant left hepatic artery，須辨識並保留。
- 「Shoeshine」測試與置入 56–60 French dilator，確保 floppy、無張力的短 wrap，降低術後 dysphagia。
- 確保腹腔內有 ≥ 3 cm 無張力食道，否則易復發或 wrap 遷移；食道太短時考慮 Collis gastroplasty。
- 最後一針將 wrap 錨定於 crus／食道，防 slipped Nissen。

## 後續證據更新

- Laparoscopic fundoplication 已是抗逆流手術的標準做法；現行 GERD 指引建議術前以客觀檢查（pH、manometry）確認診斷、排除 dysmotility，並依蠕動情況選 full 或 partial（Toupet）wrap。（Katz 2022）
- 對 PPI 有效但不願或不耐長期用藥者，抗逆流手術是合理長期選項；magnetic sphincter augmentation 等 endoluminal／device 療法為近年新增選項（正式來源請 Codex 補上並核對）。（Katz 2022）
