# Chapter 02: Systemic Response to Injury and Metabolic Support

## 核心整理

- 創傷或手術後的宿主反應由 **danger signals** 啟動：內源性 **DAMPs**（damage-associated molecular patterns，如 HMGB1、mitochondrial DNA、ATP）與外源性 **PAMPs**（如 LPS）經 **pattern recognition receptors**（TLRs、NLRs）活化 innate immunity，是「無菌性發炎」與感染性發炎共用的上游機轉。  
- 反應強度取決於 injury 大小；適度反應有助修復，過度或失控則導致 **SIRS**（systemic inflammatory response syndrome）進而 organ dysfunction。傳統上同時存在代償性的 **CARS**（compensatory anti-inflammatory response syndrome），現多以 **PICS**（persistent inflammation-immunosuppression catabolism syndrome）描述長期重症的免疫失衡。  
- 神經內分泌反應：下視丘–腦垂體–腎上腺（**HPA axis**）活化使 **cortisol** 上升；交感活化釋放 **catecholamines**；並分泌 **ADH（vasopressin）**、**aldosterone**、**glucagon**、growth hormone。整體導向保留鈉水、升高血糖、動員能量受質。  
- 代謝分兩期（Cuthbertson）：早期 **ebb phase**（數小時，心輸出下降、代謝降低、著重復甦）與後續 **flow phase**（hypermetabolism、hyperdynamic circulation、負氮平衡）。  
- **Stress hyperglycemia** 來自 counter-regulatory hormones 與周邊 **insulin resistance**（glucose 生成上升、利用下降），是創傷代謝的核心表現。  
- **Cytokines** 為主要訊息分子：**TNF-α** 與 **IL-1** 為早期促發炎；**IL-6** 濃度與 injury severity 及預後相關，並驅動 hepatic acute-phase response（CRP、fibrinogen 上升、albumin 下降）；**IL-10** 具抗發炎作用。  
- 蛋白質代謝以骨骼肌 **catabolism** 為主，釋出 amino acids 供肝臟糖質新生與急性期蛋白合成；臨床上表現為肌肉流失與負氮平衡，單靠外源熱量無法完全逆轉。  
- Gut 在重症中被視為「motor of MODS」：腸屏障功能受損、通透性上升與菌群位移（bacterial translocation）被認為助長全身性發炎，是早期 enteral nutrition 的理論基礎之一。  
- 代謝支持原則：只要腸道可用，**enteral nutrition 優於 parenteral**（較低感染併發症、維持腸屏障）；血流動力學穩定後應早期（一般 24–48 小時內）啟動腸道餵食。  
- 熱量目標約 25–30 kcal/kg/day、蛋白質約 1.2–2.0 g/kg/day（燒傷、重症更高）；避免 **overfeeding**（高血糖、hepatic steatosis、CO₂ 上升、脫機困難）。  
- 血糖控制：極嚴格控制（80–110 mg/dL）會增加低血糖與死亡（NICE-SUGAR），故重症目標多採 **140–180 mg/dL** 的中度控制。  
- 針對重症的 **immunonutrition**（arginine、glutamine、ω-3、nucleotides）效益不一致，非全體適用；常規對 ICU 病人補充高劑量 glutamine/antioxidant 未顯示益處，需依族群審慎採用。

## Clinical pearl

- **IL-6 是嚴重度的臨床指標**：術後或創傷後 IL-6 顯著且持續上升，往往先於臨床惡化，可作為過度發炎反應的 red flag。  
- **腎上腺功能不足易被忽略**：對液體與升壓劑反應不佳的 refractory shock，應考慮 critical illness-related corticosteroid insufficiency，適時給予 hydrocortisone。  
- **早期腸道餵食的前提是灌流穩定**：對使用高劑量或漸增血管收縮劑、未復甦完全的病人強行餵食，有 non-occlusive mesenteric ischemia 風險——decision point 在於血流動力學是否已穩定。  
- **避免 overfeeding 是常見陷阱**：熱量過多在重症早期反而有害；早期採 permissive underfeeding／漸進加量常較安全。

## 快速比較表

| 項目 | Ebb phase | Flow phase | 臨床意義 |
| :---- | :---- | :---- | :---- |
| 時間 | 傷後數小時 | 傷後數日至數週 | 決定復甦 vs 支持策略 |
| 心輸出 | 下降 | 上升（hyperdynamic） | 監測與液體目標不同 |
| 代謝率 | 下降 | 上升（hypermetabolic） | flow 期熱量需求高 |
| 主要處置 | 復甦、止血、保溫 | 營養支持、器官保護 | 對應不同介入 |

| 反應軸線 | 主要介質 | 生理效應 | 過度時後果 |
| :---- | :---- | :---- | :---- |
| Innate 發炎 | TNF-α、IL-1、IL-6 | 動員免疫、急性期反應 | SIRS、器官損傷 |
| 抗發炎 | IL-10、cortisol | 限制發炎 | 免疫抑制、續發感染 |
| 神經內分泌 | Catecholamines、cortisol、ADH | 升糖、保鈉水 | 高血糖、體液滯留 |
| 代謝 | Glucagon、insulin resistance | 動員能量受質 | 肌肉流失、負氮平衡 |

## 後續證據更新

- **重症輸液選擇偏向 balanced crystalloid**：大型 RCT（PLUS，NEJM 2022）在 ICU 成人比較 balanced multielectrolyte solution 與 0.9% saline，主要結局（90 天死亡）無顯著差異；但整體證據（含 SMART 等）傾向 balanced solution 可能降低腎臟相關不良事件，多數指引已將平衡液作為復甦首選。**適用／限制**：整體族群死亡率差異小，臨床意義主要在腎臟結局與特定次族群（如敗血症）。  
- **敗血症處置以 Surviving Sepsis Campaign 2021 為準**：相較 2019 教材，2021 版強調對疑似敗血性休克於 1 小時內給予抗生素、初始 30 mL/kg 晶體液後以動態指標與 lactate 導引續復甦、升壓首選 norepinephrine，並對經液體與升壓仍不穩者考慮 hydrocortisone。**適用／限制**：多為 weak recommendation／low certainty，須個別化，避免對非休克病人過度給液。  
- 血糖仍建議 **中度控制（約 140–180 mg/dL）** 而非過嚴，反映 NICE-SUGAR 後對低血糖風險的共識延續。**適用／限制**：適用一般重症；特殊族群（如神經重症）目標可能不同。
