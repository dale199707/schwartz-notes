# Chapter 23: Arterial Disease

## 核心整理

- **Atherosclerosis** 是多數周邊動脈疾病的共同病理：內皮損傷、脂質沉積、發炎、斑塊形成與鈣化，導致狹窄/阻塞或動脈瘤化。危險因子：吸菸、糖尿病、高血壓、高血脂、年齡。  
- **Peripheral artery disease（PAD）** 光譜由 **claudication（間歇性跛行）** 到 **chronic limb-threatening ischemia（CLTI，靜息痛、潰瘍、壞疽）**；以 **ankle-brachial index（ABI）** 篩檢（\<0.9 異常，糖尿病鈣化可能假性升高）。  
- **Claudication 一線是內科＋運動**：戒菸、他汀、抗血小板、控糖控壓，加 **supervised exercise**；多數不需血管介入。血管重建留給 **失能性跛行藥物/運動無效** 或 CLTI。  
- **CLTI 需積極血管重建** 以救肢：選擇 **open bypass** 或 **endovascular**，取決於解剖（病灶長度/位置/流出道）、導管品質（如大隱靜脈）與病人風險。  
- **Acute limb ischemia（ALI）** 六 P（pain、pallor、pulselessness、paresthesia、paralysis、poikilothermia）為急症；依 Rutherford 分級決定 **緊急取栓/溶栓/重建**；感覺運動喪失代表威脅存活的肢體，時間就是肢體。  
- **Carotid artery disease** 與缺血性中風相關；**症狀性頸動脈狹窄（≥50–70%）** 血管重建（CEA/CAS）＋最佳藥物治療可降中風；無症狀狹窄需權衡。  
- **Abdominal aortic aneurysm（AAA）**：多無症狀，破裂致命。**篩檢**（如 65–75 歲有吸菸史男性一次超音波）；**手術門檻約 ≥5.5 cm（男）或快速擴大/症狀**；女性與特定情形門檻可較低。  
- **AAA 修復**：**EVAR（血管內）** 圍手術期死亡率低但需終身監測（endoleak）；**open repair** 長期耐久但侵襲性高；選擇依解剖與風險。  
- **破裂 AAA** 為外科急症：突發腹/背痛＋低血壓＋搏動性腫塊，需緊急修復（EVAR 或 open），允許性低血壓過渡。  
- **Mesenteric ischemia**（動脈）：急性（栓塞/血栓）「疼痛與理學不成比例」，慢性（餐後腹痛、體重減輕、畏食）；需血管影像與重建。  
- **Renovascular disease**：動脈粥狀硬化或纖維肌性發育不良致腎動脈狹窄，可致高血壓/腎功能惡化；多數以藥物控制，選擇性介入。  
- **糖尿病足** 兼具神經病變、缺血與感染，需 **血流評估、清創、感染控制與 offloading** 之多科合作。

## Clinical pearl

- **claudication 先運動與藥物、CLTI 才積極重建**：對單純跛行貿然介入未必改善且有風險，是常見的過度治療陷阱。  
- **急性肢體缺血「時間就是肢體」**：出現感覺/運動喪失即威脅存活的肢體，需緊急重建，延遲致不可逆壞死——關鍵 red flag。  
- **糖尿病病人 ABI 可能假性正常**：血管鈣化使 ABI 偏高，改看 toe-brachial index 或波形，避免漏診缺血。  
- **搏動性腹部腫塊＋突發劇痛想破裂 AAA**：不要為完整影像而延誤，血流動力學不穩即送修復。

## 後續證據更新

- **CLTI 若有良好大隱靜脈，surgical bypass 可能優於 endovascular**：BEST-CLI（Farber 等，NEJM 2022）顯示對 **適合且具良好單段大隱靜脈** 導管者，**open bypass** 相較 endovascular 降低主要不良肢體事件（再介入、重大截肢）。**適用／限制**：優勢見於有良好靜脈導管者；無良好靜脈或高手術風險者 endovascular 仍為要角——需個別化解剖與風險評估。  
- **頸動脈狹窄 CEA 與 CAS 長期結果相近、依風險選擇**：CREST（Brott 等，NEJM 2010）長期顯示 CEA 與 CAS 綜合終點相當，但 **CAS 圍手術期中風較多、CEA 圍手術期心肌梗塞較多**，且年齡影響（年長者較利 CEA）。**適用／限制**：選擇需依年齡、解剖、共病與術者經驗個別化。
