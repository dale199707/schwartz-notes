# Chapter 38: Thyroid, Parathyroid, and Adrenal

## 核心整理

- **甲狀腺手術關鍵神經血管**：**recurrent laryngeal nerve（RLN，傷之聲音沙啞）**、**external branch of superior laryngeal nerve（影響高音）**、與四顆 **parathyroid glands**（保留其血供以防術後低血鈣）。  
- **甲狀腺結節評估**：先測 TSH；若低於正常，做 radionuclide scan 判斷是否為 hyperfunctioning nodule。FNA 不是每顆都做，而是依 ultrasound risk pattern 與 size threshold；cytology 用 Bethesda 分類，不確定者再結合重複 FNA、molecular testing 或 diagnostic surgery。  
- **分化型甲狀腺癌**：**papillary（最常見、淋巴轉移、預後佳）** 與 **follicular（血行轉移、需組織學看被膜/血管侵犯）**；medullary（C 細胞、calcitonin、與 MEN2/RET 相關）、anaplastic（極惡）。  
- **分化型甲狀腺癌手術（2025 ATA）**：單側、≤2 cm、cN0、無 gross extrathyroidal extension 的低風險癌以 lobectomy 為建議範圍；>2–4 cm 的低風險單側癌可 lobectomy 或 total thyroidectomy，依 disease features、後續 radioactive iodine/追蹤需求與偏好共享決策。>4 cm、gross invasion、clinically apparent nodes 或 distant metastasis 通常需 total thyroidectomy ± therapeutic node dissection。  
- **Hyperthyroidism**：Graves（瀰漫、TSH receptor 抗體）、toxic nodule/多結節性甲狀腺腫；治療含抗甲狀腺藥、放射碘、手術（壓迫、懷疑惡性或藥物失敗）。  
- **原發性副甲狀腺亢進（primary hyperparathyroidism）**：多為單一 **adenoma**，高血鈣＋PTH 不當升高；症狀含「stones, bones, groans, psychiatric moans」，多數現以無症狀高血鈣被發現。  
- **Primary hyperparathyroidism（PHPT）手術決策先於定位**：diagnosis 與 surgery indication 來自 calcium/PTH、症狀、骨骼與腎臟評估；sestamibi、超音波或 4D-CT 只在已決定手術後用於 localization，不能用「定位不到」否定診斷或手術適應症。影像 concordant 時可 focused parathyroidectomy；不一致、陰性或疑 multigland disease 時考慮 bilateral exploration。intraoperative PTH 對 focused surgery 有幫助，但不是每一場手術都強制。  
- **腎上腺皮質分區**：**GFR—glomerulosa（aldosterone）、fasciculata（cortisol）、reticularis（androgen）**；髓質分泌 catecholamines。  
- **功能性腎上腺腫瘤**：**primary aldosteronism（Conn，高血壓＋低血鉀）**、**Cushing syndrome（cortisol 過量）**、**pheochromocytoma（catecholamine，陣發高血壓、頭痛、心悸、盜汗）**。  
- **Adrenal incidentaloma** 評估兩問題：**功能性？惡性？**。2023 ESE：homogeneous、unenhanced CT ≤10 HU 可視為 benign，無論大小通常不需追加影像；1-mg dexamethasone suppression test 用於評估 autonomous cortisol secretion。metanephrines 主要用於非典型 benign adenoma（例如 HU >10）或疑 pheochromocytoma；aldosterone/renin ratio 只在 hypertension 或 unexplained hypokalemia。>4 cm 本身不是自動手術門檻，但 >4 cm 合併 heterogeneous 或 HU >20 為較高惡性風險，通常由 multidisciplinary team 討論手術。  
- **Pheochromocytoma 手術前必須 α-blockade（再 β）** 充分準備以防術中高血壓危象；順序錯誤（先 β）可致危象。  
- **腎上腺皮質癌（ACC）** 罕見、預後差；影像可疑/大腫瘤避免經腹腔鏡若疑惡性，強調完整 R0 切除。

## Clinical pearl

- **Pheochromocytoma 先 α 再 β、順序不能反**：未充分 α-blockade 或先給 β 會誘發高血壓危象——最關鍵且致命的 decision point。  
- **PHPT 不要拿定位檢查當診斷**：先確認 biochemical diagnosis 與手術適應症，再做 localization。focused surgery 常用 intraoperative PTH；若做標準 bilateral exploration，是否使用依中心與個案決定。  
- **甲狀腺全切後盯低血鈣**：術後 Chvostek/Trousseau、口周麻木是副甲狀腺受損的 red flag，需監測與補鈣。  
- **腎上腺切片前一定要先想 pheochromocytoma**：典型 homogeneous ≤10 HU adenoma 依 ESE 2023 可不常規測 metanephrines；但任何 indeterminate/suspicious adrenal mass 若考慮 biopsy 或手術，須先完成適當 hormonal evaluation，尤其排除 pheochromocytoma，避免操作誘發危象。

## 快速比較表

| 甲狀腺癌 | 特徵 | 轉移途徑/預後 |
| :---- | :---- | :---- |
| Papillary | 最常見 | 淋巴、預後佳 |
| Follicular | 需看被膜/血管侵犯 | 血行、預後尚可 |
| Medullary | C 細胞、calcitonin、MEN2 | 早期淋巴、需查 RET |
| Anaplastic | 老年、快速 | 極差 |

| 功能性腎上腺瘤 | 過量荷爾蒙 | 典型表現 |
| :---- | :---- | :---- |
| Conn（aldosteronism） | Aldosterone | 高血壓、低血鉀 |
| Cushing | Cortisol | 中心肥胖、高血糖 |
| Pheochromocytoma | Catecholamine | 陣發高血壓、頭痛心悸 |
| ACC | 多樣/無功能 | 大腫瘤、預後差 |

## 後續證據更新

- **DTC 現以 2025 ATA guideline 為最新參考**：治療範圍已更 risk-adapted；低風險 unilateral disease 多可 lobectomy，active surveillance 僅適用嚴格篩選的小型低風險 papillary thyroid carcinoma，且必須能可靠追蹤。radioactive iodine 與 TSH target 亦按 recurrence risk 與 response-to-therapy 動態調整。  
- **Asymptomatic PHPT 的手術門檻（Fifth International Workshop）**：即使無典型症狀，只要任一項成立便建議 parathyroidectomy：serum calcium > upper limit 1.0 mg/dL、vertebral fracture 或 T-score ≤−2.5、eGFR/creatinine clearance <60 mL/min、nephrolithiasis/nephrocalcinosis 或 hypercalciuria、或 age <50 years；所有 symptomatic PHPT 原則上均考慮手術。  
- **ESE 2023 adrenal incidentaloma 去除單純 size rule**：homogeneous ≤10 HU 無論大小通常不需追蹤影像；>4 cm 只有在 heterogeneous 或 HU >20 等可疑特徵下才構成較高 risk。mild autonomous cortisol secretion 的 surgery 需結合 comorbidity、年齡、病人偏好與 MDT 判斷。  
- **Pheochromocytoma preoperative preparation**：功能性 tumor 術前 α-adrenergic blockade、容量與鹽分補充仍是主流 guideline practice，β-blocker 僅在充分 α-blockade 後用於 tachyarrhythmia。少數專家中心對 normotensive/selected tumors 的作法不同，應由 endocrine/anesthesia/surgical team 個別化。  
- **Adrenocortical carcinoma（ACC）**：目標是在 expert center 完整、避免 tumor rupture 的 R0 resection；若有 invasion 或高度懷疑惡性通常選 open approach。selected small、well-circumscribed、無 invasion 病灶才可由具經驗團隊考慮 minimally invasive surgery。
