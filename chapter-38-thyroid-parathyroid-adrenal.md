# Chapter 38: Thyroid, Parathyroid, and Adrenal

## 核心整理

- **甲狀腺手術關鍵神經血管**：**recurrent laryngeal nerve（RLN，傷之聲音沙啞）**、**external branch of superior laryngeal nerve（影響高音）**、與四顆 **parathyroid glands**（保留其血供以防術後低血鈣）。  
- **甲狀腺結節評估**：TSH（低→做掃描找 hot nodule）、超音波特徵分級，並以 **fine-needle aspiration（FNA）＋Bethesda 分類** 決定處置；不確定者可加分子檢測。  
- **分化型甲狀腺癌**：**papillary（最常見、淋巴轉移、預後佳）** 與 **follicular（血行轉移、需組織學看被膜/血管侵犯）**；medullary（C 細胞、calcitonin、與 MEN2/RET 相關）、anaplastic（極惡）。  
- **甲狀腺癌手術**：依風險行 lobectomy 或 total thyroidectomy ± 中央區淋巴廓清；術後選擇性放射碘與 TSH 抑制。  
- **Hyperthyroidism**：Graves（瀰漫、TSH receptor 抗體）、toxic nodule/多結節性甲狀腺腫；治療含抗甲狀腺藥、放射碘、手術（壓迫、懷疑惡性或藥物失敗）。  
- **原發性副甲狀腺亢進（primary hyperparathyroidism）**：多為單一 **adenoma**，高血鈣＋PTH 不當升高；症狀含「stones, bones, groans, psychiatric moans」，多數現以無症狀高血鈣被發現。  
- **副甲狀腺手術**：術前定位（sestamibi、超音波、4D-CT）後行 **minimally invasive parathyroidectomy**，並以 **術中 PTH 監測** 確認切除足夠（PTH 適當下降）。  
- **腎上腺皮質分區**：**GFR—glomerulosa（aldosterone）、fasciculata（cortisol）、reticularis（androgen）**；髓質分泌 catecholamines。  
- **功能性腎上腺腫瘤**：**primary aldosteronism（Conn，高血壓＋低血鉀）**、**Cushing syndrome（cortisol 過量）**、**pheochromocytoma（catecholamine，陣發高血壓、頭痛、心悸、盜汗）**。  
- **Adrenal incidentaloma** 評估兩問題：**功能性？惡性？**——做荷爾蒙檢測（排除 pheo、Cushing、aldosteronism）與影像特徵（大小、含脂、washout）；**大（如 \>4 cm）或影像可疑或功能性** 者手術。  
- **Pheochromocytoma 手術前必須 α-blockade（再 β）** 充分準備以防術中高血壓危象；順序錯誤（先 β）可致危象。  
- **腎上腺皮質癌（ACC）** 罕見、預後差；影像可疑/大腫瘤避免經腹腔鏡若疑惡性，強調完整 R0 切除。

## Clinical pearl

- **Pheochromocytoma 先 α 再 β、順序不能反**：未充分 α-blockade 或先給 β 會誘發高血壓危象——最關鍵且致命的 decision point。  
- **副甲狀腺切除靠術中 PTH「確認」**：PTH 適當下降才代表病灶切乾淨，否則要繼續找（多腺體/異位）——避免手術失敗。  
- **甲狀腺全切後盯低血鈣**：術後 Chvostek/Trousseau、口周麻木是副甲狀腺受損的 red flag，需監測與補鈣。  
- **incidentaloma 一律先排除 pheo**：對任何腎上腺腫塊做手術或切片前，先生化排除嗜鉻細胞瘤，否則操作誘發危象。

## 快速比較表

| 甲狀腺癌 | 特徵 | 轉移途徑/預後 |
| :---- | :---- | :---- |
| Papillary | 最常見 | 淋巴、預後佳 |
| Follicular | 需看被膜/血管侵犯 | 血行、預後尚可 |
| Medullary | C 細胞、calcitonin、MEN2 | 早期淋巴、需查 RET |
| Anaplastic | 老年、快速 | 極差 |

| 功能性腎上腺瘤 | 過量荷爾蒙 | 典型表現 |
| :---- | :---- | :---- |
| Conn（aldosteronism） | Aldosterone | 高血壓、低血鉀 |
| Cushing | Cortisol | 中心肥胖、高血糖 |
| Pheochromocytoma | Catecholamine | 陣發高血壓、頭痛心悸 |
| ACC | 多樣/無功能 | 大腫瘤、預後差 |

## 後續證據更新

- **分化型甲狀腺癌（DTC）現以 2025 ATA guideline 為最新參考**：2015 ATA 對 thyroid nodule 超音波風險分層與 FNA 的架構仍具實用性；但 DTC 的診斷後管理已由 **2025 ATA guideline** 更新。低風險小乳突癌的 **lobectomy、選擇性 radioactive iodine 與主動監測（active surveillance）** 仍屬去升級治療的核心方向。**適用／限制**：主動監測僅限嚴格篩選、可規律追蹤的低風險病人。  
- **手術範圍與放射碘採風險導向而非一律升級**：低風險族群可避免不必要全切與放射碘；中高風險、具轉移或侵襲特徵者仍須依風險分層採較積極的手術與輔助治療。
