# Chapter 53: Skills and Simulation

## 核心整理

- 外科技能學習遵循 **從認知→整合→自動化** 的階段(Fitts-Posner motor learning model);刻意練習(deliberate practice)與回饋是精進關鍵。  
- **模擬(simulation)** 讓學習者在 **不傷害病人** 的環境反覆練習,涵蓋 **bench/box trainer、虛擬實境(VR)、動物/大體、混合模型與情境模擬(team-based)**。  
- **能力導向訓練(competency-based)** 取代單純「時數/件數」:以達到明確技能標準為畢業依據,而非僅計手術次數。  
- **評量工具**:結構化評分(如 OSATS)、目標導向的檢核表與整體評分,提供客觀、可重複的技能評估與回饋。  
- **技能轉移(transfer)**:模擬訓練的價值在能否 **轉移到真實手術室表現**;設計良好的模擬課程可縮短學習曲線。  
- **Proficiency-based progression**:先在模擬達到專家水準的熟練度,再進入臨床操作,可提升安全與效率。  
- **團隊與危機模擬**:CRM、危機情境演練提升非技術性技能(溝通、領導、情境覺察),與技術訓練互補。  
- **回饋與 coaching**:即時、具體、建設性的回饋與(影片)手術教練促進持續改進,適用於受訓者與資深外科醫師。  
- **微創/機器人技能** 特別受益於模擬:視覺-動作轉換、器械操作可在模擬平台安全建立基礎。  
- **課程設計與維持**:模擬需納入正式課程、定期練習以維持技能(skill decay),並結合評量與認證。

## Codex 驗證補強（2026）

- Simulation 是 patient-safe 的 deliberate practice 平台，但只是 curriculum 的一部分；要能 transfer 到 OR，需明確 learning objective、分段任務、重複練習、即時 feedback、有效的 assessment 與 supervised clinical progression。
- Mastery/proficiency-based progression 的 benchmark 必須由透明且具 validity evidence 的 standard-setting method 建立；不能只把少數「expert 平均分數」當成普遍及格線。不同 simulator、任務與族群的 cut score 不可直接互換。
- 早期 VR laparoscopic trial（如 Seymour 2002）是 foundational evidence，但樣本小、任務與現代平台不同；可支持 simulation 改善特定 technical performance，不能單獨證明所有手術的 patient outcome 都改善。
- 評估採 Kirkpatrick framework 時，要區分 reaction、learning、behavior transfer 與 patient/system outcome；skill score 上升是必要但不充分的 surrogate。
- 2022 standard-setting review 發現方法與 expert 定義高度異質，提醒每個訓練計畫必須驗證可靠性、公平性與 downstream transfer。[Standard-setting systematic review](https://pubmed.ncbi.nlm.nih.gov/34520423/)

## Clinical pearl

- **先在模擬達標,再上真人**:proficiency-based progression 讓學習曲線的風險發生在模擬而非病人身上——核心安全 decision point。  
- **回饋要即時且具體**:模糊的「做得不錯」無助改進;具體、可操作的回饋才驅動進步。  
- **技能會退化,要定期維持**:不常做的技能需刻意複訓,別假設一次學會就永久。  
- **技術與團隊訓練都要**:再好的個人技術也需非技術性技能與團隊演練配合。

## 快速比較表

| 模擬類型 | 訓練重點 | 特點 |
| :---- | :---- | :---- |
| Box/bench trainer | 基本操作、縫合 | 便宜、可近 |
| VR simulator | 微創/機器人技能 | 客觀指標、可重複 |
| 動物/大體 | 真實組織手感 | 成本/倫理考量 |
| 情境/團隊模擬 | 非技術性技能 | 溝通、危機處理 |

| 概念 | 內容 |
| :---- | :---- |
| Deliberate practice | 刻意、有回饋的練習 |
| Competency-based | 以達標而非時數 |
| OSATS | 結構化技能評分 |
| Skill decay | 技能退化,需複訓 |

## 後續證據更新

- **VR 模擬訓練可轉移到手術室表現**:Seymour 等(Ann Surg 2002)的隨機試驗顯示,接受 **VR 模擬訓練** 的住院醫師在真實腹腔鏡膽囊切除中 **操作更快、錯誤更少**,提供「模擬→手術室」技能轉移的實證基礎,推動模擬納入外科訓練。**適用／限制**:為早期奠基性研究;不同平台與技能的轉移程度不一,需結構化課程與評量配合。  
- 相較 2019 教材,**能力導向訓練、proficiency-based progression 與機器人模擬** 更為普及,並結合客觀評量與(影片)coaching。**適用／限制**:需課程、設備與師資投入,成效依實施品質而定。
