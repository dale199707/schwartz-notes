# Chapter 47: Surgical Considerations in Older Adults

## 核心整理

- 老年病人手術風險不能只看 **年齡(chronological age)**,而應看 **生理年齡與 frailty(衰弱)、共病、功能與認知儲備**。  
- **Frailty(衰弱)** 是多系統生理儲備下降的症候群,**獨立預測術後併發症、住院天數、機構安置與死亡**;可用 Fried 表型或 frailty index/量表評估。  
- **老年症候群(geriatric syndromes)** 影響結果:認知障礙/失智、譫妄風險、跌倒、失禁、營養不良、多重用藥(polypharmacy)、感官障礙。  
- **術後譫妄(postoperative delirium)** 常見且與不良預後相關;預防重於治療——避免誘因(疼痛控制不足、某些藥物、睡眠剝奪、感染、脫水、身體約束)。  
- **綜合老年評估(comprehensive geriatric assessment, CGA)** 系統性評估醫療、功能、認知、心理與社會面,協助風險分層與優化。  
- **藥物**:老人藥物動力/動態改變,避免 potentially inappropriate medications(如 Beers criteria),圍手術期審視多重用藥。  
- **生理儲備下降**:心肺、腎、免疫與體溫調節儲備減少,對出血、感染、輸液與藥物耐受度較低,需更精細管理。  
- **營養與失能**:術前營養不良與功能受限預測不良結果;prehabilitation(術前體能/營養/認知準備)可能改善恢復。  
- **手術決策以「目標一致的照護(goal-concordant care)」為核心**:與病人討論預後、功能與生活品質期望,而非只問「能不能開」。  
- **圍手術期整合照護**:老年—外科共管模式(如 co-management、老年評估團隊)可降低併發症與譫妄。  
- **出院與復健規劃** 應及早介入:功能、居家支持與後續照護是老人術後結果的重要決定因子。

## Codex 驗證補強（2026）

- 年齡本身不是手術禁忌；frailty、cognition、function、nutrition、polypharmacy、sensory impairment 與 social support 才能更直接揭示 vulnerability。篩檢陽性後應連結 Comprehensive Geriatric Assessment 與可執行的介入，而非只貼上「高風險」標籤。
- Delirium prevention 以 multicomponent bundle 為主：orientation、sleep、mobility、hearing/vision aids、hydration、pain control、避免 urinary catheter 與高風險藥物。除 alcohol/benzodiazepine withdrawal 等特定情境，避免 benzodiazepine；不常規使用 prophylactic antipsychotic。
- 發生 delirium 時先找 reversible cause（hypoxia、infection、retention、constipation、drug toxicity、withdrawal、metabolic disorder）；只有嚴重 agitation 威脅安全且非藥物措施失敗時，才短期、低劑量、每日重評 pharmacologic restraint。
- Prehabilitation 應以營養、exercise、smoking、anemia 與心理支持個別化；目前跨手術族群 evidence 異質，不能保證降低所有 complication 或 mortality。
- Shared decision making 要比較 operation、less-invasive option、nonoperative care 與 time-limited trial，說明 time-to-benefit、可能喪失獨立性、照護負擔及 patient-defined unacceptable outcome。

## Clinical pearl

- **年齡不是禁忌,frailty 才是關鍵**:以衰弱與功能儲備而非出生年份決定風險與策略——核心 decision point。  
- **譫妄要「預防」**:主動處理疼痛、避免高風險藥物、維持定向/睡眠/水分,比事後處置有效。  
- **術前先問「病人在乎什麼」**:目標一致的照護讓決策符合病人對功能與生活品質的期望,避免無益的積極手術。  
- **多重用藥要圍手術期審視**:停用不當/高風險藥物可降低譫妄、跌倒與交互作用。

## 快速比較表

| 評估面向 | 工具/內容 | 意義 |
| :---- | :---- | :---- |
| Frailty | Fried 表型/frailty index | 預測併發症/死亡 |
| 認知 | 篩檢(如 Mini-Cog) | 譫妄與同意能力 |
| 功能 | ADL/IADL、步態 | 恢復與出院規劃 |
| 營養 | 體重/白蛋白/篩檢 | 癒合與感染風險 |

| 術後風險 | 誘因 | 對策 |
| :---- | :---- | :---- |
| 譫妄 | 藥物、疼痛、脫水 | 預防、非藥物介入 |
| 跌倒 | 衰弱、藥物 | 早期活動、環境 |
| 功能退化 | 臥床、失能 | 復健、prehabilitation |

## 後續證據更新

- **Frailty 是勝過年齡的術前風險指標**:Makary 等(J Am Coll Surg 2010)證實 **frailty 獨立預測老年病人術後併發症、住院與機構安置**,推動以衰弱評估納入術前風險分層。**適用／限制**:量表眾多、無單一金標準;評估後仍需連結到具體優化與決策。  
- 相較 2019 教材,**老年—外科共管、prehabilitation 與譫妄預防組合(bundle)** 更受重視,並強調 **目標一致的照護與知情決策**。**適用／限制**:多為觀察性/實作證據,需制度與跨科資源支持。
