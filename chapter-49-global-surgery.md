# Chapter 49: Global Surgery

## 核心整理

- **Global surgery** 關注全球(尤其低中所得國家 LMIC)**外科、麻醉與產科照護的可近性、安全與可負擔性**;外科疾病是全球疾病負擔的重要且長期被低估的一環。  
- 三大障礙(the "three delays" 概念延伸):**尋求照護的延遲、抵達機構的延遲、接受適切照護的延遲**;交通、費用、人力與設施皆為瓶頸。  
- **龐大未滿足需求**:全球有數十億人無法及時取得安全、可負擔的外科與麻醉照護;許多可治癒的外科疾病因此致死或致殘。  
- **可負擔性與財務風險**:外科費用常造成 **災難性醫療支出(catastrophic expenditure)**,使家庭陷入貧困,是可近性以外的關鍵面向。  
- **人力與能力建構**:外科/麻醉/護理人力嚴重短缺與分布不均;**能力建構、訓練與留任** 比單次任務更能永續改善。  
- **基本外科能力(essential/bellwether procedures)**:以 **剖腹產、剖腹手術、開放性骨折處置** 作為衡量一地是否具備基本急症外科能力的指標。  
- **安全與品質**:安全麻醉、輸血、感染控制與基本設備(如脈搏血氧監測)是降低 LMIC 手術死亡的基礎。  
- **永續與在地主導**:有效的全球外科強調 **在地主導、系統強化與長期夥伴關係**,而非短期垂直計畫或「手術旅行團」式介入。  
- **資料與監測**:缺乏可靠的外科結果資料使規劃困難;建立登錄與指標是改善的前提。  
- **公平與倫理**:尊重在地情境、避免造成依賴或傷害、公平分配資源與知識轉移。

## Codex 驗證補強（2026）

- 「50 億人缺乏安全可負擔手術、每年還需要 1.43 億手術、約 3,300 萬人因手術支出遭遇 financial catastrophe」是 Lancet Commission 以 **2015 baseline/modeling** 提出的估計；可用來說明規模，但不應當成 2026 的即時精確人口計數。
- Bellwether procedures（cesarean delivery、laparotomy、open-fracture care）是衡量系統 capacity 的 proxy，不代表只要三項可做就已具備完整、安全、可負擔的 surgical system。
- National Surgical, Obstetric, and Anesthesia Plan（NSOAP）需涵蓋 infrastructure、workforce、service delivery、information management、financing 與 governance；只捐設備而沒有 maintenance、supply chain、training 與 referral network，常無法持續。
- 國際合作應由 local priorities 與 local leadership 主導，採雙向能力建構、長期 outcome/quality monitoring 與透明 authorship/ownership；短期 mission 應接上 continuity、complication management 與在地系統，而非只報手術量。

## Clinical pearl

- **外科是公共衛生議題**:安全外科與麻醉的可近性攸關大量可預防的死亡與失能,不應被視為「奢侈」。  
- **衡量能力看 bellwether procedures**:剖腹產、剖腹、開放性骨折的可及性與安全,是評估一地急症外科能力的實用指標。  
- **永續勝過一次性任務**:能力建構與系統強化的長期效益,遠大於短期手術任務。  
- **當心災難性支出**:改善可近性同時需保護家庭免於因手術致貧,否則「可得」仍不「可負擔」。

## 快速比較表

| 障礙(delay) | 內容 | 對策方向 |
| :---- | :---- | :---- |
| 尋求照護 | 認知、費用、信任 | 教育、財務保護 |
| 抵達機構 | 交通、距離 | 轉診系統、基礎建設 |
| 接受照護 | 人力、設備、品質 | 能力建構、供應鏈 |

| 面向 | 指標/概念 | 意義 |
| :---- | :---- | :---- |
| 可近性 | 及時到達手術能力 | 是否得到照護 |
| 安全 | 圍手術期死亡率 | 照護品質 |
| 可負擔 | 災難性支出比例 | 財務保護 |
| 能力 | Bellwether procedures | 基本急症外科 |

## 後續證據更新

- **《Lancet Commission on Global Surgery》確立全球外科議程**:Meara 等(2015,Global Surgery 2030)量化 **全球未滿足外科需求、人力與財務缺口**,提出到 2030 年的目標與指標(可近性、安全、財務保護),使全球外科成為全球衛生與發展的正式議題。**適用／限制**:為政策/倡議文件,實際改善依各國投入與系統強化而定。  
- 相較 2019 教材,**國家外科、產科與麻醉計畫(NSOAPs)** 與在地資料登錄的推動增加,強調系統性與永續性。**適用／限制**:落實程度地區差異大,需長期資源與政治承諾。
