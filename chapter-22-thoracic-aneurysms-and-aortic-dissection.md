# Chapter 22: Thoracic Aneurysms and Aortic Dissection

## 核心整理

- **Thoracic aortic aneurysm（TAA）** 依位置分 root/ascending、arch、descending、thoracoabdominal（Crawford 分型 I–V）。ascending 多與 **cystic medial degeneration**、二葉主動脈瓣（bicuspid aortic valve）相關；descending 多與 **atherosclerosis／高血壓** 相關。  
- **遺傳性胸主動脈疾病（HTAD）**：**Marfan（FBN1）、Loeys-Dietz（TGFBR）、vascular Ehlers-Danlos（COL3A1）**、familial TAAD、Turner 症候群等，發病早、風險高，手術門檻較低並需家族篩檢。  
- **多數 TAA 無症狀**，偶因壓迫出現聲音沙啞（喉返神經）、吞嚥困難、咳嗽；症狀性或快速擴大是不良徵兆。破裂/急性擴大表現為劇烈胸背痛。  
- **手術門檻（傳統概念）**：degenerative ascending 約 **≥5.5 cm**、descending 約 **≥6.0 cm**（endovascular 可較低）；**結締組織疾病、bicuspid valve、家族史或快速擴大（\>0.5 cm/年）** 者門檻下修。  
- **Aortic dissection 分類**：**Stanford A**（含 ascending，不論起源）與 **Stanford B**（僅 descending，左鎖骨下動脈遠端起）；DeBakey I/II/III。分型直接決定手術 vs 內科策略。  
- **Stanford A dissection 為外科急症**：立即開刀（置換 ascending aorta ± root/arch），否則死亡率極高（每小時累加）；併發 tamponade、AR、冠脈或腦血管受累致命。  
- **Uncomplicated Stanford B dissection** 傳統以 **內科為主**：控制心率與血壓（**先 β-blocker 降 dP/dt，再視需要加血管擴張劑**），目標 SBP \~100–120、HR \<60。  
- **Complicated Stanford B**（malperfusion、破裂、持續疼痛、無法控制高血壓、快速擴大）需介入，現多採 **TEVAR（thoracic endovascular aortic repair）**。  
- **初步藥物控制原則**：**anti-impulse therapy**——先 β-blockade 降低左心室收縮速率（dP/dt）再降壓；單用血管擴張劑會反射性心搏過速、增加剪力，是禁忌順序。  
- **影像診斷首選 CT angiography**（快速、可判斷範圍、entry tear、malperfusion）；不穩定或腎功能考量可用 TEE。  
- **修復方式**：open（cardiopulmonary bypass、視弓部需 circulatory arrest 與腦保護）vs **endovascular（TEVAR）**；descending/thoracoabdominal 病灶 endovascular 侵襲性較低但需合適 landing zone。  
- **主要併發症**：open 手術的 **spinal cord ischemia（截癱）**（thoracoabdominal 尤甚，術中以 CSF drainage、維持灌流壓保護脊髓）、腎損傷、中風；TEVAR 有 endoleak、retrograde dissection、access 併發症。

## Clinical pearl

- **撕裂痛＋兩側血壓/脈搏不對稱 \= 想 dissection**：突發撕裂性胸背痛、脈搏差、新發 AR 雜音或縱膈增寬，是不可錯過的 red flag；漏診致命。  
- **降壓前先降心率**：dissection 一律 **β-blocker 優先**，否則單純血管擴張劑增加 dP/dt 與剪力反而擴大撕裂——關鍵 decision point。  
- **A 型就是要開、B 型先內科**：分型決定路徑；uncomplicated B 過度積極手術未必有利，complicated B 才介入。  
- **Thoracoabdominal 修復防截癱**：CSF drainage、維持近端血壓與脊髓灌流是預防 spinal cord ischemia 的核心，不能事後補救。

## 快速比較表

| 分型 | 累及範圍 | 首選處置 | 備註 |
| :---- | :---- | :---- | :---- |
| Stanford A | 含 ascending | 緊急手術 | 死亡率隨時間累加 |
| Stanford B（uncomplicated） | 僅 descending | 內科（anti-impulse） | 控 HR/BP |
| Stanford B（complicated） | descending＋併發症 | TEVAR/介入 | malperfusion/破裂 |

| TAA 位置 | 常見病因 | 傳統手術門檻\* |
| :---- | :---- | :---- |
| Ascending（degenerative） | Cystic medial degeneration、BAV | \~≥5.5 cm |
| Descending | Atherosclerosis、HTN | \~≥6.0 cm（endo 可較低） |
| 結締組織疾病相關 | Marfan/Loeys-Dietz 等 | 門檻下修、需家族篩檢 |
| 快速擴大 | 任一 | \>0.5 cm/年即考慮 |

\*門檻為概念性整理，實際依 2022 ACC/AHA 指引與病人因素個別化。

## 後續證據更新

- **2022 ACC/AHA Aortic Disease Guideline 為現行整合基準**：相較 2019 教材，重點更新包括——對 **經驗豐富的多科 aortic team／high-volume center**，符合條件者 **ascending aorta 手術門檻可下修至 5.0 cm**；納入 **身高／體表面積校正的直徑指標（如 cross-sectional area/height）**；強調 **遺傳性 HTAD 的基因檢測與家族篩檢**，且不同基因（如 Loeys-Dietz、vEDS）採不同門檻；並對 shared decision-making 與影像追蹤標準化提出建議。**適用／限制**：部分下修門檻的前提是 **在有經驗的中心**；一般中心仍偏保守。  
- **Uncomplicated Stanford B 之 TEVAR 角色**：INSTEAD-XL（Circ Cardiovasc Interv 2013）長期資料顯示，對特定 uncomplicated 慢性 B 型 dissection，**在最佳藥物治療之外加 TEVAR 可改善晚期主動脈相關存活與 aortic remodeling**。**適用／限制**：益處主要見於 **晚期（\>2 年）** 結局與選定病人，非所有 uncomplicated B 型皆須早期介入；急性期仍以 anti-impulse 內科為基礎。
