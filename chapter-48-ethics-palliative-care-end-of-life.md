# Chapter 48: Ethics, Palliative Care, and Care at the End of Life

## 核心整理

- 醫學倫理四原則:**autonomy(自主)、beneficence(行善)、non-maleficence(不傷害)、justice(公平)**;外科決策常需在四者間權衡。  
- **知情同意(informed consent)** 要件:揭露(風險、益處、替代方案含不手術)、理解、自願、能力(capacity);外科同意應涵蓋合理替代與不確定性。  
- **決策能力(capacity)** 是特定決定、可變動的臨床判斷(理解、評價、推理、表達選擇);與法律上的 competence 不同。  
- **代理決策**:病人喪失能力時,依 **預立醫療指示(advance directive)、醫療委任代理人** 與 **替代判斷(substituted judgment)→最佳利益** 的層級決策。  
- **Surgical buy-in / 術前照護計畫**:外科手術常帶有術後積極維生的隱含期待,術前應討論 **若發生嚴重併發症時的處置界線與目標**。  
- **DNR/DNI 與手術**:進入手術室不自動取消 DNR;應 **術前重新討論(re-evaluation)** 麻醉/復甦的暫時性調整與病人意願。  
- **Palliative care(緩和照護)** 聚焦 **症狀控制與生活品質**,可與積極治療 **並行(非等於臨終)**;外科亦有 **palliative surgery/procedures**(如減壓、造口、止血)緩解症狀。  
- **症狀處置**:疼痛(含 opioid 適當使用)、噁心、呼吸困難、腸阻塞等的緩解;目標為舒適與功能。  
- **臨終照護(end-of-life)**:hospice 模式、預立醫療照護諮商(ACP)、撤除/不啟動維生治療的倫理與程序、家屬支持與哀傷照護。  
- **無效醫療與目標設定**:當治療無法達成病人目標時,應誠實溝通並轉向舒適導向照護;避免非有益的積極介入。  
- **溝通技巧**:壞消息告知(如 SPIKES)、共享決策、家庭會議;誠實、同理與尊重病人價值是核心。

## Codex 驗證補強（2026）

- Perioperative DNR 應做 **required reconsideration**：術前由 patient/surrogate、surgeon、anesthesiologist 討論哪些 intervention 可接受、適用期間與恢復條件，明確記錄；不應自動停用或自動照原指示完全不變。
- Decision-making capacity 是 **decision-specific、time-specific**：評估能否理解資訊、appreciate 後果、reason about options 並表達穩定選擇；delirium、語言或聽力障礙應先矯正可逆障礙，不能只因選擇與醫療團隊不同就判定無 capacity。
- 避免只用「futility」終止對話；應清楚說明 intervention 達成 patient-defined goal 的機率、傷害與替代方案，遇持續衝突時啟動 ethics consultation 與機構的 fair-process policy。
- **Palliative surgery** 的適應症不是「還能不能切」，而是 symptom relief 的可能性是否高於 recovery burden、complication、住院時間及剩餘生命中的 time cost；術前應定義成功目標與失敗時的照護路徑。
- Early palliative-care trial 的 survival benefit 多來自特定 cancer population，不能概括成所有外科患者都延長存活；其較一致價值是 symptom control、communication、caregiver support 與 goal-concordant care。

## Clinical pearl

- **進手術室不自動取消 DNR**:需術前明確重新討論復甦意願與範圍,忽略是常見倫理陷阱。  
- **術前談「萬一」**:高風險手術前設定「若嚴重併發症時的界線與目標」,避免 surgical buy-in 帶來非病人所願的長期維生。  
- **緩和 ≠ 放棄**:palliative care 可與治癒性治療並行,及早介入改善症狀與品質——重要 decision point。  
- **能力是針對特定決定**:capacity 隨決策複雜度與時間變動,需具體評估而非一概而論。

## 快速比較表

| 倫理原則 | 內涵 | 外科情境 |
| :---- | :---- | :---- |
| Autonomy | 尊重自主 | 知情同意、拒絕治療 |
| Beneficence | 行善 | 選擇對病人有益處置 |
| Non-maleficence | 不傷害 | 權衡手術風險 |
| Justice | 公平 | 資源分配、可近性 |

| 決策層級 | 依據 | 說明 |
| :---- | :---- | :---- |
| 病人自主 | 有能力時本人意願 | 首要 |
| 預立指示/代理人 | 事前表達/委任 | 喪失能力時 |
| 替代判斷 | 推測病人會選什麼 | 依其價值 |
| 最佳利益 | 客觀利益權衡 | 無從得知意願時 |

## 後續證據更新

- **早期緩和照護可改善生活品質甚至存活**:Temel 等(NEJM 2010)在轉移性非小細胞肺癌顯示,標準腫瘤治療 **並行早期 palliative care**,改善生活品質與情緒,且該研究中 **存活反而較長**,確立「早期整合緩和照護」的實證基礎。**適用／限制**:研究族群為特定癌症,推廣到各外科情境需個別化,但支持及早介入的原則。  
- 相較 2019 教材,**預立醫療照護諮商(ACP)、目標一致的照護與外科緩和照護整合** 更受強調,並納入高風險手術的術前溝通流程。**適用／限制**:多屬實作/共識,需制度與溝通訓練支持。
