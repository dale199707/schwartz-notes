# Chapter 06: Surgical Infections

## 核心整理

- 外科感染的結果取決於三方拉鋸：**病原菌數量與毒力、宿主防禦、局部環境（血流、壞死組織、異物、死腔）**；外科的核心貢獻常在後兩者（source control）。  
- **Source control 是外科感染治療的靈魂**：引流膿瘍、清創壞死組織、移除感染異物／導管、解除阻塞或穿孔——沒有 source control，抗生素難以奏效。  
- **手術傷口分類**（協助描述 contamination burden）：**clean、clean-contaminated、contaminated、dirty/infected**。SSI 風險還受 procedure、植入物、宿主與 surveillance 定義影響，舊式固定百分比只能作粗略教學值，不能當個別病人的預測率。  
- **Surgical antibiotic prophylaxis 原則**：通常切皮前 **60 分鐘內**完成給藥（需較長 infusion 的 vancomycin/fluoroquinolone 提前至 120 分鐘）、選擇符合 procedure flora 的窄譜藥、長手術或大量失血時 redose。對 clean/clean-contaminated procedure，CDC 建議切口關閉後不要再追加 prophylactic dose，即使有 drain；其他 procedure 依專科與院內 protocol，但不得以「最多 24 小時」合理化無指徵延長。  
- **SSI（surgical site infection）** 分 superficial incisional、deep incisional、organ/space；診斷靠臨床（紅腫熱痛、化膿）加影像；處置為 **開創引流 ± 抗生素**。  
- **Source of SSI**：多為病人自身菌叢；預防重點含適當皮膚準備、正常體溫與血糖、組織灌流氧合、無菌技術。  
- **Intra-abdominal infection**：局限者為 abscess（可經皮引流），瀰漫者為 peritonitis（需手術）；常為 polymicrobial（含 gram-negative 與 anaerobes），抗生素需涵蓋。  
- **Skin/soft tissue infections** 光譜：cellulitis（抗生素）、abscess（切開引流為主）、到 **necrotizing soft tissue infection**。  
- **Necrotizing fasciitis 是外科急症**：疼痛與全身毒性和外觀不成比例、皮膚捻髮音、水泡、快速進展；處置為 **緊急且徹底的清創＋廣效抗生素＋支持**，延遲清創死亡率極高。  
- **常見院內／導管相關感染**：CLABSI、CAUTI、VAP、C. difficile colitis；預防以 bundle 與移除不必要導管為主。  
- **Clostridioides difficile infection（CDI）**：停用不必要的致病抗生素並隔離。初次 non-fulminant episode 優先 **fidaxomicin**（資源允許），oral vancomycin 為可接受替代；metronidazole 不再是可取得前兩者時的首選。Fulminant CDI 需 high-dose oral/NG vancomycin＋IV metronidazole、早期外科會診，並依 toxic megacolon、perforation、shock/organ failure 評估 subtotal colectomy 或 diverting loop ileostomy＋lavage。  
- **Sepsis 概念（Sepsis-3）**：感染引起、以 **SOFA 上升 ≥2** 表示的器官功能失常；septic shock 是在充分 volume resuscitation 後仍需 vasopressor 維持 MAP ≥65 mmHg 且 lactate >2 mmol/L 的高風險次群。處置強調及時抗生素、依灌流反應復甦與盡快 source control。

## Clinical pearl

- **抗生素治不好就想「source 沒控制」**：持續發燒/白血球升高常代表 **未引流的膿或未清的壞死**，不是換更強抗生素——這是外科感染最關鍵的 decision point。  
- **疼痛與外觀不成比例＝壞死性感染警訊**：necrotizing infection 早期皮膚變化可輕微，劇痛、全身毒性、捻髮音是不可等待的 red flag，先進手術室探查。  
- **預防抗生素時機比藥物選擇更常出錯**：切皮前 1 小時內給、長手術 redose、術後及時停，是降低 SSI 又避免抗藥性的實務重點。  
- **膿瘍優先引流勝過單靠抗生素**：可引流的膿（腹腔、軟組織）以引流為本，抗生素為輔。

## 快速比較表

| 傷口分類 | 例子 | 相對 SSI 風險 | 抗生素角色 |
| :---- | :---- | :---- | :---- |
| Clean | 甲狀腺、疝氣（無進入空腔） | 最低；植入物會改變決策 | 依 procedure/implant protocol |
| Clean-contaminated | 進入呼吸/消化/泌尿道且可控 | 較 clean 高 | 通常 prophylaxis |
| Contaminated | 明顯溢漏、新鮮外傷 | 高 | 視污染程度為 prophylaxis 或早期治療 |
| Dirty/infected | 已有膿、壞死或既存 perforation | 最高 | 治療性 antibiotics＋source control |

| 軟組織感染 | 關鍵特徵 | 一線處置 |
| :---- | :---- | :---- |
| Cellulitis | 紅腫、無膿腔 | 抗生素 |
| Abscess | 局限膿腔、波動感 | 切開引流 |
| Necrotizing | 劇痛、毒性、捻髮音、快速進展 | 緊急清創＋廣效抗生素 |
| C. difficile infection | 抗生素後水瀉、toxin/NAAT 結合臨床 | Fidaxomicin 優先或 oral vancomycin；fulminant 早期外科會診 |

## 後續證據更新

- **選定 uncomplicated appendicitis 可討論 antibiotics-first**：CODA 顯示 30 天健康狀態 non-inferior，但 antibiotics 組約 29% 在 90 天內接受 appendectomy；有 appendicolith 者約 41%，無 appendicolith 者約 25%。一年累積 appendectomy 約四成，後續仍持續增加。**適用／限制**：需共同決策並說明急診回訪、住院與復發/手術風險；appendicolith 增加早期失敗與併發症，perforation、diffuse peritonitis 或不穩定病人不適用此論述。  
- **SSI 預防以 WHO（2016/2018）與 CDC（2017）整合指引為現行標準**：相較 2019 教材更明確的建議包括——術前 **chlorhexidine-alcohol 皮膚消毒**、維持正常體溫與血糖、適當時機給預防抗生素且 **不建議術後長時間延用**、以及不常規延長口服抗生素。**適用／限制**：多為 conditional 建議、實作需配合各院流程；抗生素縮短療程的安全性依手術類型而定。
